        “筱颖？”傅昼景轻轻唤了声。
<br/>
<br/>        童筱颖的思绪被拉了回来，她连忙将相片重新塞回钱包里的透明夹层，将钱包递给了傅昼景。
<br/>
<br/>        “你的钱包掉了！”童筱颖微微一笑，双手呈上钱包。
<br/>
<br/>        傅昼景从童筱颖的手中拿起钱包后随手塞入自己裤袋中，微笑着说了一声：“谢谢。”
<br/>
<br/>        童筱颖摇了摇头，忽然间想到了什么，试探性地问道：“昼景，你的名字是谁给你起的？”
<br/>
<br/>        “我爷爷。”傅昼景温和地回答道。
<br/>
<br/>        “那……你以前曾用过其他名字吗？”童筱颖不禁好奇地问道。
<br/>
<br/>        傅昼景苦笑着点了点头：“用过，不过被我爷爷嫌弃那名字冒犯了我父亲。”
<br/>
<br/>        “是不是叫‘龙之卫’？”童筱颖满心期待地接着问道。
<br/>
<br/>        傅昼景一脸惊讶地看着童筱颖。
<br/>
<br/>        童筱颖看到傅昼景的俊脸上露出一脸难以置信的表情，高兴地抬起手来比划着，笑容灿烂地接着说道：“我是筱颍！我是筱颍啊！小卫你忘了吗？小时候，在外婆家，我抓过鱼给你吃的！外婆要给我两一起洗澡，你不依，你说你是男孩子，男孩子不能和女孩子一起洗澡。还有，你不喜欢喝羊奶，我替你喝了。你还会吹口琴，很好听。还有……”
<br/>
<br/>        “还有你就是那个埋藏在我内心深处的小公主。”傅昼景忽然伸出手来，一边打断了童筱颖的话，一边揽过童筱颖的腰肢，温柔地将她拥入了怀中。
<br/>
<br/>        童筱颖被这猝不及防的拥抱给怔住，只觉傅昼景的这双结实的手臂，将她越抱越紧。
<br/>
<br/>        原来……
<br/>
<br/>        他还记得她……
<br/>
<br/>        这么多年过去了，她对他的记忆一直埋藏在自己的梦乡里。
<br/>
<br/>        她一直以为那只是梦，梦醒了，就什么都没有了，却不曾去回想过，原来这些全都真实存在过。
<br/>
<br/>        傅昼景也曾一样，他也以为那只是小时候做过的一个黄粱一梦。
<br/>
<br/>        梦里，有个穿着粉色连衣裙的小女孩，很漂亮、很可爱。
<br/>
<br/>        “等我长大以后，我来娶你，带你离开这座大山，去见见我们那儿的大城市，可好？”
<br/>
<br/>        小时候，离别前，他对她说过的话，忽然间记忆犹新。
<br/>
<br/>        “筱颍，我遵守了承诺，娶你为妻。”傅昼景欣慰地说道，虽然是阴差阳错，但是他和她之间的缘分更像是冥冥之中早已注定。
<br/>
<br/>        童筱颖忽然发现，自己很喜欢傅昼景的怀抱，在他的怀里，她觉得很温暖，很有安全感。
<br/>
<br/>        傅昼景紧紧地搂着童筱颖，嘴角噙着一抹发自内心的微笑。
<br/>
<br/>        从此以后，筱颍，是母亲留给他在这世上唯一一个值得他去珍惜的人了。
<br/>
<br/>        对于童筱颖的到来，季思妍第一次感到了危机。
<br/>
<br/>        明明她才是阿景和那个什么筱颖之间婚姻上的第三者，但是在季思妍看来，那个什么筱颖才是她和阿景之间爱情上的第三者。
<br/>
<br/>        不知道是不是她此刻的心理作用，仅仅只是隔着一堵墙，她就能听到阿景和那个什么筱颖的欢声笑语。
<br/>
<br/>        没过多久，她甚至还听到了大床被晃动的“嘎吱”声，以及男人的喘息声和女人的娇喘声。
<br/>
<br/>        这一瞬间，季思妍觉得自己的思绪有些混乱。
<br/>
<br/>        她独自一个人在房间里徘徊了许久许久，甚至到了三更半夜，她都无法入睡。
<br/>
<br/>        果然，她忍受不了她的阿景去睡另一个女人。
<br/>
<br/>        傅昼景是被一个电话给吵醒的，他连忙按了静音，下意识地看了一眼身旁的童筱颖。
<br/>
<br/>        童筱颖侧着身子面对着他，她的嘴角微扬，双手搭在枕头上，睡得很沉也很香，刚刚这个电话压根就没有影响到她的睡眠。
<br/>
<br/>        傅昼景会心一笑，伸手撩开童筱颖额前的刘海，俯身在她的眉心落下一个温柔的吻。
<br/>
<br/>        当他准备接着睡的时候，手机的屏幕上又亮了。
<br/>
<br/>        傅昼景拿起手机看了一眼屏幕上的内容后，吓得瞪大了眼睛，立即掀开被褥下床穿上拖鞋离开了房间。
<br/>
<br/>        他只身来到季思妍的房门口，发现房门是虚掩着的，犹豫了一下，才走进去。
<br/>
<br/>        果不其然，他看见季思妍正坐在露台的栏杆上，“啪嗒啪嗒”地掉着眼泪。
<br/>
<br/>        傅昼景疾步走了过去。
<br/>
<br/>        季思妍哭得泪眼婆娑，却在抬眸的一瞬间看到出现在露台门口的男人，早已不顾轻生的念头，快速从栏杆上跳下来，冲到露台门边一把将他抱了个满怀：“呜呜——阿景……我真的离不开你！你不要不要我，好不好？阿景……我爱你！我是真的真的很爱你！求你……求你不要放弃我！”
