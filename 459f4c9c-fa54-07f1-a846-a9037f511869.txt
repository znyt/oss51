　　假证件！？假军官！？
<br/>　　我特么拿的军官证是假的！？是来糊弄人的！？
<br/>　　慕容飞眼珠子瞪得滚圆，就差骂娘了。自己手里这本证件就算是假的，那也要有人敢做才行；可是有人敢做吗？这显然是不可能的！
<br/>　　佟千雪还没走到门卫处，就听到慕容飞和保安之间的对话。听到保安义正言辞的质疑慕容飞手中证件真实性的时候，佟千雪忍不住笑出声来，杭山市特警队队长看到都要尊敬的称呼一声“教官”的人，居然被小区保安质疑身份真假，这着实挑动了佟千雪那根并不敏感的笑神经。
<br/>　　熟悉的轻笑声，让慕容飞回过头去，看到了穿着一身居家服，拖着一双塑料拖鞋，脚上还被塑料拖鞋磨掉了一块皮的佟千雪。
<br/>　　“佟医生。”
<br/>　　保安立刻向佟千雪打招呼，眼睛中暗藏着男人对女人心怀爱慕时才会有的神情。
<br/>　　“我老婆来了，你自己问她。”慕容飞指了指佟千雪对保安说道，语气中带着些不爽，丫的，这么看我老婆，几个意思啊？不知道破坏军婚是犯法吗？
<br/>　　“你不吹牛会死啊？”保安翻了个白眼朝着慕容飞怼了回去。
<br/>　　“他没吹牛。”佟千雪微笑着走上前，对小区保安说道：“他的确是我丈夫，因为他还在部队服役，平时很少回来，你没见过他也很正常。”
<br/>　　因为脚趾被塑料拖鞋磨破的关系，所以佟千雪走路的时候有些不自然。而这自然是瞒不过慕容飞的眼睛，慕容飞赶紧蹲下身来，查看佟千雪的脚，看到她脚趾上那块因为磨掉了皮，呈粉红色的圆形小伤口，皱眉道：“这鞋子也太差了，怎么把脚磨成这个样子，肯定是‘三无’商品。”
<br/>　　“这鞋子是宋菀给我的，平时穿着挺舒服的，可能是刚才走的急，才不小心磨破了皮。”佟千雪伸出手对慕容飞说道：“抱我回去！”
<br/>　　慕容飞弯腰把佟千雪抱在手里，说道：“好，我抱你回去！”
<br/>　　“先生！”
<br/>　　等在小区外，被慕容飞和佟千雪喂了一顿狗粮的出租车司机急急忙忙走下出租车，叫住了慕容飞说道：“您车费还没付呢。”
<br/>　　被出租车司机这么一提醒，慕容飞才想起了正事，对佟千雪问道：“你带钱了吗？”
<br/>　　“没有。”佟千雪摇了摇头，说道：“我去门卫处打个电话给我姐，让她帮我把我的钱包拿下来。”
<br/>　　“佟医生不用麻烦了。”小区保安开口对佟千雪说道：“车费我先帮忙垫付，回头你再给我就行了。”
<br/>　　“那真是谢谢你了。”佟千雪对小区保安说道：“回头我让我姐姐给你送来。”
<br/>　　慕容飞抱着佟千雪往佟千雪所住的那栋楼走去，路上嘴里忍不住抱怨道：“真是抠门，一个车费还真让你还。”
<br/>　　“人家帮忙垫付车费，又不是欠你的，当然要还了。”佟千雪掐了一把慕容飞，说道：“你怎么跟个流氓一样啊？”
<br/>　　“谁让那小子对你心怀不轨来着。”慕容飞嘀咕道：“破坏军婚是违法行为，这小子是不是想去体验体验铁窗生活。”
<br/>　　“你那什么眼神啊，怎么就看到他对我心怀不轨了？”佟千雪好气又好笑的在慕容飞身上拍了一下，说道。
<br/>　　“嚯，那可厉害了，我这是狙击手的眼神。”慕容飞一脸傲娇的说道：“而且还是顶级狙击手的眼神，我告诉你，在战场上，我的眼睛就跟雷达一样，不管敌人怎么伪装，怎么躲藏，我能把他们找出来，然后再配合我出神入化的枪法，800里开外一枪打掉敌方指挥官，轻松完成斩首。”
<br/>　　“你就吹吧。”佟千雪伸出手指在慕容飞额头上戳了一下，说道：“几百年前的梗还拿出来用，你就不能有点长进？我现在严重相信刚才保安说的话，你是个假军官！”
<br/>　　“假军官？”
<br/>　　慕容飞嘴角抽了抽，盯着佟千雪说道：“你是不是皮痒痒了？”
<br/>　　“是呀，可是你能拿我怎么办呢？”佟千雪眉眼弯了起来，笑得就跟一只狡诈的小狐狸，说道：“我现在可是有王牌护身哦，你就不怕吗？”
<br/>　　不怕？怎么可能不怕！
<br/>　　自己现在要是敢找佟千雪麻烦，自己老妈绝对会扒了自己的皮，都不带犹豫的！
<br/>　　慕容飞抱着佟千雪从电梯里走出来的时候，把等在门口的佟千莹吓了一跳，以为是佟千雪出了什么事情，直到问清楚是什么情况后，才松了一口气。
<br/>　　佟千莹把药箱找出来，对坐在客厅沙发里的佟千雪说道：“这么大的人了，马上就是孩子的妈了，怎么还跟小孩子一样。”
<br/>　　慕容飞从佟千莹手里接过药箱，取出酒精棉给佟千雪磨破皮的地方消毒，然后贴上了一块创可贴。
<br/>　　佟千莹脸上挂着微笑，看着小心翼翼给佟千雪上药的慕容飞，对这个妹夫很是满意。佟千莹自己的婚姻是不幸的，所以她也一直担心自己的妹妹会跟自己一样，但现在看来，自己的妹妹比起她要幸福的多。
<br/>　　“对了，慕容。”佟千莹突然想起一件事情，对慕容飞说道：“本来还不知道要怎么跟你们家说，正好你回来了，就跟你商量商量。”
<br/>　　“姐你说，什么事情。”
<br/>　　慕容飞收拾好药箱，放到一旁。
<br/>　　“下个月25号，我要带几个学生去T国做个课题，差不多一个礼拜的时间。”佟千莹带着商量的口吻跟慕容飞说道：“小雪想要跟我一起去，那个时候小雪也三个多月了，出去走走对孕妇也有好处，而且我们全程都安排好了车辆，也不会太劳累。但是我也知道，你们家对小雪肚子里的孩子很重视，所以还是要征询一下你们家里的意见。”
<br/>　　T国，从距离上看，也不算太远，坐飞机也就4个多小时，来回很方便，而且去那边旅游的人也很多，加上佟千莹学校也有安排，其实也没有什么担心，只是慕容飞有些担忧佟千雪的身体是不是能吃得消。
