　　苏莫的打算，是先让天下所有势力臣服，归顺于苍穹神宫，以苍穹神宫为尊。
<br/>　　到时候，在慢慢融合吞并，哪怕不融合，只要力量能拧成一股绳也可。
<br/>　　他邀请各大势力的宗主、门主等人前来苍穹神宫，来的势力，自然便是表示臣服。
<br/>　　若是有哪个势力没有前来，那就直接剿灭，以震慑天下。
<br/>　　安排好了事情之后，苏莫便走出了宫殿，准备前往蛮荒山脉，接回小八。
<br/>　　唰！
<br/>　　就在此刻，空中青光一闪，两道人影来到了苏莫的身前。
<br/>　　来人之一正是长青宫守护傀儡人，青一。
<br/>　　而在青一的身边，还有一名女子，正是青雅公主。
<br/>　　“宫主！”青一向苏莫行礼。
<br/>　　“青一前辈，何事？”苏莫打量了两人一眼，心中有些疑惑，不知青一为何会带着青雅公主。
<br/>　　“宫主，此女的天赋，很适合做长青宫的传人！”青一一指身边的青雅公主，沉声说道。
<br/>　　“哦？”苏莫闻言一怔，心中很是意外，青雅适合做长青宫的传人！
<br/>　　“是啊！她的木属性天赋，接近木属性灵体的地步，而且拥有天级二阶的剑武魂，各方面天赋都不错！”青一点头道。
<br/>　　苏莫闻言沉默了，既然青一说了此女适合，那就肯定不会有错了。
<br/>　　只不过，他虽然也想尽快安排好长青宫的传人，但心中还是有点不太愿意选择她人。
<br/>　　毕竟，长青宫的传人，他是为宏青璇准备的。
<br/>　　可是现在宏青璇遭难，不知道什么时候能解救出来，也不知道对方能否平安无事！
<br/>　　只希望，海明江尽快抓捕黄泉魔宗余孽，能找到会搜魂术之人。
<br/>　　望着沉默的苏莫，青雅公主心中忐忑，她也知道长青宫传人是何等重要，将来要统御一宫。
<br/>　　“宫……宫主，我并不需要做传人，能加入苍穹神宫我已经很满足了！”青雅公主轻咬红.唇，轻声说道。
<br/>　　苏莫闻言轻叹一声，摇了摇头道：“青雅，从今日起，你便是长青宫的宫主，努力修炼，将来带领长青宫走向强盛！”
<br/>　　言罢，苏莫转头看向青一道：“我离开几日，开启护宫大阵。
<br/>　　随即，苏莫身形拔天而起，急速消失在了天际的尽头。
<br/>　　青雅公主神情怔怔，目光望着苏莫离去的背影，久久无声。
<br/>　　良久之后，青雅攥了攥小拳头，深深的吸了口气，呢喃自语：“青雅不会让你失望的！”
<br/>　　苏莫离开苍穹神宫之后，便径直向东方飞去，目标蛮荒山脉。
<br/>　　他最终还是同意了让青雅作为传人，毕竟传人之事，事关重大，他不能一直等待下去。
<br/>　　至于宏青璇，将来若是能救出对方，再行安排吧。
<br/>　　……
<br/>　　辽阔的蛮荒山脉延绵数百万里，仿佛无边无际，蛮荒圣山如同擎天之柱，屹立的天地之间。
<br/>　　这里是苍穹世界的禁地，任何的超级势力，都不敢踏足此地。
<br/>　　这一日，一道犀利的剑光，洞穿了虚空，闪电般的飞临到了圣山。
<br/>　　随即，剑光在通往圣山内部的通道口停了下来，化为了一名冷峻的青年。
<br/>　　青年自然便是苏莫。
<br/>　　吼！嗷！啾！！
<br/>　　苏莫到来，顿时引起了圣山上各种妖兽的吼叫，霎时间，无数的妖兽四散飞舞。
<br/>　　苏莫并未进入圣山中，而是伫立在圣山的通道口，静静的等待。
<br/>　　他的气息已经散发而出，方圆数千里之内都能轻易的感受到。
<br/>　　“苏莫大哥！”
<br/>　　不多时，一声惊呼声从圣山内部响起，而后便见到一名俏生生的紫衣少女，从通道中疾驰而来。
<br/>　　“晓晓！”苏莫见到少女，顿时微微一笑，来人正是小八，不过现在的名字，叫做翼晓晓。
<br/>　　“苏莫大哥，你怎么来了？”翼晓晓来到苏莫的身前，满脸惊喜的问道。
<br/>　　“我来接你，离开蛮荒山脉，去苍穹神宫！”苏莫微笑着说道。
<br/>　　“真的？”
<br/>　　翼晓晓闻言，顿时眼眸一亮，不过，随即她便脸色暗淡了下来，撇嘴道：“怕是师父不会同意！”
<br/>　　她师父有令，待她成为了七级妖兽，才能离开蛮荒山脉，外出历练。
<br/>　　但它现在只是六级八重妖兽，距离七级妖兽，还有一段距离。
<br/>　　“没事！谁也无法阻拦！”苏莫摇了摇头。
<br/>　　“好狂的口气！”
<br/>　　苏莫话音一落，一道低沉浑厚的声音陡然响起。
<br/>　　随即，便见到通道中，光头大汉和那名黑衣中年人走了出来。
<br/>　　出言之人，正是光头大汉。
<br/>　　“两位前辈！”苏莫见两人现身，向两人微微抱拳，言语客气。
<br/>　　上次他遭遇二魔老的追杀，还是靠这两人才逃得一命，尤其是光头大汉，更是出手将二魔老击伤。
<br/>　　“小子，你说要带走小丫头，问过我们了没有？”光头大汉满脸横肉，声音如同闷雷。
<br/>　　“呵呵！既然两位前辈来了，那我就明说了，在下今日一定要带走晓晓！”苏莫笑着说道，以他现今的实力，天下之大已经没有人能抗衡他了。
<br/>　　“你认为你有这个实力吗？”冰冷的声音响起，开口之人是晓晓的师父，黑衣中年人。
<br/>　　其眼眸打量了苏莫，面色不见微微动容，苏莫的修为提升速度，让他也是感觉有些心惊。
<br/>　　不过，想从他们两人手中抢人，苍穹世界还没有谁有这个实力！
<br/>　　“两位前辈可以试试！”苏莫面色淡然，眸中蕴含无敌的自信。
<br/>　　“小子，你可真够狂的，你若是能接住本帝一招而不败，我们就让你带走小丫头。”光头大汉见苏莫一脸傲气，不将他们放在眼中，立刻大声咆哮了起来。
<br/>　　“可以！”苏莫点了点头，心中有些好笑，这位远古战象，明明只是准帝，并不是真正的武帝，却口口声声以本帝自居！
<br/>　　“苏莫大哥不要！”
<br/>　　翼晓晓闻言顿时小脸一变，急声道：“你打不过象叔叔的，你们有事好商量嘛！”
<br/>　　翼晓晓常年待在蛮荒圣山，对于自己的师父和光头大汉的实力，实在是再清楚不过了。
<br/>　　作为远古异兽的后裔，两人的实力恐怖至极，远超同阶的人类武者。
<br/>　　而苏莫才修炼多少年，就算再厉害，也不可能接得住她象叔叔一招！
<br/>　　“无妨！”
<br/>　　苏莫微微摇了摇头，随即，他目光扫视了一眼光头大汉和黑衣中年人，笑道：“两位前辈还是一起上吧！”
