　　“想你了，来看看你啊。”
<br/>　　叶少阳指着黑板上乘法口诀：“你……怎么教他们这个东西啊。”
<br/>　　“不光这些，我把能想到的人间的常识都交给他们。”芮冷玉道，“他们不缺修行法门，缺的是对人间文化额了解啊，毕竟有足够的文化，有成熟的价值观和世界观，才能看懂很多深奥的经文……这些都是修道的基础。”
<br/>　　叶少阳揉了揉鼻子，道：“我也没什么价值观和世界观啊，不也照样修道。”
<br/>　　“你有的，只是你自己没有总结出来而已，你是实践派的。”芮冷玉带他来到“教室”旁边，那里有一个柜子，打开来，里面全都是书，叶少阳大体看了一下，左边柜子是教科书，小学教材，中学教材……右边是各种杂七杂八的书，古典小说、各种杂谈什么的都有，还有十万个为什么。
<br/>　　“这些都是我托军师从人间弄来的，怎么样，不错吧。”芮冷玉洋洋得意。
<br/>　　“额，你怎么想起当老师来了？”
<br/>　　“一开始，是因为太清闲，给自己找点事做，后来就慢慢就觉得挺有意思的，而今不光是修道的，附近有很多生灵都慕名前来，为了听我的课，他们回去之后，会将这些知识四处传播，让这个世界逐渐变得文明起来。”
<br/>　　“他们哪里不文明了。”
<br/>　　芮冷玉白他一眼，“这里的文明，不是那个意思，是文化的意思。所以，我感觉自己就像是一个拓荒者，这感觉挺有趣。”
<br/>　　叶少阳虽然不懂，但是也能理解，跟她一起出了内堂，来到后面的卧房里。
<br/>　　叶少阳一眼看到了道风，在床上盘膝坐着，正在打坐。
<br/>　　跟当初的他对比起来，现在的他好像瘦了一些，脸色也不是那么好，就是头发长了，两边披散下来，芮冷玉拿了一把梳子，过去给他梳头。
<br/>　　“我自己可以了。”道风接过梳子，说道。
<br/>　　“大伯！”瓜瓜打了招呼。
<br/>　　叶少阳在床边坐下去，看着道风。
<br/>　　道风也看着他。
<br/>　　“死了一次？”叶少阳呆了半晌，问道。
<br/>　　道风从桌上拿起了一个竹筒，像个正常人似地喝了口水，点点头。
<br/>　　“那你现在……算是什么状态，人，鬼，还是元神？”
<br/>　　“都不是。”
<br/>　　“那是什么。”
<br/>　　“我已跳出三界五行，这是一种不生不灭的生命状态，说了你也不懂。”
<br/>　　“证道了？”叶少阳心情激动起来。
<br/>　　道风想了一下说道：“是证道了，不过想要恢复修为，并且超过我以前的实力，还需要一段时间的修炼。”
<br/>　　叶少阳皱眉道：“这是什么道理，李浩然证道了，为什么就可以一下子变得很厉害？”
<br/>　　道风笑道：“每个人的道不一样，证道的过程也不一样，怎能一概而论。”
<br/>　　叶少阳一想也是，道：“那你多久能恢复？”
<br/>　　“半个月到一个月吧。”
<br/>　　芮冷玉在一旁说道：“少阳，道风走得这条路，比一般证道的路都要凶险难走，因此上限也高，一旦他元神恢复了，比一般证道的强者都要厉害的。”
<br/>　　叶少阳震惊，随后也替道风感到高兴，冲他咧嘴笑道：“总算等到这一天了啊。”
<br/>　　道风轻轻抿了下嘴唇，道：“你在古墓里怎么样？”
<br/>　　“说来话长。”叶少阳只好把事情又讲了一遍——从古墓里出来，已经是第三遍还是第几遍了。但道风和芮冷玉是第一次听到，听他讲完，道风还好，脸上没什么表情，芮冷玉确实吃惊不已。
<br/>　　“那星月奴居然也证道了……”芮冷玉倒吸一口凉气。
<br/>　　道风沉默半晌，说道：“各人有各道，她徒弟之中，都有两三个证道的，她原本几百年前就该证道的，如今已是晚了。”
<br/>　　叶少阳闻言，心中一动，道：“是啊，她是轩辕上帝的弟子，修行也有千儿八百年了，为什么到现在才证道？”
<br/>　　道风跟他讲解起来：
<br/>　　“证道，不是看修行时间长短，这个你大致懂的，但各人有各人的道，证道的难易程度也不同，再说星月奴一向管着法术公会的事务，没法闭关清修，修为在几个长老之中最差，而今也是放弃了之前的道，借助了天下信徒的愿力，算是速成的邪修了，她就算证道，实力也有限，当然，这个有限是相比较而言。”
<br/>　　道风很罕见地说了一大串话，叶少阳听完，心中有感，问道：“有差距我知道，难道差距很大吗？有多大？”
<br/>　　“这个……怎么好去形容。”道风也犯了难。
<br/>　　叶少阳突然想到一个人，说道：“李浩然，李浩然不也才证道不久吗，星月奴跟他比，怎样？”
<br/>　　道风想了一下道：“大概……就像我跟你之前的差距吧。”
<br/>　　叶少阳愕然。
<br/>　　差距……这么大？
<br/>　　道风补充道：“李浩然轮回百世，一心求道，一朝飞升之后，三界之内罕有匹敌，五个星月奴在一起也不是他对手。”
<br/>　　想了想，又道：“他的实力，大概跟是你这次遇到的鸠摩罗什差不多，略逊于几位真神，不过相对于别的法师和邪物来说，差不多可以碾压一切。”
<br/>　　“真神是什么？”
<br/>　　“只是一种通俗的说法，就是三界之内最高的那几位，轩辕上帝，酆都大帝，东岳大帝，东皇太一，地藏菩萨，接引道人……大概就是这几位了，哦对，还有无极鬼王。”
<br/>　　叶少阳默默想了下这几个名字，每个都是一方生灵或者领域的统治者，但有个共同的特征：这几位大爷好像从来都不露面，没什么存在感？
<br/>　　当下将这个疑问道出，道风笑道：“他们这个级别，已经是修行的登峰造极，弹指之间，能改变三界格局，正因此这样，彼此之间是一种平衡关系，谁也不能妄动，就像……”
<br/>　　“就像是各国的核武器一样，虽然牛逼，但更多是用来互相威慑，不能真的使用？”
<br/>　　道风瞥了他一眼道：“你这个比喻好新奇，不过大概就是这样，而且，这其中有许多已经化成了意志，并不是我们理解的真实存在的人……可以说，这个世界的各个空间、规则，就个这几人联合起来制定的，他们是原始推动力，但除非遇到三界毁灭的大事，否则是不会露面的。”
<br/>(本章完)
