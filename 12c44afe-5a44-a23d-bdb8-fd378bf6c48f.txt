<br/>        “行，你说了算，我来约局。”

<br/>
<br/>        当初那个项目呼延寰宇不让他参加，说是他公司没有那个能力，拿不下来，他心里不服气，托关系将那个项目拿下来了，后来折腾了好几年，最后还是呼延寰宇找到他，抽时间帮他完成的。
<br/>
<br/>        按照呼延寰宇那个犟驴脾气，他是不会主动帮他的，也只有陈子昂会帮她，想一下这几年和陈子昂处处作对，心里有那么一丝丝愧疚，但这愧疚也只是一刹那而已，很快就被呼延临庆忽略了。

<br/>        “小易总，你怎么还不走？”

<br/>        “我等呼总啊。”

<br/>        “当然，不然我站在这里干嘛呢？”
<br/>
<br/>        “你怎么知道我要走。”
<br/>
<br/>        “事情办完了，你不走还等着顾恩赐赶你走吗？”
<br/>
<br/>        “也是，那么小易总等我干什么呢？不会想真请我去你们公司喝你那七年的白茶吧？”
<br/>
<br/>        “哈哈哈，这也不是不可以的，我就想知道呼总洛城的项目做得如何了？听说你们到处找人进行二次开发，费时费力，不如让我们陈总直接帮帮你们不香吗？”

<br/>        “你们陈总会帮我吗？”

<br/>        呼延临庆心里一动，刘清影他们都挖过来了，这个小女孩也还不错，不如.......

<br/>        “你肯定适合去，都是我们这里的人，老宋、许弋澄、顾恩赐，当然还有夏总。”

<br/>        “哇，都是大佬，我怕是不适合过去，我还是不去了，呼总，欢迎你到我们公司做客，你想好要真的需要我们陈总帮忙，你给我说。我还有事情，我先走了，拜拜溜。”

<br/>        他有些烦躁，点了一根烟，其实他很少抽烟的，最近感觉事情不顺利，父亲生病，公司内部出现问题，也不知道子昂在北京怎么样。

<br/>
<br/>        “你还知道你是我老爹啊，这都几天住公司都不回家了啊？”陈天天的个子不小了，但是嗓音没有转换过来，脆生生的夹着童音。

<br/>        “是你们家庭环境好，天天无忧无虑。陈总，我今天见了顾恩赐和呼延在一起，估计他们在一起没有什么好事情，顾恩赐这个见利忘义、过河拆桥的家伙我们以后一定要防着点。”

<br/>        “你去见见江总，让他抽时间我们开会。”
