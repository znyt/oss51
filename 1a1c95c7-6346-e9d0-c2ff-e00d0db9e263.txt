        &lt;h3&gt;第三卷第二百四十九章  戏精上身&lt;/h3&gt;
<br/>
<br/>        俗话说，兵来将挡水来土掩，寒伯安知道弟弟没有事情，他也就安心了，现在唯一担心的就是安男，不过安男在安家，他们也不能将安男怎么样，最起码老爷子还在。
<br/>
<br/>        老爷子虽然宠爱苓如兰，总是偏向苓家一些，但是在大事情上还是能辩是非的，要不然寒家也不可能有现在的这种地步，再说，机智如安男，也应该能发现端倪。
<br/>
<br/>        他们在第二天迎来了沧岚雪。
<br/>
<br/>        寒伯安昨天的金蝉脱壳之计并没有骗得了沧岚雪这只老狐狸，他早就在寒伯安公司的周围安排了人手，无论是寒伯安和李长卿，一个都不能离开，直到他们的合同签署完成。经过三年的努力，沧岚雪已经在这里创下了一片天地，为他们后期做好铺垫，只等谷强的信息化项目完工，有可能将来会利用网络打通世界各地。
<br/>
<br/>        他们的灰色事业也一直在不屈不挠的进行着，俗话说道高一尺魔高一丈，为了荣华富贵，不，现在可以说是他们的事业，对，为了他们伟大的目标他们都在努力着，他们离不开运输业，无论是明面上的还是暗处的。
<br/>
<br/>        寒家多少年的老企业，在业界响当当，光凭着寒家的名声，寒家就算运输一些东西都不会有人怀疑，拿下寒家等于就是拿下运输业中的整个链条。
<br/>
<br/>        他们在很久之前就开始谋划，原本想拖着他们下水，可是寒伯安态度强硬，软硬不吃，于是只好从寒家身边的人动心思。
<br/>
<br/>        这一次刚好也是一个锲机，他们对付的是落雪，没有想到寒伯安机缘巧合之下也到了美洲，落家和寒家关系极好，寒冰玉和落雪同时出事情，寒家也会受到打击。
<br/>
<br/>        寒伯安到美洲只带了一个李长卿，将自己的妻子留在家里，恰好，苓家和他们也有来往，如果安男不在寒家老宅的话，他们更容易得手，但是安男被喊去了寒家老宅，他们只好安排苓希去寒家，在合适的时机掌控安男逼寒伯安就范。
<br/>
<br/>        沧岚雪顶着熊猫眼到了寒家分公司的时候，寒伯安正在悠闲的抽着他的雪茄，吐着圆圆的眼圈惬意的斜躺在沙发上。
<br/>
<br/>        李长卿狗腿的站在寒伯安的身后，两只手灵活的在寒伯安的肩膀上敲打着。
<br/>
<br/>        “我再也不离开你了，虽然我很想当总裁，但是你也不用离家出走，出走就出走，还要威胁我，让我为你提心吊胆，你说你都多大的人了，还不让我省心。”
<br/>
<br/>        “我怎么就不让你省心了？你想当总裁可以啊，你来啊，现在就让你当。”
<br/>
<br/>        “我是想当总裁吗？我只是想和你在一起，只要你将身边的花花草草清除了，我就一辈子不离开你。”李长卿狠狠的掐了一下寒伯安。
<br/>
<br/>        沧岚雪就那样站着看着俩个大男人卿卿我我，唧唧歪歪，他感觉他的三观碎了一地。
<br/>
<br/>        “怎么清除，你知道，那些都是摆设，你还要我怎么样？”寒伯安呲了一下牙，拉过李长卿的手，握在手中。
<br/>
<br/>        “咳咳。”沧岚雪已经看不下去了，出声打断了寒伯安和李长卿。
<br/>
<br/>        李长卿迅速抽回了手，有些羞涩的说：“沧总来了啊，请坐，我给你们沏茶。”
<br/>
<br/>        寒伯安：“要你沏什么茶？不知道让别人沏？”
<br/>
<br/>        李长卿：“你咋这样，你看，这有外人呢。”
<br/>
<br/>        “沧总是外人吗？沧总昨天还看见你发飙呢。”
<br/>
<br/>        沧岚雪听寒伯安如此一说，心中恍然大悟，他就说一个助理怎么敢对自己的上司动手，这样一想，他心里就释然了。
<br/>
<br/>        千算万算谁都没有算到寒伯安喜欢的人就在他自己的身边，那个安男怕也是没有多大的用处，只不过这个李长卿倒是可以利用。
<br/>
<br/>        “沧总，请坐吧，哎呀新闻快要开始了，不如我们一起看看新闻？”寒伯安连身子都没有抬，依然惬意的斜依在沙发上。
<br/>
<br/>        “不，寒总，我不看新闻，我还是想和你谈谈合作的事情。”沧岚雪对于新闻他没有兴趣，今天他就想拿到和寒家的合约。
<br/>
<br/>        “长卿，请沧总坐下，看看沧总的条件。”
<br/>
<br/>        “你的意思今天我做主？”李长卿欣喜的问。
<br/>
<br/>        “今天你就做主了。”寒伯安淡淡的笑着。
<br/>
<br/>        “快，沧总，你的文件让我看看，别等一会他反悔了。”李长卿催促沧岚雪拿出文件。
<br/>
<br/>        沧岚雪拿出文件交给李长卿，李长卿看也不看跑去办公桌上拿起一支笔，刷刷的就签下了自己的名字。
<br/>
<br/>        “沧总，拿好，字我签了，等回国了你拿过来我给你签章。”李长卿兴奋的说。
<br/>
<br/>        “你连看都不看？”沧岚雪感觉蹊跷。
<br/>
<br/>        “你不想要啊，这个可是他给我的机会，不要算了，拿来我撕了，他还说所有的利润都归我呢。”李长卿有些遗憾。
<br/>
<br/>        “不，不，不，我没有想到寒总对李兄弟如此厚爱，我看走眼了，看走眼了。哈哈哈哈。”沧岚雪忍不住大笑。
<br/>
<br/>        “看新闻，看新闻。”寒伯安对李长卿和沧岚雪说。
<br/>
<br/>        新闻里是落雪和寒冰玉还有他们的朋友并肩而行，谈笑风生的情景。
<br/>
<br/>        沧岚雪惊得站了起来：“怎么可能？他们不是已经........”
<br/>
<br/>        “我也觉得不可能，哈哈哈哈，沧总，你看看他们是不是真的？我这老眼昏花的，看不清楚，看不清楚！”寒伯安放声大笑，看着沧岚雪那吃惊的样子，他心情很好，不过不知道他的主子们也像他这样吃惊？
<br/>
<br/>        “呵呵呵，恭喜寒总，贺喜寒总，寒二少没有事情就好，没有事情就好啊。”沧岚雪也大笑。
<br/>
<br/>        “同喜，同喜，我们也签下了合作协议不是，沧总回国找长卿盖章吧！”
<br/>
<br/>        “好，寒总什么时候回去？我们一起？”
<br/>
<br/>        “就这几天吧？苓希这几天在我家里，既然沧总和她也认识，不如到时候一起坐坐。”
<br/>
<br/>        “苓希确实是在你家里陪着你的那位妻子，俩个人还成了姐妹，真的很佩服寒总。不过听说，你的那位妻子和陈子昂走了。”沧岚雪既然拿到了合约，倒也不会再去为难寒伯安，说这话就是摆明了，安男如果有什么事情也不是他们做的。
<br/>
<br/>        “长卿，你看，你应该和苓希、安男学习学习，别成天动不动的就对我动手，像一个泼妇一样。”寒伯安的心情很好，对长卿的话里多了一份宠溺。
<br/>
<br/>        沧岚雪听得起了一身的鸡皮疙瘩，但是心里却乐开了花！
<br/>
<br/>        ......
<br/>
<br/>        看着沧岚雪满意的离开，寒伯安和李长卿哈哈哈大笑。
<br/>
<br/>        “长卿，你酸到我了。”
<br/>
<br/>        “你要对人家负责啦，清白都被你毁了。”李长卿捏着鼻子。
<br/>
<br/>        “滚。”寒伯安大长腿一伸照李长卿的腿上来了一脚。
<br/>
<br/>        “典型的过河拆桥！我要去找安男姐姐投诉，你天天欺负我。”李长卿躲开了寒伯安的那一脚，跳得远远的喊。
<br/>
<br/>        结果不错，他们需要打道回府了，还有更长的路要走，一切都是未知数。
<br/>
<br/>        “长卿，收拾收拾，我们回家！”
