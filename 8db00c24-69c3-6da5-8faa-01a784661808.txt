　　“可是，法术公会已经蛊惑了大半个法术界，我们要拯救的人，其实就是这些反过来对付我们的人，这可真是讽刺。”
<br/>　　李浩然转头看着她，说道：“你记着，我们谁都拯救不了，所做一切，无愧于心而已。”
<br/>　　碧清反复咀嚼这句话，点头说道：“多谢师兄教诲！”
<br/>　　第二天，叶少阳三人一道去了桃花山主峰下，还没到跟前就被法术公会的人拦住了，远远看去，偌大的山峰已经被人团团围住，每隔二十米左右，地下就插一面旗子，颜色各不相同。
<br/>　　叶少阳自然知道这是旗阵。
<br/>　　三人绕着山峰缓缓行走，一路来到山门下。这里是主阵所在，旗帜更多，分隔出了一块空地，中间有数十道人影，围着一座法坛，盘膝而坐。
<br/>　　叶少阳虽然不知道这是什么阵法，但也能通过这些人坐在地上的位置，看出主阵者是谁：正对着法坛的三人，呈品字座次，一个和尚两个道士。
<br/>　　“这和尚是大明寺住持空明禅师，大明寺是不出名，不过这空明禅师实力不凡，仅次于几大山门的掌教，是最早依附法术公会的，算是骨干。”
<br/>　　道渊真人为叶少阳做着介绍。
<br/>　　“后面两个道士……一个是函谷关的李大先生，一个是津门仙灵山的天珠道长，也都是被法术公会收编了的。”
<br/>　　叶少阳缓缓点头，目光落在法坛的另一边，那里有两个人在下棋，一个浑身穿着破烂衣服，蓬头垢面的，看上去像个乞丐，另外一个一身白色长衫，鹤发童颜，叶少阳一眼看过去，感觉就像是电视广告里经常出现的那种卖养生药的白胡子爷爷。
<br/>　　两人对着一个石桌而坐，桌子上摆放着一只棋盘，两个人正在下棋。
<br/>　　“这两个人，什么来头？”叶少阳看着好奇，问道。
<br/>　　“那个乞丐就是陈七袋了，我之前跟你说过的，当今法术界最强的几个人，一道二僧三头陀，四丐五仙六巫婆，这个陈七袋就是其中之一，法术倒是一般，不过是个堪舆大师，布阵的本事可以说是冠绝天下，堪比地师家族。也不知道是在哪里学的，依我看，这次围困桃花山的阵法，就是他指挥布置的。对面那个白衣老头，是‘四丐五仙六巫婆’中的‘仙’，东北马仙，姓陈，人称陈大仙。扶乩的本事也是一绝。”
<br/>　　东北马仙，也叫出马弟子，还有一个更通俗的名字叫“跳大神的”，是西域巫术流传到北方之后，发扬出来的一大民间教派。
<br/>　　“我听说，出马弟子，都供奉家仙，他们的实力高低，实际上比的是家仙的修为？”
<br/>　　毛小方插了一句：“是这样，这个陈大仙，据说供奉的是一只千年黄皮子，修为极深，轻易不露面。当年法术公会为了收伏一个门派，恶战一场，这陈大仙打先锋，请出家仙，力战并杀死了一位地仙……可见实力之强。”
<br/>　　叶少阳不解问道：“有这么强的实力，干什么不好，非要给人当走狗？”
<br/>　　毛小方笑了一下，道：“一道二僧三头陀，四丐五仙六巫婆，除了那一道二僧，其余都归顺了法术公会，说到底，还是为了一个名分。”
<br/>　　“名分？”
<br/>　　“对，你看这些人，三位密宗苦行僧，丐帮出来的堪舆大师，波月洞鬼婆，东北出马弟子……你想到什么？”
<br/>　　叶少阳怔怔地看着他。“都是……偏门？”
<br/>　　毛小方点点头。“这些人实力够，一心修行，不好金钱女人什么的，法术公会笼络他们，其实就是一个‘名’字。”
<br/>　　“哦，我懂了。”叶少阳是真的懂了。过去，在元朝之前，中原汉文化盛行，法术界也是一样，有正统的说法，即道佛二宗，这其中的佛门，也是以禅宗为首的汉传佛教。民间散修的地位其次，至于那些中土之外的各种巫师、大仙，甚至包括藏传佛教的宗师，对于中原的法术界来说，都是偏门，是没什么话语权的。
<br/>　　这一点，倒是很想过去江湖上的门户之见，“中原武学”这个词，本来就带着一种高高在上的感觉。法术界也是一样。
<br/>　　不过，在当时这也不能怪中原法术界，以为那时候少数民族与中原不属同一个国家（至少是属国），中原法术界对中土之外的法术不是很了解，自然也谈不上有多尊重。
<br/>　　后来天下一统，蒙古入住中原，虽然蒙古人崇信道教，但也有少数民族的巫术，被巫师带进了中土，出现在法术界的视野之中，再后来，大清国的鞑子又加深了这种文化融合。
<br/>　　民族文化融合，其中也包括了法术界。
<br/>　　在清朝中叶，有几位德高望重的宗师一起发动了改革，每逢重大决议，也会邀请一些非中土的有名法师参加，歧视也没有了。
<br/>　　不过，在很多中土法师的骨子里，尤其是道佛二宗，对待民间散修都有点蔑视，更不要说对这些中土之外的法术门派，因此，这些什么大仙巫婆的，虽然个人非常牛逼，但缺乏的是一种认可和地位……法术公会想要拉拢他们，靠的就是这个。
<br/>　　说到底，就是一种存在感。
<br/>　　叶少阳完全懂这种感觉。不过，在自己那个时代，真正的文化大融合，这些门户之见早就没有了。
<br/>　　在旗阵的外面，站着两排好几十个人，有年轻的也有老一点的，看穿着都不是一个门派的。一个个站着不动，望着山门方向。
<br/>　　道渊真人告诉他，这些不是法术公会的弟子，而是那些归服他们的小门派，各自派出人来，在这里值守。
<br/>　　叶少阳抬头朝山上看去，桃花山上，果然到处都是桃花，正值四月，也正是桃花盛开的时候，看上去漫山遍野，倒是好风景。只是山上一个人都看不到。
<br/>　　“今天是给桃花山的最后一天，如果今天山上的道士再不下山，明天法术公会就要攻打山门了。”道渊真人道。
<br/>　　叶少阳笑道：“既然要攻打，还故意给出三天时间，法术公会也是想要立牌坊，不过也就是表面工夫了，如果人家想下山，早就下山了，到现在还不下来，说明肯定是不想下山了，就算等一年也没用。”
<br/>　　（祝大家中秋节快乐！青帮蒸蒸日上！）
