　　第二天早晨，整个基地静悄悄。
<br/>　　苏重一身黑色武士服，全服武装。腰间别着一把长刀，推开门从容来到训练场。
<br/>　　平日里热火朝天的训练场面消失不见。取而代之的是一群黑衣蒙面，肃然而立的影武者。
<br/>　　杜卡一身劲装，站在众人之前，肃然看向苏重。
<br/>　　“苏，你已经来基地半年。所有训练都已经完成，现在，是时候检验成果了。”杜卡转身，从木桌上拿起一个陶质小碗，里面放着几块燃烧鲜红的木炭块。
<br/>　　杜卡拿出一把晒干的蓝紫色花朵，揉碎之后扔陶碗内。
<br/>　　嗤！
<br/>　　一震烟雾升腾，如梦似幻，杜卡伸手对苏重挥手：“来！深吸一口气”
<br/>　　致幻剂？苏重了然，知道这就是能够唤起人内心恐惧的致幻药剂。在药物作用下，能暴露人内心深处的恐惧。
<br/>　　只是，那满满一把的量，当我是大象吗？当初布鲁斯韦恩，只用了一朵就陷入幻境。杜卡这是打算来探自己的底？
<br/>　　苏重饶有兴趣走上前，他还真想看一看自己的恐惧。
<br/>　　吸！
<br/>　　一股淡淡馨香冲入鼻腔，然后一股凉意刺入大脑，一股从心底泛起的寒冷，让苏重皮肤猛然泛起一阵鸡皮疙瘩。
<br/>　　闭上眼睛，苏重仔细感受有些模糊的精神。
<br/>　　“苏！告诉我，你来影武者联盟，是为了什么！”杜卡凑到苏重耳边，猛然一声大喝。
<br/>　　锵！哈！
<br/>　　数十个全副武装武士同时大喝。肃杀气息骤然笼罩苏重。
<br/>　　“放松，放轻松，慢慢说，告诉我。”杜卡声音呢喃，又变得轻柔而充满蛊惑。
<br/>　　“杜卡老师，你想让我说什么？”苏重睁开眼，猛然转头。双眼清明，盯着杜卡戏谑道。
<br/>　　“你……你没事？”杜卡被吓的猛退后一步。
<br/>　　“老师很希望我有事吗？”
<br/>　　“那可是能够引起恐惧的致幻烟雾。那种计量，你的感官应该无限放大才对！蚊蚋声如雷，视线模糊一片，怎么可能保持清醒？”
<br/>　　“声音如雷？不过是些杂音而已。”苏重淡然道。
<br/>　　耳朵里的声音确实杂乱，眼前景象也变得光怪陆离。但苏重灵魂强大，意志如铁，这具身体又只是普通常人，些许干扰刺激，根本就无法让苏重失去理智。
<br/>　　“那么杜卡老师，我通过考验了吗？”苏重玩味道。
<br/>　　杜卡深吸一口气，很快就镇定下来，脸上恢复优雅笑容：“当然，你的表象让我刮目相看！”
<br/>　　“驾驭心中恐惧，完全保持清新个。很好，第一关，你完美通过。”杜卡笑道。
<br/>　　“下面我们进行第二项测试。”杜卡右手摆了一个手势，两名黑衣武士立刻脱离队伍。离开训练场。
<br/>　　很快两人回来，不过却押着一个人走了进来。这人带着头套，看不到面目。不过从露出来的皮肤可以推测，这是一个壮硕的白人。
<br/>　　对方不是挣扎两下，动作强劲有力，手指上有一层厚厚老茧。一看就不是普通人，而是一个训练有素的士兵。
<br/>　　“这个人本来是组织的一员，可惜被贪婪蒙蔽双眼，竟然背叛联盟！”杜卡饶有深意的看了一眼苏重：“即便他组建了一个强大佣兵团自保，但背叛者，不论躲到哪里，都必须付出代价。苏，你说我给如何审判他？”
<br/>　　噗嗤！
<br/>　　一道血线骤然飞起，头颅滚落地面咚咚作响。苏重把刀收回刀鞘：“还有其他考核吗？”
<br/>　　杜卡再次怔了怔，他没想到苏重下手这么果决。他很快反应过来：“叛徒的下场就是死亡，苏，你做的很好。你通过了测试，从明天开始，你就是基地格斗训练助教。相应的，你可以支配的资源也会提高。”
<br/>　　苏重点点头：“还有其他事吗？我要去练枪了。”
<br/>　　“去吧，不要着急。每个人都有自己的长处和缺点。不必气馁，想一想你的格斗术，那才是你的强项。”杜卡语重心长。
<br/>　　苏重不置可否，微微躬身行礼，转身离开了肃穆的练功场。
<br/>　　杜卡看着苏重背影，眯起眼睛，对苏重越发戒备警惕。
<br/>　　为了探底，他特地加大致幻剂药量。那种剂量，甚至能把人逼疯！他自己都不敢试，没想到苏重竟安然无恙！
<br/>　　吸入那么多致幻烟雾，竟能保持平静。不是意志过人，就是彻头彻尾的疯子！他见过那种人。看着正常，可谁也不知道他心里想什么。
<br/>　　不管苏重是属于那种，杜卡都有种失去掌控的感觉。他更不敢放苏重离开基地。这个基地很重要，但更多是象征意义，本质上是一个训练基地。联盟的重要项目核心，并不在这里。
<br/>　　而且还有他自己坐镇，这才是他敢把苏重拴在这里的原因。
<br/>　　不管杜卡怎么想，苏重毫不在乎。
<br/>　　这半年时间，他可不是白过。杜卡的格斗术几乎被他掏干净，有技能面板作弊，每一项格斗术，几乎都被他练到圆满。
<br/>　　加上他的武道境界，只论格斗术，苏重有信心挑翻整个训练基地！就连亨利·杜卡也不例外。
<br/>　　有这身本事，他在影武者联盟，已初步站稳脚跟。不必再担心，半夜三更被人偷偷暗杀。
<br/>　　除非动用热武器，苏重还真不怕影武者的所谓潜心暗杀。真要有人敢来，还说不定谁杀谁呢。
<br/>　　助教的工作实际很轻松，只是在旁边辅助训练。涉及到训练计划制定，苏重根本插不上手。他也不想插手。
<br/>　　他的工作，更多的是陪练。如果是别人，这个陪练就等于沙包。但轮到苏重，受训者反而成了沙包。
<br/>　　苏重每天的工作，就是把所有参与训练的人揍趴下。然后就什么也不管，自己闷在靶场练枪。
<br/>　　有技能面板辅助，又有海量子弹练习。苏重的枪械技能，终于不再停滞不前。
<br/>　　虽然依旧慢如龟爬，但好歹在进步。只是消耗的子弹越来越多。好在助教这份工作真的不错，所有资源全都被苏重淘换成各色子弹。竟勉强能支撑苏重枪械训练。
<br/>　　这一天，苏重像往常一样走出房间，准备到训练场去揍人。
<br/>　　可走出房门却发现，整个基地静悄悄。好似所有人都消失一般。
<br/>　　心中一动，苏重溜溜达达的走向基地大厅。这里是众人平日里叫交流用餐的地方。而那位傀儡大师拉斯·阿古，也经常在这里端坐冥思。
<br/>　　刚刚走近大厅，就听到了一阵噼里啪啦的打斗声。苏重脚步如猫，悄无声息走近。周围暗处围拢着不少影武者，都在看着中间正在打斗的两人。
<br/>　　与其说是打斗，不如说是杜卡单方面碾压。
<br/>　　尽管这个衣衫褴褛的白人，有着一手不错的格斗术。但动作太过死板，出手犹豫不决，看似凶狠，实则缺少杀气。加上体力衰弱，根本就不是杜卡的对手。三招两式就被杜卡放到。
<br/>　　这就是布鲁斯韦恩？这个世界的主角？
<br/>　　找了一根圆木柱子，斜斜靠在上面，苏重饶有兴趣的看戏。
<br/>　　布鲁斯·韦恩到来，意味着剧情的正式开启。意味着影武者联盟基地，距离毁灭已经不远。
<br/>　　而基地毁灭，正是苏重离开的大好时机。
<br/>　　一片混乱，少那么一两个人，谁也不会发现。离开这里，潜入茫茫人类社会，更是难以被发现。
<br/>　　既能脱离联盟，又能避免发生正面冲突。这是苏重一直在等待的机会。
