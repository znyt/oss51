　　“好。”
<br/>　　楚风眠点了点头，也不客气，一把将这仙石收下。
<br/>　　这些仙石，对楚风眠冲击半步天人境，有着大用。
<br/>　　有了这些仙石，楚风眠手中的仙石也有着二百枚左右了。
<br/>　　不过在收下了这些仙石的同时，楚风眠也是拿出了一枚空戒，递给了尹龙仙。
<br/>　　“这是？”
<br/>　　尹龙仙打开空戒，精粹的龙力，都是从空戒之中散发了出来。
<br/>　　一眼望去便是看见了其中足足三千枚升龙丹，摆放在了其中。
<br/>　　“这是你应得之物。”
<br/>　　楚风眠也不会白拿这尹龙仙的仙石，这三千枚升龙丹，是之前楚风眠就答应要给尹龙仙的。
<br/>　　“多谢楚公子。”
<br/>　　尹龙仙接过这空戒先是一愣，旋即大喜的说道。
<br/>　　她困在九劫古帝境界之中，想要突破几乎是无比艰难，现在得到了这些升龙丹，她突破境界，踏入半步天人境，几乎是板上钉钉的事情。
<br/>　　一旦是她踏入了半步天人境，未来在苍龙宗的地位便是就会截然不同，直接回成为老祖宗级别的人物，而不再是少主了。
<br/>　　就在这个时候，从众人的头顶之上，一道巨大无比的龙眼，突然浮现而出。
<br/>　　这正是苍龙宗的秘术，秘龙寻仙术。
<br/>　　“看来苍龙宗的人，还不死心。”
<br/>　　楚风眠目光看向这龙眼，嘴角露出几分冰冷之色道。
<br/>　　他之前放了他寰龙城主一次，也是让他去给苍龙宗提个醒，让苍龙宗不要招惹到楚风眠的头上来。
<br/>　　可现在看来，这苍龙宗还是没有死心，才吃了一个亏之后，还来找楚风眠的麻烦。
<br/>　　“这，我可以去宗门之中，去跟宗主解释。”
<br/>　　尹龙仙看向楚风眠，急忙的说道。
<br/>　　她可不想要楚风眠跟苍龙宗起冲突。
<br/>　　“怎么？楚公子跟苍龙宗有所恩怨？”
<br/>　　旁边的狂龙帝，听到这交谈，突然出声问道。
<br/>　　“不错，抽了一个所谓少主的龙筋，杀了几个金龙卫罢了，也不算什么恩怨。”
<br/>　　楚风眠随意的开口道，如同是这做得，只是一些微不足道的小事罢了。
<br/>　　旁边的海公主跟尹龙仙听到这话，倒是苦笑几声。
<br/>　　苍龙宗的少主，已经是弟子一代之中最为耀眼的人物了，绝世天才，楚风眠直接抽了那恒龙少主的龙筋，等于直接废了他。
<br/>　　这还算好，可后来楚风眠又是杀了五位金龙卫，这金龙卫代表的可是苍龙宗的威严。
<br/>　　楚风眠说杀就杀，简直是不将苍龙宗放在眼中。
<br/>　　现在苍龙宗要是善罢甘休的话，那未来可就没有任何的脸面了。
<br/>　　“的确不算什么大事，不过楚公子若是觉得麻烦的话，就让老朽去一趟苍龙宗吧。”
<br/>　　狂龙帝开口道，他的语气之中，都露出了几分冰冷。
<br/>　　“师尊，你要去苍龙宗？”
<br/>　　尹龙仙听到狂龙帝的话，吃惊的说道。
<br/>　　狂龙帝跟苍龙宗，其实的确有着几分关系，不过狂龙帝离开苍龙宗，已经是数万年前的事了，她在刚刚加入苍龙宗的时候，都未曾听说过狂龙帝。
<br/>　　现在狂龙帝居然是要出山，去苍龙宗了，这真的将自己摆在了奴仆的地位上，来为楚风眠做事了。
<br/>　　“既然苍龙宗的小家伙没有张眼睛招惹了楚公子，还是让老朽自己去解决吧，免得惹出更多的麻烦来。”
<br/>　　狂龙帝开口道。
<br/>　　他可是知晓楚风眠的恐怖，天地异种，同时掌控两种本源，别看现在楚风眠只是九劫古帝，一旦是楚风眠发怒，只怕是苍龙宗，都要死伤大半。
<br/>　　狂龙帝跟苍龙宗还有着一段关系，也不想要让苍龙宗毁于一旦，不如让他去解决。
<br/>　　“不用，既然苍龙宗要找我，那我自己去苍龙宗一趟罢了，免得日后还有麻烦。”
<br/>　　楚风眠语气冰冷的说道。
<br/>　　要是日后苍龙宗接二连三的还来找麻烦，楚风眠解决起来，也会不厌其烦，不如现在直接一口气解决了。
<br/>　　“那老朽就随公子去一趟吧。”
<br/>　　狂龙帝在一旁看到楚风眠表态，也是顺从的说道。
<br/>　　“走吧，去苍龙宗一趟，免得那些人一直来窥探于我。”
<br/>　　楚风眠看着尹龙仙说道。
<br/>　　尹龙仙听到楚风眠的话，心中虽然觉得几分不妙，可现在她也无法阻止，毕竟她只是一位少主，在苍龙宗的地位还不足以决定许多事。
<br/>　　她摆平不了，也只能够让楚风眠去解决了。
<br/>　　“还请楚公子，手下留情。”
<br/>　　尹龙仙小声的说道。
<br/>　　“好。”
<br/>　　楚风眠点了点头道。
<br/>　　“走吧。”

<br/>　　他恨不得要将楚风眠剥皮抽筋，才能化解心中的愤怒。
