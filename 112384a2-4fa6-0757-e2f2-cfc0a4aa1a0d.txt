　　“嗯？”
<br/>　　那武者，收起手中妖兽内丹后，显然注意到了不远处的萧逸等人。
<br/>　　“做什么？”武者面露忌惮之色。
<br/>　　在外行走，有这等警惕很正常。
<br/>　　但这抹警惕之色，很快就化作了寻常。
<br/>　　“幻光险地的规矩，你们该很清楚，我劝你们别打我主意。”
<br/>　　说罢，武者闪身而离。
<br/>　　原地，萧逸愣了愣，随后反应了过来。
<br/>　　想来，这武者是怕他们杀人越货。
<br/>　　不过，幻光险地的规矩？
<br/>　　三人继续前行，方向始终是往深处而去，不过速度并不快罢了。
<br/>　　路上，也不时遇到其他武者。
<br/>　　幻光险地这里，虽不如别的险地，动辄一队队武者，但来此历练的武者亦不在少数。
<br/>　　起码算不得稀少的地步。
<br/>　　一路上，萧逸虽没有特地拦下一个武者来询问。
<br/>　　但旁敲侧击，通过这些武者的话，三人也大概知晓了这幻光险地的事。
<br/>　　“难怪进入幻光险地，需要得到幻光府的手谕。”萧逸自语一声，摇了摇头。
<br/>　　那所谓的规矩，便是幻光府定下的规矩。
<br/>　　进入幻光险地历练的武者，互相不得厮杀。
<br/>　　另外，在这里所得的妖兽内丹等战利品，出去后，需要上缴幻光府一半。
<br/>　　所谓去拿手谕，便是去幻光府做下记录。
<br/>　　有了这些记录，幻光府便知晓有多少武者进入了幻光险地。
<br/>　　这些武者离开幻光险地后，还需受幻光府武者的搜查。
<br/>　　“幻光府，有够霸道的。”公孙火舞愤愤地说道。
<br/>　　路上，萧逸随手杀了一头妖兽。
<br/>　　取了内丹后，感知一番，面露惊色。
<br/>　　果然，这里的妖兽，有些特殊。
<br/>　　妖兽内丹上，拥有武者魂师的魂力。
<br/>　　萧逸取出一些天材地宝，配合内丹，炼制成丹药。
<br/>　　吞服后，一股莫名的气息在他身上爆发。
<br/>　　“原来如此，魂力增长了。”萧逸面露恍然之色。
<br/>　　这里的妖兽内丹，就如金光险地那里的妖兽内丹一样，珍贵无比。
<br/>　　不同的是，金光险地那里的妖兽内丹，炼制而出的，是金光丹。
<br/>　　金光丹，是提升武者实力的丹药。
<br/>　　而这里的妖兽内丹，炼制而出的，是增加武者魂力的丹药。
<br/>　　更准确来说，是增加魂师的魂力。
<br/>　　如同正常武者，一身元力，乃是修为的象征。
<br/>　　对于魂师来说，一身魂力，也是修为的象征。
<br/>　　更简单来说，这里的妖兽内丹，炼制成丹药后，对魂师而言，绝对是极好的修炼之物。
<br/>　　难怪幻光府如此霸道，将整个幻光险地都据为己有。
<br/>　　还大费周章定下各种规矩。
<br/>　　进幻光险地历练的武者，还有另外一条规矩。
<br/>　　那便是炼药师和修习火之一道的武者，需要在幻光府接受更严格的登记和搜查。
<br/>　　对于正常武者来说，进入幻光险地历练，即便斩杀了妖兽，也无法立刻使用内丹。
<br/>　　妖兽内丹，武者无法直接服用吸收，需要炼制成丹药后，方能使用。
<br/>　　而炼药不同，进来险地后，直接斩杀妖兽，马上就能炼制成丹药使用了。
<br/>　　故炼药师和修习火之一道的武者，之后离开险地，需要上缴更加大量的妖兽内丹。
<br/>　　“我们十八府，各府都不曾如此占据一个险地；倒是幻光府，此举有些霸道了。”
<br/>　　顾怜星摇了摇头。
<br/>　　萧逸脸色淡漠，“这里的魂丹，毕竟珍贵无比，幻光府作为这附近的霸主势力，自然不会吃亏。”
<br/>　　这里的妖兽，称作魂兽，拥有魂力，体内内丹，亦称作魂丹。
<br/>　　说起来，魂兽，作为妖兽的一个分支；亦如魂师，乃是无数武者中的一份职业，稀有无比。
<br/>　　魂师，以手段诡异莫测著称。
<br/>　　魂兽，同样如此。
<br/>　　魂兽，作为妖兽的一个分支，同样稀有。
<br/>　　就萧逸来说，他来中域闯荡过的各大险地中，都未曾遇到过魂兽。
<br/>　　但这不代表只有幻光险地这里才拥有魂兽。
<br/>　　偌大个中域，别的一些特殊险地，也会有魂兽。
<br/>　　只不过他没遇到，而且魂兽栖息生存的地方有些特殊罢了。
<br/>　　“走吧，继续前往深处。”萧逸说了一声。
<br/>　　三人继续前行。
<br/>　　这幻光险地内，必然也会存在禁地洞府。
<br/>　　且这里的幻光异象，也必然是源于禁地洞府。
<br/>　　禁地洞府，不可能会在险地的外头，只可能在深处。
<br/>　　故无论如何，三人还是要前往幻光险地深处。
<br/>　　事实上，这里的妖兽内丹，如此特殊；如果有可能的话，萧逸也打算在这里历练一段不短的时间。
<br/>　　毕竟他也算是魂师。
<br/>　　以前他很少修习魂师上的实力，主要原因在于魂师的修炼比较麻烦。
<br/>　　且他的其他底牌，还有其他武道方面，有更好的提升手段。
<br/>　　而现在，这里的魂兽内丹，乃是极好的魂师修炼之物。
<br/>　　加之他的控火兽武魂已踏入紫色阶品，施展魂师手段时，威力也会极强。
<br/>　　现在修习魂师一道，是再好不过了。
<br/>　　不过，现在最重要的，还是救治顾怜星之事；毕竟顾怜星剩下的时间，不多了。
<br/>　　他也只能选择日后有时间，再回来这幻光险地历练一番，届时，必有不错的收获。
<br/>　　想罢，萧逸收敛了心神，继续前行。
<br/>　　不多时，三人终于来到险地深处。
<br/>　　萧逸也很快便遇到了一头圣皇境以上的魂兽。
<br/>　　轰…
<br/>　　一股无形的力量，陡然席卷而来。
<br/>　　那是魂力风暴，诡异至极，而又威力惊人。
<br/>　　萧逸连忙打出数道星光力量，护住了顾怜星二人。
<br/>　　随后冷焱剑凭空而现，挡下了这股魂力风暴。
<br/>　　魂兽也罢，魂力也罢，诡异是诡异，但不代表修习别的武道的武者就无法对付。
<br/>　　魂力同样是力量的一种。
<br/>　　若是正常武者，以元力武技等等，同样能斩杀魂兽，不过是比较麻烦罢了。
<br/>　　魂力的无形，往往让得正常武者措手不及。
<br/>　　当然了，以萧逸实力，要斩杀面前这头圣皇境妖兽，轻而易举罢了。
<br/>　　不过，他倒想借此机会，锻炼一下自己的魂力掌控。
<br/>　　反正距离十五月圆，还有两天时间。
<br/>　　这两天，便权当历练一番了。
<br/>　　轰…
<br/>　　一股无形力量，从萧逸身上爆发。
<br/>　　同样席卷而出，与面前魂兽打出的魂力风暴，剧烈碰撞了起来。
