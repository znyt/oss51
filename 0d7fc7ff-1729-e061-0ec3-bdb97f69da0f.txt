　　其余人继续前进，一路斩杀，自己这边，也不断有人死在阵法之中，一时间精魄飞舞，掺杂在花海之间，使整个场面看上去更加华丽而血腥，惨叫声此起彼伏……对在场大部分小门派的弟子来说，他们可能一生都没见过这么多人的大战，简直触目惊心，一个个都惊骇地不敢出声。
<br/>　　大战，说开始就开始了啊。
<br/>　　叶少阳深吸一口气，跟大伙一起继续观战。
<br/>　　在法术公会的人上到一半的时候，守阵的那些桃花山弟子也分出了一半，挡住了这些人，双方厮杀起来，因为漫天飞舞的桃花和精魄，阻挡了众人的视线，只能看到里面血肉横飞，惨叫声迭起不停……
<br/>　　“你们六个，上，去抓那个邪魔！”
<br/>　　鬼姨一声令下，除了张晓寒之外，其余梧桐等人，飞升入阵……
<br/>　　叶少阳立刻从人群中走出去，来到峰下，紧张地望着梧桐的背影。
<br/>　　他不想参与这双方的战斗，更不要说还有陈晓宇这样的人在里头，想让自己帮法术公会，那是绝无可能，但他很是关心梧桐的安危，如果梧桐真的遇到什么危险，他肯定是要挺身而出的。
<br/>　　“你在干什么？”张晓寒盯着叶少阳，发现他一直盯着梧桐的背影，看穿了他的想法，嘲讽道：“凭你，难道还想保护梧桐。”
<br/>　　叶少阳转头看了他一眼，道：“你为什么不去？”
<br/>　　“强者，总是最后出场。”张晓寒淡淡说道。
<br/>　　叶少阳冷笑了一声。
<br/>　　梧桐等人很快冲到半山腰，因为那二十来个人的阻挡，这六个人根本没有停留，直接越过去，朝山顶那个蓝衣人奔去……
<br/>　　真正的战斗，就要开始了。
<br/>　　所有人都屏住了呼吸，睁大眼睛朝峰顶望去。在他们心中，既然法术公会出动了这么多强者前来围剿，这蓝衣人输肯定是输定了，他们只是想看他能坚持多久。
<br/>　　在这一行六人快要爬上山顶的时候，那个蓝衣人总算站了起来，抬起右手，一道银光从手中溢出，形成了一把光剑。
<br/>　　难道真的是他……叶少阳心中更加充满了疑惑。
<br/>　　这个时候，梧桐等人已经来到蓝衣人身边，立刻抢攻而上。
<br/>　　蓝衣人不急不慌，挥剑劈砍，一道巨大的剑光，对着六人兜头砸下去，然后散开，成为一个道纹，一下子就挡住了六人的进攻。甚至将几人击退了几步。
<br/>　　“不会吧！”
<br/>　　“怎么可能！”
<br/>　　人群中爆发出声声惊叹。以一敌六……这些人做梦也没想到，这蓝衣人居然能将他们生生逼退。
<br/>　　“真是不可思议，怪不得不怕法术公会闯山，居然有这等实力！”毛小方在叶少阳身边惊叹。
<br/>　　梧桐等人也是吃了一惊，互相看了下，再度冲上去，这一次将蓝衣人团团围住，展开围攻，一时间无数道灵力、符印、法器，一起朝蓝衣人身上招呼。
<br/>　　蓝衣人双手放在胸前，一上一下地虚握，周身泛起一道紫气，以自己为中心，形成了一道鸡蛋形状的结界，在中间岿然不动，所有进攻的手段，都被结界挡在外面。
<br/>　　“你到底是谁？”梧桐近距离打量蓝衣人，忍不住问道。
<br/>　　蓝衣人一言不发，操控结界，气定神闲地抵抗着众人的攻势。
<br/>　　“怎么办？”有人问道。他们能明确感觉到这一道结界的强大，就算是集合他们六个人的力量，也根本无法突破结界。
<br/>　　眼前这个蓝衣人的强大，远远超过了他们的预想。
<br/>　　“你们不是对手，先下山来！”鬼姨也看出端倪，大声喊道。
<br/>　　六人刚要动身，结果突然从蓝衣人的身后，突然飞出了一朵花，花叶尖尖，白里透红，看上去像是一株含苞未放的莲花。
<br/>　　莲花飞到蓝衣人头顶上空，突然绽放，果然是一朵莲花，花瓣展开的时候，无数灵光溢出，如果水波一样朝四周飞快到荡漾开来，将六人裹在中间。六人立刻觉得行动困难，骇然失色，本来他们是在包围蓝衣人，没想到自己反而被包围了！
<br/>　　“呵呵呵……”一道娇媚的笑声，在耳边回荡，六人环顾左右，却是找不到人在哪里，正在惶恐之中。
<br/>　　从峰顶的小亭的顶部，飞下来一道清丽的人影，手提一方好像钟鼎的东西，面带邪魅的笑容，朝他们缓缓走来。
<br/>　　“碧清……”梧桐一下子认出了对方，正是不久之前在那座古墓里见过的那个强大的邪物，万万没有想到会在这里遇到她，心中骇然不已。
<br/>　　“妹子，又见面了。我可以最后一个杀你。”碧清说着，飞身入阵，手里提着那口钟鼎，将坛口对着梧桐边上一个师兄，口中念咒，一道黑气，化成两只巨大的手，从两边抓住这人的肩膀，朝坛子里面拉去。
<br/>　　炼尸缸！
<br/>　　梧桐顿时回过神来，招呼大家一起上。
<br/>　　这时候蓝衣人出手，一道结界，从地上刷的一下立起来，将五人都挡在了后面。
<br/>　　人有三灯，分别在头顶和两边肩膀上。那黑气所化的两只举手，分别抓住了这人的地魂和人魂，用力朝炼尸缸里拉。
<br/>　　这人是梧桐的师兄，也是星月奴在人间选出的得意弟子之一，名叫崔晓明，实力自然不弱，也是个准地仙，自己双肩被抓住之后，立刻感到魂魄要被拉扯出去，急忙念咒定魂，苦苦抵挡。
<br/>　　然而，那两只手所代表的炼尸缸的力量实在太强，任凭他死命抵抗，最后还是有一道虚影——他的魂魄，被从身体里生生拉了出去，眼看着就要完全脱离身体，碧清的嘴角露出一丝胜利的微笑。
<br/>　　就在这时，崔晓明突然大喝一声，如一道晴天霹雳，即将被从他体内拉出来的鬼魂突然碎裂，化作精魄……
<br/>　　众人震惊，然后立刻明白发生了什么事：为了魂魄不被碧清所掌控，崔晓明将自己的魂魄震碎了。
<br/>　　惨烈的牺牲……
<br/>　　“师兄！”
<br/>　　“师弟！”
<br/>　　梧桐等人发出悲愤的吼叫，一起发力，震碎了结界，朝碧清扑过去……
