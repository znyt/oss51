        &lt;h3&gt;第三卷第一百九十二章  泉城上湾&lt;/h3&gt;
<br/>
<br/>        陈虞和落妍有些咳嗽，陈子昂就没有回到父母亲身边，原本想着将父母亲也接到北京，但是他们不愿意。
<br/>
<br/>        俩个老人的心思其实大家都知道，二哥陈子建离婚了，连个家都没有，如果父母亲都被接走了，二哥的处境就尴尬了。
<br/>
<br/>        陈子昂没有回老家，七哥陈子霖带着老婆娃回老家了，这几年陈子霖在西城发展，将老婆和孩子都接到了西城，老婆有时候两头跑，回去照顾上两天老人，再回来照顾孩子，她也已经学会了开车，从西城到泉城上湾也就三个多小时。
<br/>
<br/>        继火车开通之后，高速路也相继开通了，他们现在回家最多三个小时，国家大力发展民生，强国富民，他们那里原本住在高山里的人们都被国家安置了，每家每户都住上了宽敞明亮的高楼，用上了自来水，这就是伟大的移民搬迁工程，农民也已经休耕罢种，退耕还林，都过上了好日子，再也没有十几年前的吃不饱穿不暖的时光了。
<br/>
<br/>        七嫂有时候很内疚的，当初子昂上学的时候没有好好对子昂，其实那也不怪她，她确实是没有钱，如果有钱她也不会那么做的。
<br/>
<br/>        如今他们在西城吃什么没有呢？别说供一个大学生上学，就是供十个大学生也不在话下，而且家里的生意虽然小，却也是红红火火，将上弯镇子上的人们都带动起来了，他们的土特产已经卖到了网上，在西城各个旅游景点都有超市，日子过得也算不错了。
<br/>
<br/>        放了寒假她就回了老家，将家里收拾的焕然一新，等着兄弟姐妹们回家过年，当初盖房子的时候子霖一心要贷款盖上这栋小洋楼，如今她才明白子霖的用意，他们兄弟姊妹多，要是真的都回家了必然都会拖家带口，你说兄妹这么多人没住的地方可怎么行？
<br/>
<br/>        这个新年子昂会带孩子一起回来，二哥这一年多在家里也帮了很多忙，已经给家里都装上了暖气，不过都是自己烧的那种，这个在乡下已经是非常好的了，现在乡下取暖都是烤炉子。
<br/>
<br/>        陈子霖一个人从北京回来了，带回来很多的东西，都是子昂捎回来的，说是子昂不回来，俩个孩子感冒了，父亲和母亲有些失落，不过好的是大姐，二哥还有四姐五姐会回来，三姐和三姐夫只能在医院过年了。
<br/>
<br/>        父母亲已经快八十岁了，不过农村的老人经常干活，身体还算是硬朗，眼不花耳不聋的，也不糊涂，将家里打理得也很好，本来陈子昂在西城买了房子，都已经装修好了，让俩个老人去西城，但是老人在西城呆不习惯，呆了一个礼拜就回去了，再怎么叫都不去西城。
<br/>
<br/>        年夜饭的时候，父亲看着孩子们，叹了口气：“你们小的时候盼着你们长大，长大了又盼着你们回家。人啊，总是这么矛盾的。”
<br/>
<br/>        母亲：“看着你们都成家立业了，心里是高兴的啊，原本想着一个个的有了出息，我这颗心啊，也算是能放下了，不曾想啊，子建离婚，子澜出事了，老六还是不知所踪。子昂带着俩个孩子也没有着落，我这颗心又吊起来了。”
<br/>
<br/>        “妈，你也不要担心，我二哥离开冯睿那个女人是他的福气，不信你看，冯睿后期是不会有好下场的，我三姐也就是不小心出了车祸，也不是什么大问题，修养一段时间就好了，子昂你就更不用担心了，现在的子昂那能是一般人？至于六哥，你也不用操心，我们都在找着，说不定过一两年就给你带回来一个帅气的六哥。”陈子霖安慰着妈妈。
<br/>
<br/>        陈子霖知道陈子昂年后是要去甸城的，有可能甸城的谷强就是六哥，如果六哥找到了，他们家里就可以真正的团圆了。
<br/>
<br/>        “爸，我有件事情一直都没有给你说，关于手镯的事情，子昂有个朋友有另外的一只手镯，只是......”
<br/>
<br/>        “只是怎么了？”
<br/>
<br/>        “只是冯睿将我们家的那种手镯送人了。”陈子建难过的说。
<br/>
<br/>        “你......你.....咳咳咳。”父亲气急了指着陈子建说不出来话。
<br/>
<br/>        “老头子，你别急，你别急。”母亲颤巍巍的用手拍着父亲的背。
<br/>
<br/>        父亲捂住嘴艰难的站起来朝外面走去。
<br/>
<br/>        “爸！”大家齐声喊。
<br/>
<br/>        “谁也不要跟着我。”父亲怒声喊。
<br/>
<br/>        “老头子，老头子。”母亲扶着父亲朝门外走去。
<br/>
<br/>        “子建，你看看你大过年的，你说这事情干嘛啊。”子惠埋怨陈子建。
<br/>
<br/>        “我，我一只憋到现在，就是看大家回来了我才说，平时我不敢说。”陈子建悔恨不已，他这一生都做了一些什么事情啊。
<br/>
<br/>        “也别埋怨二哥了，知人知面不知心，当初谁也不知道冯睿是这样的人，手镯丢了再想其他办法呗。”陈子乔劝解着。
<br/>
<br/>        “既然子昂的朋友知道，改天我问问子昂，看子昂能不能想办法找到那种手镯的消息，如果能找到我们也可以去看看，也不一定姑姑呢。”陈子霖也说话了。
<br/>
<br/>        陈子悦：“现在的信息这么发达，如果真的知道那只手镯的主人，花点钱做个dan就行了。”
<br/>
<br/>        陈子悦一家四口一起回来的，他们大年初一就得回西城。
<br/>
<br/>        母亲扶着父亲走到卫生间：“老头子，你怎么了。”
<br/>
<br/>        父亲拿下捂着嘴的手摊开了手心，手心里是一口红艳艳的血。
<br/>
<br/>        母亲：“老头子，你......”
<br/>
<br/>        父亲：“别说话，也别说出去，大过年的。”
<br/>
<br/>        母亲：“你是不是哪里不舒服？”
<br/>
<br/>        父亲：“子建气的，你知道那个手镯吗？在我们家里最无助的时候我的爸爸妈妈都没有卖掉，他们宁愿饿死都不愿意将手镯挡掉，就是想要找到阿凉啊。”
<br/>
<br/>        母亲：“那为什么你后来不回老家去找？”
<br/>
<br/>        父亲：“我怎么没有去找，我找了很多次，可是谁也不知道代家搬到哪里去了。”
<br/>
<br/>        母亲：“你现在也不要着急，不是有了另外一只的消息了吗？也许很快就能找到。”
<br/>
<br/>        父亲：“罢了，出去吧。”
<br/>
<br/>        母亲：“可是你这身体......”
<br/>
<br/>        父亲：“一时半会死不了。”
<br/>
<br/>        母亲：“你说的什么话呢，你.....”
<br/>
<br/>        父亲：“阿莲，我告诉你啊，你不要怪我隐瞒着你，我们老六干大事情去了。”
<br/>
<br/>        母亲：“干什么大事情？”
<br/>
<br/>        父亲：“几年前，有个叫蒋靖的来我们家，告诉我老六干大事情去了，还将老六的工作证给我看了，所以这几年我一直都没有让他们去找他，另外啊，老六做的事情很危险，也是为了保护这几个孩子和老六吧，我一直都没有告诉你，你要记得你家老六是值得你骄傲的，就算是老六不回来，这件事情你也要压在心底，谁也不能说出去。”
<br/>
<br/>        母亲哭着说：“老头子，你瞒得我好苦，我还以为老六不在了。”
<br/>
<br/>        父亲：“好了，别哭了，你要知道你这些孩子都是好样了，我很感谢你辛辛苦苦为我老陈家生了这么多优秀的孩子，这一生辛苦你了，来，我们出去和孩子们在一起吧。”
<br/>
<br/>        父亲和母亲手拉手走出了卫生间，他们的银丝在灯光下闪着银白色的光。
