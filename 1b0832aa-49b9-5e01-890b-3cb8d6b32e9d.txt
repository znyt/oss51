<br/>        “十爷没说理由吗？”童筱颖直接无视掉季雅姗的话，捡重点问。
<br/>
<br/>        季雅姗仍旧摇头，哭诉道：“他说，以后我不必再见他了……”
<br/>
<br/>        童筱颖听到这里，沉默了。她心想着，或许是跟婴宁有关。

<br/>        但是，婴宁已经死了……
<br/>
<br/>        十爷是突然觉得愧对婴宁吗？

<br/>        童筱颖刚想拒绝，嘴巴还未张口，季雅姗便伸出食指来，贴在了童筱颖的粉唇上。

<br/>        季雅姗咧嘴一笑，拿起自己刚刚那瓶未喝完的酒，与童筱颖碰了碰瓶子：“不醉不休！”

<br/>        其中一个“坏男人”就是季雅姗的同父异母的哥哥季运恒。

<br/>        季运恒身旁的狐朋狗友里面，有两个男人对季雅姗感兴趣，而他作为哥哥，虽然是同父异母，但是已经警告了身旁的人，不许打季雅姗的主意。
<br/>
<br/>        至于季雅姗身边的另外一个女人……
<br/>
<br/>        季运恒觉得听她的声音很耳熟。
<br/>
<br/>        小谢？！

<br/>        听说，小谢被“夜惑”的股东之一给莫名其妙地开除，所以，小谢早已不在“夜惑”里唱歌。

<br/>        他现在好不容易治好了命根子，却没有“美人”也很是令他头疼。
<br/>
<br/>        邻座两边的包厢里，一起八个男人在等中间这VIP雅座里的童筱颖和季雅姗喝醉酒，他们好捡便宜。
<br/>
<br/>        季雅姗已经喝了两瓶了，只是脸部涨得通红，双眼迷离。
<br/>
<br/>        童筱颖一瓶下去，直接喝晕了过去，倒在了沙发上一动不动了。
<br/>
<br/>        季雅姗往童筱颖身旁挪了挪，伸出手来，有气无力地摇了摇童筱颖，咧嘴一笑，打了声“嗝”，索性也趴在童筱颖身上，依偎着她睡了过去。
<br/>
<br/>        当两边的男人伸头探望，确定中间这VIP包厢里的两个女人喝醉了后，纷纷起身，准备下手时，岂料过道里突然窜出四个黑衣保镖，将他们欲要行动的计划给硬生生地逼停。

<br/>        没过多久，一个身形修长的男人从过道暗影中走了出来。
<br/>
<br/>        季运恒看到那个男人后，下意识地往雅座后躲。
<br/>
<br/>        那个男人是叫“连少”吧！
<br/>
<br/>        他每次遇到他，他就准倒霉。

<br/>        傅夜沉的气场，总是会在无形中让身旁的男人自惭形秽。
