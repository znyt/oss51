　　唐清雅想了想，或许也知道，自己肯定不是对方的对手，于是一咬牙，点了点头，道：
<br/>　　“好！我可以你跟你走，但是你不能伤害我的室友！”
<br/>　　那黑影再次桀桀一笑，
<br/>　　“没问题！他们只是中了我的幻术，暂时陷入梦境之中，等明天天一亮，自然就会醒来！”
<br/>　　唐清雅听到这话，松了一口气，
<br/>　　“好，我跟你走！”
<br/>　　说完这话，唐清雅双手在床上一按，然后整个人轻飘飘的落在了地上。
<br/>　　她之前并没有入睡，所以衣服还没有脱掉。
<br/>　　“走吧！”
<br/>　　那黑影猛然伸出手掌，一把抓住唐清雅的手腕。
<br/>　　下一刻，
<br/>　　两人已经从窗户窜出，跨越数十米，落在女生宿舍楼前的一片草地之上。
<br/>　　然而，
<br/>　　就在两人落地的那一刹那，周围的环境，忽然一下子变了！
<br/>　　四周的建筑竟然消失不见，两人仿佛进入了一个封闭的空间之中。
<br/>　　那黑影顿时全身一震，
<br/>　　“空间禁制！什么人？出来！”
<br/>　　轰！
<br/>　　黑影的声音才刚刚落下，忽然感觉眼前一花，跟前已经出现了一道人影，朝他一掌拍了过来！
<br/>　　“咦！”
<br/>　　那黑衣顿时惊呼了一声，显然对来人的实力颇为意外，当即一把将旁边的唐清雅推了出去，然后整个人迅速暴退到了数十米外。
<br/>　　唐清雅感觉到自己，被一股强悍无比的掌力所包裹，似乎可以轻易将自己活生生撕碎！
<br/>　　唐清雅心头顿时升起一股绝望，
<br/>　　“我这就要死了吗？好不甘心啊！”
<br/>　　这道念头才刚刚升起，唐清雅忽然有感觉那股强悍的力量，一下子消失的无影无踪了，整个人一下子便栽倒在那道突然出现的人影怀中。
<br/>　　“嗯？这感觉……好熟悉！”
<br/>　　唐清雅连忙抬头一看，顿时便看到了一张无比熟悉的笑脸。
<br/>　　“清雅，你没事吧？”
<br/>　　此人自然正是叶尘！
<br/>　　唐清雅先是一愣，继而趴在叶尘怀里大哭起来。
<br/>　　“臭叶尘！我还以为你早把我给忘了呢！哼哼！”
<br/>　　叶尘顿时有些尴尬，
<br/>　　他这阵子只顾着曦月，还真把唐清雅完全抛到脑后了，若非唐清山及时把情况告诉了他，说不定此刻，唐清雅已经被人掳走了！
<br/>　　另外一边，
<br/>　　那道黑影并没有逃走，而是静静的观察对叶尘观察起来。
<br/>　　达到他们这个层次，黑夜几乎和白天没有什么两样。
<br/>　　那人在观察叶尘的同时，叶尘也在观察着他，
<br/>　　这是一名约莫头发灰白的老者，满脸的阴鸷，一看就知道是心思歹毒之辈。
<br/>　　那老者观察了叶尘片刻，顿时桀桀一笑，
<br/>　　“你就是最近，在世俗界搞出大动静的那个叶狂仙吧？没想到你比我想象的还要年轻！居然是一个毛头小子！桀桀！”
<br/>　　叶尘冷冷一笑，
<br/>　　“有志不在年高，无志空活百年！像你这样的废物，年纪再大，也只是一只老废物罢了！只会干一些见不得人的下流手段！”
<br/>　　那老者听到叶尘这话，顿时气得肺都要炸开了，
<br/>　　“小子！你果然和传言的一样，够狂！”
<br/>　　“可惜，狂妄的人，大多短命，你也不会例外！”
<br/>　　叶尘将怀里的唐清雅缓缓推开，同时拍了拍有些褶皱的衣衫，淡淡道：
<br/>　　“我会不会短命，我不知道，可是我却知道，你今天肯定活不了！”
<br/>　　那老者两眼一瞪，几乎要喷出火来，
<br/>　　“狂妄的小子！那就让本座来试试，你有几斤几两！”
<br/>　　说完这话，那老者猛然遥空一拳砸出！
<br/>　　轰隆！
<br/>　　拳劲如涛，犹如掀起了惊涛骇浪，铺天盖地朝叶尘两人拍落了下来。
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，随手一掌拍出，
<br/>　　“雕虫小技！”
<br/>　　彭！
<br/>　　那看似气势如虹的磅礴掌力，竟然被叶尘轻飘飘的一掌，瞬间击退！
<br/>　　与此同时，
<br/>　　蹬蹬蹬！
<br/>　　那老者还感受到一股极强的反弹之力，竟然将他震得连连倒退！
<br/>　　“什么！你竟然——已经做到，将空间法则，融入掌力之中？这怎么可能！”
<br/>　　要知道，便是元婴老祖级别的强者，能将法则之力融入掌力和拳劲之中，那也是凤毛麟角的存在！
<br/>　　一般只有那传说中的化神天君，才有可能调动天地之力。
<br/>　　可是眼前的叶尘，有明显不是化神天君，甚至老者可以确信，他极有可能连金丹境都还没有达到！
<br/>　　要知道，他可是金丹巅峰之境，里元婴老祖境界，也仅有一步之遥而已。
<br/>　　居然被对方一掌击退！
<br/>　　“这他娘的，到底是个什么怪物？”
<br/>　　老者越想越觉得不安，不自觉的已经生出了几分退意。
<br/>　　可惜，他想走，叶尘又岂会任由他离开？
<br/>　　“你也来接我一拳试试！”
<br/>　　叶尘脚下步子不动，同样遥空一拳轰出！
<br/>　　“虚空神拳！”
<br/>　　轰隆！
<br/>　　拳劲依旧是轻飘飘的，似乎好无任何力道，与之前老者那磅礴的掌力，简直有天壤之别。
<br/>　　然而，
<br/>　　老者见状，却猛然瞳孔一缩，一脸的骇人之色，
<br/>　　“不好！”
<br/>　　老者意识到对方这一拳的可怕，慌忙飞速倒退，瞬间便退到了百米之外。
<br/>　　然而下一刻，
<br/>　　老者却还是清晰无比的感觉到，那道恐怖的拳劲，竟然穿梭空间，出现在了他的身后！
<br/>　　如此一来，就等同于，他主动撞在了那道恐怖的拳劲之上！
<br/>　　彭！
<br/>　　几乎没有任何悬念，
<br/>　　老者犹如被一把巨锤砸中，直接便倒飞而出，同时狂喷了一口鲜血！
<br/>　　“逃！”
<br/>　　老者又惊又骇，顿时便再也没有了继续打下去的勇气，慌忙便想要从地上爬起，迅速逃命！
<br/>　　然而这时，背上猛然一沉，却已经被叶尘一脚踩在了他的背上。
<br/>　　“噗！”
<br/>　　老者顿时再次狂喷了一口鲜血，任凭他如何拼命的挣扎，却根本无法从叶尘的脚下逃脱。
<br/>　　与此同时，
<br/>　　叶尘那淡漠的声音，也再次响起，
<br/>　　“老东西！我说过，你今天非死不可！”
<br/>　　以叶尘如今的实力，连等同于元婴境后期的魔皇，都可也斩杀，区区一个连元婴境都不是的废物，对叶尘来说，简直就像捏死一只蚂蚁一样简单！
<br/>　　直到此时，那老者终于慌了，
<br/>　　“不！你不能杀我！你若是杀了我！药王宗是绝对不会放过你的！”
<br/>　　“而且，你在世俗界搞出了那么大的动静，已经违背了仙、神、妖、魔四界当初的约定，如今你已经是四界公敌了！”
