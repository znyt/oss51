　　噗通！
<br/>　　眉心中弹的反政府武装势力指挥官跪倒在地上，身体向前扑倒在干燥的街道上。
<br/>　　献血从他眉心的弹孔中汩汩的往外冒，很快就在他头下形成了一滩血泊。
<br/>　　堕天使手中两把订制的银色MK23手枪，在他手中舞起绚丽的枪花，在他撩起身上风衣的一瞬间，两把银色手枪落入他大腿两侧的枪套中，看向四个K国学员躲藏的破屋，说道：“里面的四个人出来吧！”
<br/>　　Marcus、Alicia、Kim三个人从破屋中走了出来，Riggs因为伤势的关系，无法移动，只能待在破屋中。
<br/>　　牛仔帽下，堕天使那双仿佛能看穿人灵魂的眼睛在三个人身上扫过，问道：“你们应该有四个人，还有一个呢？”
<br/>　　“他受了很重的上，不方便移动。”Marcus回答道。
<br/>　　“我是堕天使，‘暗月十二主天使’之一。”堕天使朝着破屋走去，在从Marcus、Alicia和Kim三个人身边经过的时候，对三个人问道：“想报仇吗？”
<br/>　　Marcus、Alicia、Kim闻言，不仅是身体，就连灵魂都在震动。报仇！？这是什么意思！？
<br/>　　很快，Marcus明白了，Alicia和Kim也想通了。
<br/>　　破雾中，Riggs躺在杂乱不堪的地上，他身上有三处枪伤，右手臂也被破片削去一大块皮肉，因为缺少医疗用品，缠裹在伤口上的破布脏污不堪；因为缺少医疗用品，Riggs的伤口已经开始化脓并且发生感染，伤口散发出一股股伤口腐烂后产生的恶臭。
<br/>　　即便是受了这样的伤，Riggs还没有死，不得不说，他的命很大。如果不是因为如此，堕天使也不会出现，并且愿意给四个人复仇的机会。
<br/>　　“他，怎么样？”堕天使对跟着自己进入破屋的长袍男问道。
<br/>　　“很重。”
<br/>　　长袍男闷声回答道：“他的这条手臂保不住了，伤口溃烂诱发感染，如果不做截肢处理，他会没命。”
<br/>　　“把他带走，进行截肢手术。”
<br/>　　堕天使留下一句话，转身离开了这间弥散着伤口腐烂恶臭的破屋，回到街道上。比起破屋里的腐烂恶臭，堕天使更喜欢街道上的血腥味。
<br/>　　“想知道你们为什么会被丢弃在‘无政府地带’吗？我可以给你们答案。”堕天使看着自己面前的Marcus、Alicia和Kim三人，说道：“我可以给你们最好的治疗，不亚于‘炼狱’的训练，先进的武器装备和优秀的战斗人员，我可以给你们复仇的机会，向‘炼狱’复仇的机会！但是你们必须把自己献给‘暗月’，献给我，从此以后，你们就是我堕天使手中杀人的刀！”
<br/>　　Marcus、Alicia、Kim和Riggs短暂的犹豫后，答应了堕天使。他们还有选择吗？没有！
<br/>　　因为他们知道，如果自己拒绝，堕天使和长袍男会立刻离开，然后在十几分钟后，人数更多的反政府武装人员会感到这里，他们会死，而且会死的很惨。
<br/>　　四个人跟着堕天使和长袍男离开了。堕天使把他们带离了“无政府地带”把他们送去接受治疗，从点头答应堕天使的那一刻起，他们的身份就成了堕天使手里的刀，用来杀人的刀。
<br/>　　欧洲，某私人医院。
<br/>　　这家私人医院跟“暗月”有直接联系，是“暗月”经济收益渠道之一。
<br/>　　在这里能接受世界一流的医疗服务，但同样的，高昂的价格不是一般人所能承受的。
<br/>　　住院大楼楼顶，堕天使换了一身全手工订制的西装，头上戴着一顶礼帽，胸前西装口袋中插着一支艳丽的红玫瑰，两侧腋下枪套中装着那两把银色MK23手枪，两把银色MK23手枪在空气中闪着银光。
<br/>　　“你是不是感到很不理解，为什么我会亲自去把那四个人带到这里来医治？”堕天使对站在自己身后的长袍男，问道。
<br/>　　长袍男依旧用长袍裹着自己，遮住了头，那把长柄镰刀用一块灰色的亚麻布包着，被长袍男反手背在身后。
<br/>　　“是！”
<br/>　　长袍男闷声回答。
<br/>　　“仇恨。”堕天使回答道：“他们心中的仇恨会让他们变得强大、变得疯狂，变成杀人机器。我在他们心中种下了复仇的种子，早晚这复仇的种子会发芽，会开出艳丽的仇恨之花，就跟这朵玫瑰一样艳丽。”
<br/>　　堕天使拿起那支插在胸前口袋中的红玫瑰，放在鼻子下轻嗅着，手一扬，将手中扔了出去，红玫瑰在空中打着转，向楼下坠落下去。
<br/>　　玫瑰摔落在地上，几片花瓣被摔得脱离了花托，散落在地面上，一辆缓缓驶来的汽车将地上的玫瑰花和花瓣碾压的面目全非。
<br/>　　堕天使转身从天台护栏前离开，经过长袍男身边的时候，停下了脚步，说道：“我不允许有‘炼狱’这样的势力存在，尤其是这样的势力还不能被我所控制。”
<br/>　　“我明白了。”长袍男沉声应道。
<br/>　　其实在长袍男心中还有着疑惑，但是他很聪明的没有把自己心中的疑惑拿出来向堕天使询问，因为他明白，堕天使做的事情不会错，这是一种几乎盲目的信任！
