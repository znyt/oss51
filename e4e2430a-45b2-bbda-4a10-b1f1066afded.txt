　　剑道门之中，有着楚风眠得到的无数资源，都藏在了一座座的宫殿之中。
<br/>　　这些七大宗门弟子，冲进去的刹那之间，目光便是都看到了那一座座宫殿之中琳琅满目的无数宝物。
<br/>　　一道道目光，无比贪婪，疯狂的冲了进去，在大量的掠夺了起来。
<br/>　　烧杀抢掠！
<br/>　　如同一尊强盗，走进了一座宝库之中，无情的开始掠夺起来。
<br/>　　寒月漓，洛汐，青梦，燕皇等，皆是站在了一起，联手之下，抵挡着无数攻击。
<br/>　　可是在那足足十五位帝尊无敌级别强者的联手轰击之下，他们的联手，也是开始节节败退。
<br/>　　任何一位帝尊无敌级别的存在，所爆发出来的力量都是无比令人恐惧的，更何况是足足十五位联手。
<br/>　　纵然是以青梦的天纵奇才，加上无数机缘，能够以一敌二，都是极限了。
<br/>　　可现在却是足足十五位，这种力量，足以让人绝望。
<br/>　　一股股灵力爆炸的波动，在这剑道门之中响彻而起。
<br/>　　剑道门之中无数布置下来的防御阵法，都被纷纷催动，做着最后的血拼之战。
<br/>　　“门户被破了！”
<br/>　　在宫殿之中的楚风眠，都是突然张开双眼，他已经是清晰的感觉到，这剑道门的门户，已经是被打破了。
<br/>　　可他现在，却是无法脱身，他已经是达到了瓶颈的最后一步，必须要踏出这一步，才可以真正的踏入九劫古帝的境界。
<br/>　　一旦是这个时候，停止突破，不止是会损失这一次突破的机会，更是可能受到反噬。
<br/>　　箭在弦上，不得不发。
<br/>　　“给我破！”
<br/>　　轰隆隆！
<br/>　　强大的龙威从楚风眠的身上爆发而出，撕裂这他身上的牢笼枷锁，每撕裂开一个口子，楚风眠便是凝聚龙皇精血。
<br/>　　他身上的龙皇精血，已经是达到了九千九百九十九滴。
<br/>　　只差一步，便是凝聚万滴。
<br/>　　“别跟这些废物纠缠了，那个楚风眠？在哪里？”
<br/>　　另一边，在围攻寒月漓洛汐等人的人群之中，一道身影闪烁而出，正是那东宫厉。
<br/>　　他的目光突然扫视着整个剑道门之中，疯狂的寻找着楚风眠的身影。
<br/>　　剑道门，是以楚风眠一己之力建立而起的。
<br/>　　他们这一次前来的目的，可不止是要毁了剑道门，真正目的，却是要杀了楚风眠。
<br/>　　“看着龙威，来自地下！”
<br/>　　又一尊老者的身影，显现出来，那枯叶也脱身而出，目光扫视，突然看向地底之下。
<br/>　　来到这剑道门之中，他们便是都感觉的出来，楚风眠身上龙威，皆是来自于地下。
<br/>　　“走！去杀了他！”
<br/>　　嗖！
<br/>　　东宫厉，枯叶二人的身影，一个闪烁，便是向着地底之下飞去。
<br/>　　万米地下，对凡人来说，是遥不可及的区域。
<br/>　　但是对东宫厉，枯叶这等帝尊无敌的人物来说，根本不在乎，刹那之间，两股灵力打入到地下之中。
<br/>　　万米地下，当即是被撕裂开来，突然之间一座雄伟的宫殿，映入到了他们的目光之中。
<br/>　　这一切龙威的来源，都是在这一座宫殿之中。
<br/>　　“这小子就在宫殿之中！”
<br/>　　“轰碎他！在杀了那小子，最好将那小子活捉，他现在正在突破境界，没有任何的反抗能力，是活捉他的好机会！”
<br/>　　枯叶，东宫厉，交谈了一声，二人凝聚灵力，同时出手，一道剑光，一道拳风，同时轰击而出，轰击在了那一座宫殿大门之上。
<br/>　　轰隆隆！
<br/>　　整座宫殿，都因为这两道力量的轰击，开始剧烈的抖动起来。
<br/>　　但是端坐在其中的楚风眠，却是毫无动作，他的一切思想，一切的注意力，都集中在突破境界上面。
<br/>　　哪怕是天塌下来，都无法影响到现在楚风眠的意识。
<br/>　　那剑光拳风之力，轰击在这一座宫殿之上，整个宫殿之上，都浮现出了不少的裂痕。
<br/>　　就在这一座宫殿，都要即将破碎的时候，在这宫殿大门之上，一道金色的剑光，突然浮现而出。
<br/>　　就是这一道剑光，散发出一股惊天剑意，将这宫殿之上，破碎的无数裂痕，硬生生的修复过来
<br/>　　“极道剑印？”
<br/>　　枯叶，东宫厉目光看到这一道金色剑光的一刻，同时说道，眼神之中露出几分震惊。
<br/>　　极道剑印，在如今这个时代之中的年轻人，只怕是没有几人知道。
<br/>　　但是在万年之前，这极道剑印，却是赫赫有名，这可是曾经剑道之主的标志。
<br/>　　只有剑道之主本身，以及他的弟子，才有资格掌控的一种印记。
<br/>　　在万年之前，哪怕是七大宗门的天才，见到这极道剑印的一刻，都要退避三舍，不敢招惹。
<br/>　　“这小子居然就连极道剑印都能掌控，此人绝对是得到了传承那么简单！”
<br/>　　枯叶眼神之中露出几分兴奋之色。
<br/>　　今日楚风眠的身份越是神秘，对他们的好处也就越多，甚至可能楚风眠会知晓曾经剑道门的无数隐秘，无数宝藏。
<br/>　　“剑道之主已经死了，这小小的极道剑印也想要拦住我们？破了他！”
<br/>　　东宫厉也是脸色兴奋无比，狰狞一笑，当即凝聚灵力在他的拳风之上。
<br/>　　轰隆隆！
<br/>　　一拳打出，跟这极道剑印，凌空相撞。
<br/>　　极道剑印，的确强横无比，因为曾经极道剑印代表的，可是剑道之主的力量。
<br/>　　现在剑道之主以死，经历万年的时间，这极道剑印的威力，已经是远远无法跟万年之前所媲美了。
<br/>　　一拳轰击之上，这一把金色小剑，在空中与拳风之力交锋，轰然间剧烈的抖动了起来，下一刻，破碎开来。
<br/>　　极道剑印，被破！
<br/>　　那枯叶，更是抓住这个机会，身形猛然间向着宫殿的方向冲了过去，手中的剑锋当即斩下，狠狠的斩在这宫殿大门之上。
<br/>　　咔嚓！
<br/>　　失去了极道剑印，这宫殿，也承受不住如此之强的力量，宫殿大门被直接打破。
<br/>　　宫殿打破被破。
<br/>　　如今在外，一眼便是足以看见，坐在其中的楚风眠，以及楚风眠背后显现而出的无数虚影。
<br/>　　“擒下他！”
<br/>　　“荒古圣塔！”
<br/>　　那东宫厉凝聚灵力在掌心之上，猛然对着楚风眠抓了过去，在他的掌心之中，灵力凝聚突然化为了一座七层宝塔，一把对着楚风眠镇压而去。
