　　其实，
<br/>　　这百万吨级的核弹，威力不可谓不大！
<br/>　　也多亏叶尘反应比较快，及时在核弹爆炸的那一瞬间，整个人瞬间退身到数千米之外，开启八咫镜的同时，还将琉璃金身、法相天地一通施展出来。
<br/>　　核弹爆炸的威力虽强，但终究都会有一个延迟的扩散过程，而正是短短零点几秒，就足以让叶尘做出防备。
<br/>　　说到底，武器终究只是武器！
<br/>　　这就好比，你手里虽然有枪，而对方的反应的速度远远超过你，在你抬起枪的那一瞬间，对方就已经反应过来，并且可以预判出子弹的轨迹，你的枪自然也就打不中他。
<br/>　　不过，饶是叶尘在最后关头，逃离了核弹爆炸的最核心地带，强大的冲击力，还是险些让他受伤。
<br/>　　八咫镜的防御护罩瞬间便被冲破，连琉璃金身和法相天地，也未能持续太久，好在他自从炼化了一滴神龙之血后，肉身强度突飞猛进，最终愣是硬抗了下来。
<br/>　　“怎么？看你的样子，似乎很是失望啊！”
<br/>　　叶尘从废墟中窜出之后，身体一阵闪烁，很快便来到数百里外的苏珊跟前，直接一把抓住她的手腕，冷冷一笑道。
<br/>　　这女人，之前以不想面对昔日的同事为由，并没有跟随叶尘前往第九区军事基地，而是远远躲在了数百里之外。
<br/>　　现在看来，她很显然也知道，超能局下方埋藏着核弹，而且猜到米国政府会引爆核弹！
<br/>　　苏珊此刻，还没有从之前的震撼中回过神来。
<br/>　　别人不清楚，她身为超能局十二主神，又岂会不知道，这超能局下方，可是埋藏着将近200万吨级的核弹，居然这样都没能将叶尘炸死！
<br/>　　这个男人，也太恐怖了吧！
<br/>　　“没……没有！”
<br/>　　面对叶尘仿佛要吃人的目光，苏珊吓得浑身娇躯一颤，连说话也变得结巴起来。
<br/>　　“走！立即带我去见你们的总统大人！”
<br/>　　叶尘也懒得和这女人计较，直接再次冷冷命令道。
<br/>　　苏珊哪里敢说一个不字，当即连忙点头称是。
<br/>　　“那个恶魔，似乎朝华盛顿的方向去了！”
<br/>　　“看来之前那枚核弹，已经彻底将这个恶魔激怒了！”
<br/>　　“这下米国恐怕要有大麻烦了！”
<br/>　　……
<br/>　　诸国的众高层，看到这一幕，顿时又纷纷议论起来。
<br/>　　一时间，就连和米国一直不对付的苏国，库京和苏国众高管，也全都露出一抹凝重之色。
<br/>　　虽然最近数十年，他们苏国一直被米国所压制，看到米国吃瘪，他们心头感到无比爽快，可是眼看着那道如同恶魔般的身影，一步步朝米国国都靠近，又不由得升起一股兔死狐悲的感觉。
<br/>　　其余诸国的高层，也几乎全都是同样的想法，
<br/>　　倘若连当世第一大国，都奈何不了这个如同恶魔一般的男人，那他们岂不是更加束手无策？
<br/>　　而与此同时，
<br/>　　米国各大论坛，以及脸书、推特等社交网站之上，早已经彻底炸开了锅。
<br/>　　虽然米国政府，拼命的想要掩盖，可是米国的媒体实在太强大了，很快便所有的事情，扒的一清二楚。
<br/>　　甚至一些网站之上，已经开始出现核弹在第九区爆炸景象，还有人用无人机技术，锁定了叶尘的行踪，在各大论坛中直播起来。
<br/>　　而这一爆炸性的新闻，很快又通过网络传遍了世界！
<br/>　　一时间，几乎全球数十亿人的目光，全都吸引了过来，可谓是举世瞩目。
<br/>　　……
<br/>　　而此时，
<br/>　　米国国会之中，早已经乱成了一锅粥。
<br/>　　“看看吧！这就是贸然出手的下场！”
<br/>　　“我早就说过，不能轻易动手，现在好了吧？”
<br/>　　“这下我们都要完蛋了！”
<br/>　　之前的主和派，顿时纷纷指责起来，而那些主战派则个个哑然。
<br/>　　“百万吨不行，那就千万吨！亿吨！大不了我们将所有核弹引爆，大家同归于尽！”
<br/>　　不知道是谁，忽然怒声来了一句。
<br/>　　的确，米国所储存的核弹量，足以将整个地球打穿。
<br/>　　真要是到了那一步，他们的确具备那样的能力，可若是真到了那一步，那将会是全球的末日。
<br/>　　众人一阵议论之后，最终也没能拿出一个完美的方案。
<br/>　　而就在这时，总统助理急急忙忙从外面冲了进来，
<br/>　　“他……他来了！”
<br/>　　唰唰唰！
<br/>　　米国国会到场的数百名议员，包括总统普朗克，全部霍然站立起来。
<br/>　　“这么快！”
<br/>　　从第九区军事基地，到米国国都华盛顿，相距上千里，对方居然只用二十分钟左右的时间。
<br/>　　“总统阁下，第一骑兵师、第一机械化步兵师、大红一师等七支部队，早已经准备就绪，要不要……”
<br/>　　还没等米国国防部长拉姆斯的话说完，普朗克摆了摆手道：
<br/>　　“还是先按兵不动吧！普通军队对此人，已经构不成任何威胁，只会白白牺牲！”
<br/>　　众人听到这话，顿时一阵黯然，
<br/>　　的确，连百万吨级的核弹，都杀不了此人，普通军队的飞机、坦克，在对方眼中，估计也就和小孩的玩具差不多。
<br/>　　“总统阁下，诸位议员，我们还是赶紧前往紧急避难基地吧！”
<br/>　　有人开口提议道。
<br/>　　“恐怕……已经来不及了！”
<br/>　　普朗克摇头叹了一口气，道：
<br/>　　“更何况，我们若是躲了起来，谁来和此人谈判？”
<br/>　　……
<br/>　　另外一边，
<br/>　　叶尘进入米国国都华盛顿之后，速度也降了下来，开始缓缓踏空而行。
<br/>　　“叶先生请看！那里就是国会山，国会山之上，是我们米国的最高象征国会大厦，总统阁下现在应该就是那里！”
<br/>　　一旁的苏珊指着远处的一座山丘，向叶尘开口解释道。
<br/>　　叶尘点了点头，身体一晃，很快便跨越了数十里的距离，来到国会山的上空。
<br/>　　此时，下方华盛顿的市民们，早已经事先得到消息，正在纷纷朝城外逃离。
<br/>　　而整个国会山周围，也早已经被米国军队重重保护起来，看到天空上出现一道金色人影，顿时纷纷用武器瞄准，随时准备进攻。
<br/>　　然而，叶尘却犹如没有看到一般，目光在下面扫视了一圈之后，最终落在国会大厦前，一名白人中年的身上，
<br/>　　“你就是米国总统普朗克？”
<br/>　　声音落下，犹如九天神雷，滚滚而来，众人只感觉耳膜都要被震破了，浑身不由自主的一阵乱颤，险些晕厥过去。
<br/>　　就这，还是叶尘刻意收起了威压，否则普通人在他面前，恐怕连站起来的能力都没有。
