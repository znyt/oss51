　　在太阳落山之前，黄昏时分，两人终于赶到了基地。
<br/>　　和他离开的时候相比，基地的防御似乎更加严密了，包括那厚重的石质围墙，都是加厚了许多。
<br/>　　城墙上的巡逻队伍也明显增加了数量，看样子，基地是在防备着什么。
<br/>　　门口守城的，是两个挎着特支步枪的军人，穿着迷彩服，年龄不大，脸上还带着一丝稚气。
<br/>　　却也不是之前熟悉的面孔了。
<br/>　　因为什造是第一次入城，所以还需要一些手续，入城检查也麻烦了许多。
<br/>　　刘哔很耐心的等着，随后有人将他们分别带入了一个大厅。
<br/>　　“这里是隔离区，每个从外面回来的人，都要在这里待四个小时，以免会有意外发生。”
<br/>　　看到什造不解的眼神，刘哔解释了一声，细心嘱咐道：“等会我们出来去登记档案，你跟着我，不要乱走。”
<br/>　　说话的时间，不远处走来一个监视官，将两人分别安排好了相邻的两个单人房间，随后退了出去。
<br/>　　屋内光线有些阴暗，随着刘哔走进，后面传来了锁门的声音。
<br/>　　耸了耸肩，刘哔躺在房间内的行军床上，翻看着商城里面的物品。
<br/>　　有些无聊。
<br/>　　他来过很多次，对这里自然是无比熟悉，只要你在规定时间内没有尸变，自然不会有任何问题。
<br/>　　这次喰种位面的行程，让他有些疑惑，很多东西，都和刘哔摸不到头绪。
<br/>　　比如，整整两个月的时间，他都没有接到哪怕一个任务。
<br/>　　不管接触到再多的剧情人物，甚至说参与了所有的主线剧情，但依旧没有任务出现。
<br/>　　他问过系统，但是没有解释。
<br/>　　或许是自己将剧情改变太多了？
<br/>　　还是因为提前推动了剧情发展？
<br/>　　他有点疑惑，这还要在下一个位面等待证实。
<br/>　　包括从第一个位面抽奖得来的灵砖，到现在他也没有找到使用它的办法。
<br/>　　还有那串珠子，要不是他用尽办法都无法损伤到珠子，甚至刘哔都会将它当做废品丢掉。
<br/>　　躺在床上，刘哔思考着这些问题，但依然没能有任何头绪。
<br/>　　线索太少了。
<br/>　　四个小时的时间并不算短，中间陆续有拉动铁门的声音，那是有新的幸存者进来，或者有人离开。
<br/>　　终于，刘哔和什造的时间也到了，房间门口，一个陌生的男人走了过来，满脸横肉，手里拿着一串钥匙。
<br/>　　“时间到了，你们可以离开了。”
<br/>　　出乎意料的，男子看起来虽然十分凶悍，但语气却十分温柔。
<br/>　　铁门上的锁链，传来‘哗啦啦’的抖动声，刘哔从床上下来，穿好鞋子，走出了大门。
<br/>　　门外有几个和他一起出来的幸存者，什造也在其中，正不知所措的等候着他。
<br/>　　身边的幸存者们都有说有笑，陆续有人走向来迎接他们的同伴，什造小小的身影，在其中显得有些萧瑟。
<br/>　　“走，我们回家。”
<br/>　　牵起什造柔软的小手，刘哔带她去登记了一下资料，随后一起向家中走去。
<br/>　　……
<br/>　　贫民窟，一处破败的木板房里，三人坐在一张同样破旧小方桌前，桌上摆满了丰盛至极的食物。
<br/>　　各式各样的罐头，压缩食品，甚至还有种类丰富的酒水，这样的晚宴，在末世这种环境下，可以用穷奢极致来形容。
<br/>　　奇怪的是，这些食品上面的商标以及生厂日期，都被特意撕去或涂改了，让人看不清是这些食物是出自什么地方。
<br/>　　“哥，这些东西……你是从哪里带回来的？”
<br/>　　刘莽呆呆的坐在桌前，语气震惊，他知道这些食品的价值。
<br/>　　“在一个……不一样的地方。”
<br/>　　轻笑了一声，刘哔并没有多过解释，并非不信任，而是这种东西，还是不要暴露出来为好。
<br/>　　总有些特殊手段，可以强行读取别人的记忆，刘莽不是异能者，很容易就能将消息泄露。
<br/>　　一旦泄露出去，恐怕会引起不可预料的后果。
<br/>　　“……哥，这次回来，就不会走了吧？”
<br/>　　知道老哥不想多说，刘莽也不多问，他知道老哥肯定有不能说的原因。
<br/>　　他们兄弟两人相依为命这么多年，很多东西，都不需要点名，相互之间，自然会去理解。
<br/>　　“谁知道呢。”无奈的叹了口气，刘哔拆开一瓶黄桃罐头：“到时候再说吧，先吃饭，我们很久没吃过水果了吧。”
<br/>　　刘莽动了动筷子，却也没舍得下手，这一瓶黄桃，在基地不知道能换到多少东西。
<br/>　　水果罐头，这是最为稀缺的物资，一般都是很有身份的大人物，才有资格享用的。
<br/>　　“吃吧，这些东西，以后我们要多少有多少。”分别给刘莽和什造夹了一块黄桃，刘哔目光有些愧疚：“我们，以后再也不需要受苦了。”
<br/>　　谁能想到，二十年前那两个纯真的少年，会活成如今这种样子。
<br/>　　又有谁会想到，早在十年之前，那个被所有势力当做笑话看待的两名幸存者，竟然还能有翻盘的机会。
<br/>　　圣殿，这是多么久远的名字，但它也曾统御了一方。
<br/>　　但自当年那场战斗之后，这个末世初期最大的幸存者组织就此埋没了。
<br/>　　三阶特殊属性异能者，被二阶火系异能者打败，甚至如果没有至亲之人相救，都会丢掉自己的性命。
<br/>　　直至那时，所有人方才看清，所谓的圣殿殿主，只是一个空有等阶的废物。
<br/>　　随后是内部成员叛乱，以及无穷无尽的追杀。
<br/>　　说来可笑，那些他曾经帮助过的人，反而比他的敌人更加凶猛，为的，就是抹除他们身上的污点。
<br/>　　被废物帮助过的污点。
<br/>　　在一个废物面前摇尾乞怜的污点。
<br/>　　以至于他带着自己的弟弟，只能隐姓埋名，远走他乡。
<br/>　　十年的时间，他看清了太多事情。
<br/>　　这或许也是一件幸事。
<br/>　　如果没有这份经历，以他原本的性格，几乎不可能从这样人吃人的世道上活下去。
<br/>　　越向往光明的人，在经历过某些事情之后，就越喜欢黑暗。
<br/>　　而他又想起了父亲生前告诉自己的话语，很多事情，人教人是教不会的，只有事教人才能教会。
<br/>　　但是这样，往往会付出很大的代价，或许结果，就是一蹶不振。
<br/>　　万幸的是，虽然这次摔的很疼，但他终究爬起来了。
<br/>　　他回来了。
