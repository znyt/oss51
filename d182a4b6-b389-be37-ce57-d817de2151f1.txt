        “只要为了他好，我什么都无所谓。”季雅珊释然地笑了笑。
<br/>
<br/>        童筱颖顿时感到不可思议地看着季雅珊。
<br/>
<br/>        如果换做是傅夜沉，有其他女人要跟他单独约会，她可不会这么大方地就答应。
<br/>
<br/>        所以，她并不大理解季雅珊对冷逸峰的那种爱。
<br/>
<br/>        随后，季雅珊将童筱颖安排睡到了傅夜沉的主卧里。
<br/>
<br/>        依山别苑是傅夜沉，这事童筱颖并不知道，只是，当她踏进这幢别墅的时候，却有一种似曾相识的感觉。
<br/>
<br/>        好像曾经，她也在这里住过一段时间。
<br/>
<br/>        有些模糊的画面，在脑海里闪过，她曾经，好像焦虑地在这里住着等待过谁。
<br/>
<br/>        第二天，季雅珊为童筱颖搭配好了约会时所穿的衣服，还说了一些关于冷逸峰的喜好。
<br/>
<br/>        童筱颖看得出，季雅珊确实很了解冷逸峰。
<br/>
<br/>        这么好的一个女孩，就在身边，为什么冷逸峰这个男人还要去思念着一个已经死去多年的女人？
<br/>
<br/>        出门后，冷逸峰亲自开车，童筱颖则坐在了副驾驶座上。
<br/>
<br/>        到了市中心后，除了不牵手、不拥抱、不接吻、不上床，其他情侣之间会做的事情，他们都正在进行。
<br/>
<br/>        吃饭、逛街、看电影……
<br/>
<br/>        童筱颖觉得很是无趣。
<br/>
<br/>        更确切来说，如果面前的这个男人换成傅夜沉的话，她才会觉得有趣。
<br/>
<br/>        冷逸峰像是看穿了童筱颖的心思，两人从馆子里吃完饭出来回到车上后，他便问道：“你想回小沉的身边？”
<br/>
<br/>        这还用问吗？
<br/>
<br/>        童筱颖坐在副驾驶座上，却只能十分无奈的口是心非地说道：“我们今天这么做，其实对你来说没有一点意义。我始终不是婴宁。”
<br/>
<br/>        “那你能告诉我，你的一举一动，为什么和婴宁一模一样？”冷逸峰微蹙起了眉头。
<br/>
<br/>        他和婴宁第一次约会的时候，他问婴宁想吃什么的时候，婴宁说的是火锅，她童筱颖也亦是如此。
<br/>
<br/>        就比如，刚刚在馆子里吃火锅的时候，童筱颖唰了清汤粉丝，然后放碗里拌上了花生酱。
<br/>
<br/>        这是婴宁生前最喜欢的吃法。
<br/>
<br/>        还有现在……
<br/>
<br/>        跟他冷逸峰在一起时，心里想的却是其他男人。
<br/>
<br/>        他把当年第一次和婴宁约会时的情景，和童筱颖重新走了一遍，而童筱颖所表现出来的，竟然和婴宁当年的行为举止一模一样。
<br/>
<br/>        “如果，季雅珊死了，你是不是也会像怀念婴宁一样去怀念季雅珊？”童筱颖忽然无厘头地问道。
<br/>
<br/>        冷逸峰不以为然地回答：“雅珊怎么可能会死。”
<br/>
<br/>        “所以，你对她看得并不是很重对吗？”童筱颖仰头，淡定地对上冷逸峰的眼睛，“身边有个女孩，全心全意地对你。而你却心心念念着一个死去的女人？你会不会痴情过头了？又或者说，婴宁的死，跟你有关系，你内疚，所以才会久久不能释怀？”
<br/>
<br/>        童筱颖说到这里，冷逸峰放在方向盘上的双手不经意间地握成了拳。
<br/>
<br/>        就在他沉默不过几十秒后，他突然伸手抓住了她的手腕。
<br/>
<br/>        冷逸峰的这一举动，将童筱颖吓了一跳。
<br/>
<br/>        随着他手下的力度，她的身子也跟着朝他凑近了一些。
<br/>
<br/>        “你说得一点都没错，婴宁的死，确实跟我有关。”冷逸峰语气凝重地回答。
<br/>
<br/>        童筱颖有些心慌意乱地瞪着冷逸峰，甚至还有些不寒而栗。
<br/>
<br/>        就在她神情恍惚的时候，冷逸峰不知从哪儿掏出了一把匕首，趁着她不备之际，向她挥了过来。
<br/>
<br/>        童筱颖只觉脖子后一阵刺痛，随即感到有一丝温热的液体划过肌肤。
<br/>
<br/>        冷逸峰收回手时，他的指腹上沾了血，还有一粒十分微小的东西立在上面。
<br/>
<br/>        “原来小沉也不是百分百的信任你，居然在你身上安装了纳米监听定位器。”冷逸峰一边说着，一边将指腹上的那个微小的东西给捏碎了。
<br/>
<br/>        童筱颖吃痛地捂着自己的脖子后，闷闷不乐地皱着柳眉。
<br/>
<br/>        冷逸峰随之在储物箱里翻出了一个创可贴撕开后，递给了童筱颖：“贴上吧！只是皮肉之伤，不会要了你的命。”
<br/>
<br/>        “你没事取它做什么？”童筱颖接过创可贴不满地问道。
<br/>
<br/>        她都不在乎，他这么一个外人又有什么好在乎的？
<br/>
<br/>        “因为，我不打算把你还给小沉了。”冷逸峰忽然冷冷地说道。
<br/>
<br/>        童筱颖顿时瞪大了眼睛，本能地侧过身去，想去拉开副驾驶座的车门，但是被冷逸峰先一步将车门给反锁了，并且只有他驾驶座那边的按钮才能操控解锁。
<br/>
<br/>        “你到底想做什么？”童筱颖忍不住怒吼道。
