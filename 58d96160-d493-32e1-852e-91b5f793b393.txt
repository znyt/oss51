        “大哥，谢谢你救了我。但是，现在，请你不要管我。我就想一个人走走。”童筱颖后退了一步，与傅夜沉保持着一段距离。
<br/>
<br/>        傅夜沉看着此刻失魂落魄的童筱颖，心里终于明白，他错过了什么。
<br/>
<br/>        他从刘管家那儿拿到了筱颖之前的资料。
<br/>
<br/>        就是那段日子，他职责所在，为了解救人质，也就是那个被歹毒绑架的季雅姗而身受重伤，躺在医院里昏迷不醒。
<br/>
<br/>        也正是那个时候，她童筱颖过得最悲惨的时候，她急需要钱，她急需有人能帮她一把，而出现在她身边的人，是傅昼景，不是他傅夜沉。
<br/>
<br/>        所以，从那一刻起，她心里的那个避风港，是阿景，不是他。
<br/>
<br/>        童筱颖转身往海边走去，傅夜沉只能远远地默默地跟在她的身后。
<br/>
<br/>        来临海城也有一年了，她却没有走进沙滩里去近距离地接触过大海。
<br/>
<br/>        她多么希望，此时此刻，陪伴她的人是傅昼景。
<br/>
<br/>        傅夜沉远远地守着童筱颖，从日落守到了天黑。
<br/>
<br/>        童筱颖仍旧一个人孤零零地坐在沙滩上，吹着海风，看着远处的大海。
<br/>
<br/>        此时，傅夜沉的手机响了起来，他一边盯住那边童筱颖的一举一动，一边接听了电话。
<br/>
<br/>        是警局那边打来的电话，说是已经抓到了季运恒，想请示他的意思。
<br/>
<br/>        “先关他三个月，日后再说。”傅夜沉淡淡地命令道。
<br/>
<br/>        这个电话刚挂，又一个电话打了进来，是助理郑忠威的电话。
<br/>
<br/>        傅夜沉立马接听了电话。
<br/>
<br/>        “大少爷，不好了。”郑忠威焦虑道。
<br/>
<br/>        傅夜沉剑眉微蹙，目光下意识地从童筱颖的身上收了回来，望向了无边无际的大海，深沉地说道：“有话直说。”
<br/>
<br/>        “就是‘远东城军机部项目’，上级派人下来查账了！”郑忠威急匆匆地回答。
<br/>
<br/>        傅夜沉眉头紧锁，反问：“这个项目，不是阿景在负责吗？为什么上面的人会突然下来查账？”
<br/>
<br/>        远东城军机部项目，是政府安排给他们傅氏集团代建。
<br/>
<br/>        傅夜沉当初是想交给堂叔冷岩峰全权负责。
<br/>
<br/>        因为这个项目斥资比较大，傅夜沉考虑的是傅昼景经验不够，所以才只是让傅昼景辅佐堂叔冷岩峰去完成。
<br/>
<br/>        但是堂叔冷岩峰却把机会让给了傅昼景，让傅昼景去独立负责，他来辅佐。
<br/>
<br/>        当时对于堂叔冷岩峰的提议，傅夜沉认为不妥。
<br/>
<br/>        但是堂叔冷岩峰坚持这么做，声称傅昼景应该得到更大的锻炼机会，傅夜沉最后不得不妥协，同意让弟弟傅昼景去负责。
<br/>
<br/>        如果这次上面派下来的人来查账，若是出了问题，那么……
<br/>
<br/>        傅夜沉若有所思地接着问道：“爷爷知道这件事情吗？”
<br/>
<br/>        “动静太大，老太爷已经知道了。并且……”郑忠威欲言又止。
<br/>
<br/>        傅夜沉焦虑不安地追问道：“并且什么？”
<br/>
<br/>        “并且，老太爷说，二少爷想独吞‘远东城军机部项目’，想一人控权五方。如果真查出了问题，老太爷打算把二少爷推出去！说是二少爷的个人所为，与傅氏集团无关。”郑忠威接着说道。
<br/>
<br/>        “爷爷是想‘大义灭亲’吗？！这让阿景怎么办？”傅夜沉反问。
<br/>
<br/>        郑忠威顿时沉默了。
<br/>
<br/>        傅昼景也是因为“远东城军机部项目”的这个工程，才选择跟季思妍“同流合污”，背着童筱颖与季思妍“苟合”。
<br/>
<br/>        一人控权五方，代建方、招标方、施工方、审计方和监理方。
<br/>
<br/>        季思妍是想让傅昼景的野心蓬勃，成为这四方幕后真正的大Boss。
<br/>
<br/>        也就是说，最后这个“远东城军机部项目”所赚的钱，全都进入了他傅昼景和她季思妍的口袋里。
<br/>
<br/>        这种事情，一旦被发现，情节若是严重的话，是要坐牢的。
<br/>
<br/>        傅夜沉一直都知道阿景为了这个项目，在暗中蠢蠢欲动，爷爷也三番五次地暗示过他，他心知肚明，但是他对于阿景的这件事情一直都是睁只眼闭只眼。
<br/>
<br/>        只是他没料到，阿景会如此处事不小心，把事情闹大，让上头派人下来查账。
<br/>
<br/>        爷爷傅炎廷的态度也已经很明确了，爷爷也早就知道这件事情，同样选择了暗中不动。
<br/>
<br/>        而此时此刻，他该如何抉择？
<br/>
<br/>        到时候，真的要把阿景推出去吗？
<br/>
<br/>        就在傅夜沉为这件事情走神的时候，不经意间地转身，抬眸才看到不知何时出现在自己身后的童筱颖，身心一怔。
<br/>
<br/>        “这件事情，晚点再说，先挂了。”傅夜沉连忙跟郑忠威挂了电话，他不确定筱颖刚刚有没有偷听到他的通话内容。
