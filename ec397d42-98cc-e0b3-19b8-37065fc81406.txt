　　这无数龙党弟子，一齐凝聚出来强大无比的万龙之王。
<br/>　　如今去是在楚风眠的手中，被轻而易举的给镇压，就连任何一丝丝的反抗都做不到。
<br/>　　“吼！”
<br/>　　万龙之王，被镇压在阵法之下，还在疯狂咆哮。
<br/>　　但是随着楚风眠指尖一点。
<br/>　　这万龙之王的身躯，便是猛然爆炸开来，在其中，一件木质图腾，直接飞了出来。
<br/>　　“不好！”
<br/>　　“快将龙王图腾收起来！”
<br/>　　“此人是打算，夺取龙王图腾！”
<br/>　　那些龙党弟子，看到楚风眠的举动，一个个疯狂的咆哮起来，还打算要阻止楚风眠，收取龙王图腾。
<br/>　　“一群蝼蚁，我不先杀你们，你们还敢来反抗？既然如此，我就先杀了你们！看看谁还敢挑衅我的威严！”
<br/>　　楚风眠看到那些龙党弟子，眼神中的杀意渐浓。
<br/>　　在他说话之间，楚风眠的身躯再度动了。
<br/>　　他的一爪，从空落下，但是这一爪上，其中的五指，却是直接分化成为了五道剑芒。
<br/>　　这五道剑芒，直接向下，斩杀而去。
<br/>　　周围的空间，都是嘶嘶作响，好像是这剑芒的锋利，要将空间都给撕裂开来一般。
<br/>　　要知道这北邙圣域，可不再九域之中，而是曾经远古大陆的遗留，法则坚韧无比。
<br/>　　可是就算是这样，这周围空间都在嘶嘶作响，可以看得出，这剑芒到底有着多么锋利。
<br/>　　那些龙党弟子，一个个都是惊恐万分，似乎是察觉到了这剑芒的危险。
<br/>　　一个个连连打出无数武技，身躯连连后退，想要抵抗。
<br/>　　但是这剑芒闪过，这些龙党弟子，一个个都是被直接斩杀，就连任何一点血肉都没有留下，直接被剑芒卷入其中，彻底灭亡。
<br/>　　在场的地上，只剩下一滩滩的鲜血。
<br/>　　看到这些鲜血，周围围观的无数种子弟子，都是脸色骇然。
<br/>　　生死台上，陨落一些弟子，并不稀奇。
<br/>　　北邙学院的种子弟子中，其中陨落的大部分，都是死在了这生死台上。
<br/>　　但是如今像是眼前这一幕，却是闻所未闻，前所未见的，七八名龙党弟子，就这样死在了楚风眠的手中。
<br/>　　以一敌多，甚至还足以轻易击败。
<br/>　　每一名种子弟子，放在外面都是绝世天才，可如今在楚风眠的手中，好像都是捏死的蝼蚁。
<br/>　　无敌。
<br/>　　除了这个词，在场的众人，根本想不到任何还可以形容楚风眠的词。
<br/>　　“这种实力，还是人吗？龙党如此多人的围攻，居然都被他轻易化解，甚至是一招之下，将所有人斩杀！”
<br/>　　“此人到底是什么来路？他跟寒月漓，两个人一个比一个妖孽，本以为一个寒月漓就极为少见的，没想到还有他，这二人还是夫妻，怎么可能！”
<br/>　　“此人的实力，已经是绝对无比的巨头了，足以跟龙未央，夜天君，陆画，以及还有极为核心弟子这种巨头抗衡了！”
<br/>　　“龙党这次，可算是踢到了铁板了！”
<br/>　　“不过龙未央，可还没有出手，龙党的强大，可不是那些龙党弟子，而是龙未央，一个龙未央，比起整个龙党都要强大，他出手，这还不好说。”
<br/>　　一尊尊种子弟子，都在纷纷开口议论到。
<br/>　　这楚风眠跟龙党的战斗，已经是吸引了整个北邙圣域中无数弟子的眼光，纷纷赶来，来观看这一切。
<br/>　　在这生死台的远超，一座山峰上的亭子中，几尊弟子，坐在其中，饮茶，每个人，也都是看向生死台上的种种。
<br/>　　这几个人，每一个人身上的气息，都要远超种子弟子，是真正的巨头，都北邙圣域的核心弟子。
<br/>　　核心弟子，每个人都是如同龙未央，夜天君一样的巨头，他们有些人，行事低调，所以没有建立一方势力。
<br/>　　但是这并不代表这他们的实力弱小，相反，每一尊核心弟子，都是将来有希望争夺北邙学院下一任院长的人。
<br/>　　就算是争夺失败，也至少是太上长老，副院长等。
<br/>　　如今一共三尊核心弟子，便是坐在亭中饮茶，正看着楚风眠，一人轰杀了那些龙党弟子。
<br/>　　“这楚风眠的实力，倒是不错，一个新人，本来他刚刚进入到圣域的时候，还只能算是一个有些实力的小家伙，没想到现在，居然都快要跟我们平起平坐了。”
<br/>　　一尊核心弟子，一边喝着茶，一边饶有兴致的看向楚风眠，开口道。
<br/>　　“看来夜天君倒是招惹了一个大麻烦。”
<br/>　　“平起平坐？卢清平，你倒是看得起他，就凭他，也配跟我们平起平坐？”
<br/>　　令一尊核心弟子，不屑的开口道。
<br/>　　“就这表现的实力，还足以对夜天君造成麻烦？我看他今日，就要死在龙未央的手中了，这种实力，虽然不错，但是想要跟龙未央抗衡，还不可能。”
<br/>　　“古傲寒，上次你遇到寒月漓，便是这幅态度，现在如何？寒月漓已经是学院的红人了，还得到了先祖传承，等到她出关，只怕是你我都不是他的对手了。”
<br/>　　那卢清平，平静的开口道。
<br/>　　“不过这楚风眠，做人的确是太嚣张了，这里是北邙圣域，水深得很，就连我们都不敢太过放肆，木秀于林风必摧之，这个道理，这小子倒是不懂。”
<br/>　　“哼，那寒月漓，不过是凭借着身躯，先天神体玄寒之体，才可以胜我一招，这玄寒之体，只要是有人可以得到了她，就足以将她玄寒之体的力量夺走！”
<br/>　　古傲寒冷冷的开口道。
<br/>　　“这小子跟寒月漓虽然是夫妻，但是寒月漓身上玄寒之体的力量，却是并没有被他得到，只要我得到了寒月漓，便是就足以夺取了他的力量，倒时候我就是真正的无敌了！”
<br/>　　“那你也要的得到才行，现在想要追求寒月漓的人也不少，就连那南域皇室的皇子，都来北邙学院提亲了。”
<br/>　　卢清平摇了摇头，面带讥讽道。
<br/>　　“你觉得你，争的过南域皇室的皇子？”
<br/>　　“你！”
<br/>　　古傲寒听到这讥讽，脸色阴沉，刚想开口，却是旁边另一尊核心弟子打断道。
<br/>　　“先不用争了，龙未央要出手了！”
<br/>　　随着他的话，在场才平静下来，无视视线，再次向着生死台的方向，看了过去。
