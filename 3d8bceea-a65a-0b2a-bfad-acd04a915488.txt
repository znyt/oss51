夏夕绾九岁就被送到了乡下，她本不该对这个所谓的父亲夏振国再抱有任何期待了，果然这一通电话没有任何的意外。
<br/>
<br/>        夏振国还是她认识的那个夏振国，一心痴迷医学，最爱虚荣和面子，还想将夏氏医疗发扬光大。
<br/>
<br/>        现在他最引以为傲的女儿就是夏妍妍，而她这个从乡下回来的女儿可以用来冲喜，陪睡。
<br/>
<br/>        “爸，我知道了，明天我会去的。”
<br/>
<br/>        她的乖顺听话让夏振国语气松软了一些，“夕绾，你嫁过去是冲喜的，你那个病入膏肓的丈夫很快就会死的，等王总的事情解决了，到时爸再给你找一个好人家。”
<br/>
<br/>        “那就先谢谢爸了。”夏夕绾将电话给挂断了。
<br/>
<br/>        将手机关机，夏夕绾在陆寒霆的怀里闭上眼，其实心里是很难过的，她就是一个孤儿。
<br/>
<br/>        她想像正常的孩子那样，被自己爸妈疼爱着，有着简单平安的生活，可是这对于她而言简直就是奢望。
<br/>
<br/>        她没有家。
<br/>
<br/>        她就是一个没爹疼没娘爱的野孩子。
<br/>
<br/>        也许感觉到了冷，夏夕绾往陆寒霆的怀里蜷缩了些，他的怀抱挺括结实而温暖，可以给任何一个女人遮风挡雨。
<br/>
<br/>        她的脑袋枕在他的心房上，砰，砰，砰，他一下下强而有力的心脏博动让她很有安全感。
<br/>
<br/>        夏夕绾以为自己会失眠的，但是在这个男人的怀里她一觉睡到了天亮，一夜好眠。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆缓缓睁开了眼，现在已经是翌日清晨了，璀璨的晨曦透过层层窗幔镀了进来，在空气里洒下无数晕黄的因子。
<br/>
<br/>        陆寒霆眸里染着初醒的惺忪，还有片刻的茫然。
<br/>
<br/>        很多年了，很多年没有睡到清晨，在这美好的晨曦里自然睁开眼。
<br/>
<br/>        陆寒霆闭了闭眼，去搂怀里的女孩。
<br/>
<br/>        他知道她一夜都睡在他的怀里，因为他的怀里还有她残留的温软和体香。
<br/>
<br/>        但是，什么都没有搂到，怀里空空如也，夏夕绾已经不见了。
<br/>
<br/>        陆寒霆一下子睡意全无，掀开蚕丝被起了身。
<br/>
<br/>        这时书房门被推开，管家福伯一脸喜色的走了进来，“少爷，你醒了？少奶奶走的时候让我不要打扰你，再让你睡一会儿，这都多少年了，少爷竟然睡到了自然醒，就连南渊先生都没有办到的事情，少奶奶竟然办到了，这少奶奶身上究竟有什么魔力啊？”
<br/>
<br/>        福伯觉得太不可思议了，自家少爷的身体状况他最清楚不过了，昨晚少奶奶进去后他就很担心，但是少爷竟然抱着少奶奶睡了一晚上。
<br/>
<br/>        陆寒霆看了看门外，“少奶奶人呢？”
<br/>
<br/>        “回少爷，少奶奶说她出去处理些事情，晚上再回来。”
<br/>
<br/>        “她有说去哪里？”
<br/>
<br/>        “没有。”
<br/>
<br/>        “知道了。”
<br/>
<br/>        陆寒霆回到了卧室里，进了沐浴间冲澡，当他将白色衬衫脱掉时，就在镜面里看到了自己肩头那个深深的小牙齿印。
<br/>
<br/>        她咬的。
<br/>
<br/>        光看着这个小牙齿印就可以想象她当时咬的多么用力，现在都在他的身上留下她的印迹了。
<br/>
<br/>        陆寒霆今天没有去公司，而是在书房里办公，晚上的时候他看了一下腕表，都八点了，夏夕绾还没有回来。
<br/>
<br/>        他拿出手机，她没有给他发信息，也没有给他打电话。
<br/>
<br/>        陆寒霆觉得心口闷闷的，有些不爽，这时一串悠扬的手机铃声响起了，来电话了。
<br/>
<br/>        陆寒霆按键接通，“喂。”
<br/>
<br/>        霍西泽的声音迅速从那端传递了过来，“二哥，你都好久没有出来玩了，奶奶给你娶了一个新娘子，你是不是沉迷其中，已经过上家庭主男的生活了？”
<br/>
<br/>        家庭主…什么？
<br/>
<br/>        陆寒霆英气的剑眉一蹙，不耐道，“再废话就挂了。”
<br/>
<br/>        “别啊二哥，出来玩啊，我跟夜瑾哥在1949酒吧等你。”
<br/>
<br/>        ……
<br/>
<br/>        1949酒吧。
<br/>
<br/>        一处偏僻的包厢里，陆寒霆坐在主位上的沙发上，修长的两指里夹了一根香烟在抽着。
<br/>
<br/>        烟雾缭绕模糊了他的俊颜，只隐隐可以看见他紧蹙着眉心，清冷薄寒。
<br/>
<br/>        霍西泽在倒酒，“二哥，你这是怎么了，怎么进来就在抽烟啊，我看你是内火太大，需要好好的消消火。”
<br/>
<br/>        说着霍西泽将身边的一个美女推了过去，“二哥，这是1949新来的头牌，干净着呢，我专门为你留的，小樱，能不能将这杯酒喂给我二哥喝就看你的本事了。”

<br/>        1949酒吧一向是男人的销金窟，里面最不缺的就是美女，来这里消遣的男人一晚上豪掷千金，当然这个酒吧是霍家的产业。
<br/>
<br/>        海城四大豪门陆顾霍苏，今天齐聚了其三，这三个人穿一条开裆裤玩到大的。

<br/>        被推到了男人的身边，小樱一张清纯漂亮的小脸倏然一红，今晚的陆寒霆穿着简单的黑衣黑裤，这位陆氏掌舵人哪怕抽烟的时候也从骨子里弥漫着一股成熟成功男士的迷人质感，再加上他那完全没死角的俊颜，哪怕是不给钱小樱也愿意陪的。

<br/>
<br/>        小樱漂亮的小脸瞬间一白。
<br/>
<br/>        霍西泽迅速将小樱赶走了，“二哥，你都清汤寡水这么多年了，真的对女人没兴趣啊，奶奶已经不许我去找你玩了，就怕你跟我搞基啊。”
<br/>
<br/>        这时一边的顾夜瑾开口道，“寒霆，听说夏家替嫁了一个女儿给你，叫夏夕绾。”
<br/>
<br/>        听到这个名字，陆寒霆抬眸看了顾夜瑾一眼。
<br/>
<br/>        顾夜瑾生的十分俊美，脸上戴着一副斯文的金丝眼镜，他抿了一口手里的红酒然后看向前方，“看看那是谁？”

<br/>        陆寒霆抬头一看，迅速看到了一道纤柔的身影，夏夕绾。

<br/>        夏夕绾身边还有一个男人，是那个大腹便便的王总。

<br/>        “靠，”霍西泽拍桌而起，“二哥，这个夏夕绾怎么在陪一个老男人喝酒，她竟然敢给你戴绿帽子！”
<br/>
<br/>        霍西泽拿了一个酒瓶就要冲上去，霍氏太子爷，就是海城的小霸王，“二哥，我去教训他们给你出气！”
