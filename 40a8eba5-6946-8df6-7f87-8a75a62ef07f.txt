        “那我也替十爷查查夏玄烟的事情吧！毕竟，童筱颖现在与夏玄烟出现同化现象应该不是偶然。”连华生接着说道。

<br/>        傅夜沉点了点头。
<br/>

<br/>
<br/>        “我明白你的意思。”傅夜沉微微皱起了眉头。

<br/>
<br/>        傅夜沉明白连华生的一片好意，所以并未拒绝他的提议。
<br/>
<br/>        挂了电话没多久，傅夜沉的手机便收到了连华生发过来APP安装包的信息。
<br/>
<br/>        随后，他收拾了一下书桌上的文件和资料，离开了书房。
<br/>
<br/>        傅夜沉刚走到卧室门口，就听到里面传来童筱颖的呓语声。
<br/>
<br/>        “不要——不要——”
<br/>
<br/>        她的声音透着哽咽，听起来像是很痛苦的样子。
<br/>
<br/>        傅夜沉连忙疾步走到床边，见童筱颖伸出了双手在半空中扑腾，他立即抬手抓住她的双腕紧紧地按在自己怀中。
<br/>
<br/>        当他想要唤醒她的时候，她又突然安静了下来，就好像刚刚什么事也没发生过一样。
<br/>
<br/>        不过，童筱颖的眉心微微蹙起。
<br/>
<br/>        傅夜沉不得不坐到床上去，在她身旁躺下，一边抚平她的眉心，一边将她搂入怀中，轻轻地拍抚着她的背脊。
<br/>
<br/>        或许是感觉到了一股熟悉的气息，童筱颖的眉心渐渐地舒展，甚至本能地往傅夜沉的怀里钻了钻，并在他的怀中找了个舒适的位置，抱着他，嘴角微微上扬。
<br/>
<br/>        翌日。
<br/>
<br/>        童筱颖睡醒时，傅夜沉早已不见了踪影。
<br/>
<br/>        她扶着额头从床上坐了起来，脑海里依稀记起了昨晚所做的那个梦。
<br/>
<br/>        这是她在做过的梦里，第一次出现除了大哥傅夜沉以外的男人。
<br/>
<br/>        而这个男人，就是昨晚在咖啡厅里跟她交谈的那个男人。
<br/>
<br/>        十爷……
<br/>
<br/>        冷逸峰……
<br/>
<br/>        童筱颖也不知道自己是怎么着，就得知了那个男人的名字。
<br/>
<br/>        梦里，她第一次见那个男人的时候，那个男人戴着假面眼镜，坐在背光的阴影里，指间夹着袅袅的香烟。
<br/>
<br/>        后来，她还梦到，那个男人命令手下放狗来咬她。
<br/>
<br/>        那条狗……
<br/>
<br/>        真的太可怕了！
<br/>
<br/>        童筱颖屈起双腿，双手的手肘抵在膝盖上，两根大拇指分别揉了揉发痛的太阳穴。
<br/>
<br/>        “不要忘了你的任务。”
<br/>
<br/>        突然一个阴森暗哑的声音，毫无征兆地在脑海里响起。
<br/>
<br/>        随即，一阵悦耳的手机铃声，却将她像是做贼心虚了般吓了一跳。
<br/>
<br/>        童筱颖在床头柜上找到了自己的手机，手机屏幕上是一组陌生来电。
<br/>
<br/>        按下接听键后，听筒里传来刚刚那个脑海里突然响起的阴森又暗哑的声音。
<br/>
<br/>        “我再给你一个星期的时间，一个星期后，如果你不能杀了傅夜沉，那么我就来抓你回去。”
<br/>
<br/>        “嘟——”
<br/>
<br/>        随即，电话被挂断了。
<br/>
<br/>        童筱颖本能地回拨了过去，听筒里却传来对方是空号的提示音。

<br/>        到底是谁给她下达的这通任务？
<br/>
<br/>        她怎么也记不起来了。

<br/>        就在她为这事苦恼的时候，又有一通电话打了进来。
<br/>
<br/>        童筱颖不再看来电显示，而是接通电话后，直接怒吼道：“你到底是谁？为什么要给我下那样的命令？”

<br/>        一声和蔼可亲的自我介绍，让童筱颖的坏心情瞬间烟消云散。

<br/>        “裴姨！”

<br/>        “嗯，好，那我们就在实验室里见面。”裴姨接着说道。
