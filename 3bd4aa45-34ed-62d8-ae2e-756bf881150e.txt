        女孩不施粉黛，素颜的皮肤依旧嫩得出水，柳眉大眼，小嘴粉莹，长得十分漂亮。
<br/>
<br/>        洛羽第一眼时，有些看出神。
<br/>
<br/>        连星儿却误以为是他一时间还没有想起自己，连忙接着帮他回忆：“就是小时候，在公园里，我被一条恶狗追，跌倒在地，其他小朋友都躲起来了，只有你跳出来帮我把恶狗赶跑，救了我。”
<br/>
<br/>        她这话一出，身后的女粉丝们纷纷笑了起来。
<br/>
<br/>        “这是什么鬼搭讪方式啊？”
<br/>
<br/>        “就是！”
<br/>
<br/>        “拿了签名照，就赶紧走吧！别耽误洛羽休息了！”
<br/>
<br/>        “对啊！”
<br/>
<br/>        女粉丝们随即又一个个义愤填膺起来，生怕连星儿耽误了洛羽的休息时间。
<br/>
<br/>        洛羽温和地微笑着，将签好名地照片递给连星儿。
<br/>
<br/>        连星儿见洛羽没表态，接过照片，悻悻地说了一声：“谢谢。”
<br/>
<br/>        各自拿到签名照后，群主又整队点名，他们不能在这里随意逗留，必须在规定的时间内离开。
<br/>
<br/>        出了电影城后，群主便对大家说可以自由活动，想同路回家的可以来她那儿报名，想继续留在云城旅游的可以自行随意。
<br/>
<br/>        连星儿瘪着嘴，将洛羽的签名照夹到了笔记本中，然后又将小纽扣挂回了脖子上。
<br/>
<br/>        当她准备离开的时候，身后突然传来呼唤声：“小美女，你等等！”
<br/>
<br/>        连星儿没在意，以为是在喊别人，步伐继续前进，直到一个人影突然窜到她面前，将她吓了一跳，才停下了步伐。
<br/>
<br/>        “你是……洛羽的经纪人？！”连星儿顿感受宠若惊。
<br/>
<br/>        起初，和他们群主在交接一些事宜的女人，就是洛羽的经纪人。
<br/>
<br/>        “对，我是洛羽的经纪人，我叫林苏。大家都叫我苏苏姐，你也可以这么叫。”林苏微笑着自我介绍道。
<br/>
<br/>        连星儿立即欣喜地向林苏点了下头：“苏苏姐，你好，我叫连星儿，是洛羽的死忠粉。”
<br/>
<br/>        “是洛羽让我来找你的，他说，如果你方便的话，可不可以先在我们住的酒店里先安顿下来。等到晚上，他休息了，再过来找你。”林苏温和亲切地说道。
<br/>
<br/>        连星儿高兴得连连点头，应道：“嗯，我很方便的，都听你们的安排。”
<br/>
<br/>        “那就这么定了，这是我们住的酒店的房卡，你先把行李放过去。晚上洛羽会来找你。”林苏接着递给了连星儿一张房卡。
<br/>
<br/>        连星儿见房卡上标注的酒店，是五星级的酒店，立即便接了过来，并很有礼貌地颔首谢道：“谢谢你！”
<br/>
<br/>        “不要跟任何人说噢！这是我们的秘密！”林苏抬起手来，在红唇上比了一个噤声的手势。
<br/>
<br/>        连星儿乖巧地点了点头。
<br/>
<br/>        她回到酒店，整理自己的行李时，和顾千凝道了别，然后离开酒店拦了一辆出租车去了洛羽所住的五星级酒店。
<br/>
<br/>        明丽五星级大酒店里，正举行一场国内网络安防竞技比赛。
<br/>
<br/>        明司寒作为这次MS战队的老板，正目不转睛地盯着台上自己队员的表现。
<br/>
<br/>        当主持人宣布MS战队获胜时，明司寒情不自禁地扬了扬嘴角。
<br/>
<br/>        比赛结束后，他接到了二哥明希哲打来的电话。
<br/>
<br/>        “比赛赢了吗？”明希哲问道。
<br/>
<br/>        明司寒得意地笑道：“你不看看我是谁。”
<br/>
<br/>        “看样子是赢了。”明希哲的声音听起来却十分丧气。
<br/>
<br/>        明司寒笑容渐敛，关问道：“二哥你最近很消沉吗？”
<br/>
<br/>        “我现在就在云城拍戏，下午可以休息，我来明丽酒店找你。你陪我喝喝酒。”明希哲接着说道。
<br/>
<br/>        明司寒笑着应道：“好，那我们今天不醉不归。”
<br/>
<br/>        挂了电话后，他走出比赛现场的大门往电梯口走去时，看到了一抹熟悉的身影。
<br/>
<br/>        女孩正抓着行李箱的拉杆，站在电梯门前等电梯。
<br/>
<br/>        明司寒看到这个女孩时，有些头疼地挠了挠额头，但又不得不故作淡定地走过去。
<br/>
<br/>        电梯抵达后，门一开，连星儿拖着行李箱走进去按了楼层按钮，便低头看手机，且很自觉地往里边站。
<br/>
<br/>        明司寒跟着进去后，利索地背过身去，准备按楼层按钮十，看到女孩按的楼层跟自己一样，一脸无奈地挑眉笑了笑。
<br/>
<br/>        冤家果然路窄！
<br/>
<br/>        “叮——”楼层到了，电梯门自动打开。
<br/>
<br/>        明司寒先行走了出去，走到自己的房门口，掏出房卡准备刷卡进门时，他又下意识地往一旁看去。
<br/>
<br/>        就隔壁房间门口，只见那个女孩一手握着手机和行李箱的拉杆，一手拿着房卡刷卡进门，完全没留意到他的存在。
<br/>
<br/>        反倒是他，居然在莫名其妙地“窥视”她。
