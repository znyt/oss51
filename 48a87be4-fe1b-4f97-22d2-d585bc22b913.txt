　　斗尊呐，可不是斗王斗皇随处都能见到，哪怕是在中州也都属于一方霸主。
<br/>　　在他们一群学员眼中，斗尊就是传说中的存在，如果斗帝是神，那么他们的地位与神已经无比接近。
<br/>　　激动的同时，心中也升起一股难以言喻的恐惧，看向彩鳞怀中的小医仙，目光满是复杂之色。
<br/>　　后者到底是什么身份，居然能牵扯出一位斗尊。
<br/>　　不仅学员，就连迦南学院的长老们也都惊骇莫名，不明白为何好端端的出现一位斗尊强者！
<br/>　　特别是拦住彩鳞去路的三位斗宗长老，他们此刻已经萌生退意，在斗尊无与伦比的威压下，灵魂都颤栗起来。
<br/>　　“你先带她离开，本尊断后，想走恐怕也不容易……”
<br/>　　瞥了眼彩鳞，天火尊者语气凝重，随后目光看向百米外的虚空。
<br/>　　只见视线尽头的空间轻轻波动了一下，旋即一名身穿紫袍，发丝苍白的老者出现在虚空中。
<br/>　　老者双手瞬间结印，将整片空间全部封锁。
<br/>　　一时间，整片空间仿佛化作沼泽，上空如同压着一座高山，令人寸步难行。
<br/>　　“来路不明，并且破坏学院的规矩，创立学院以来多久没有过了，为了学院的颜面，老夫不得不留下你们。”
<br/>　　老者话语刚落，沧桑的双眼定格在天火尊者身上。
<br/>　　“阁下虽同是斗尊，但只是魂体，一身恐怖威能保留下几成？莫以为真能肆无忌惮不成？”
<br/>　　天火尊者眼神眯了眯，捋了捋白花花的胡须，道：“呵呵……本尊成名的时候，你估计都还在穿开裆裤呢，魂体如何，就凭你也想拦住？”
<br/>　　老者白眉一挑，微怒道：“既然如此，那就……”
<br/>　　“嗡嗡……”
<br/>　　然而，老者话语未落，脸色巨变，天火尊者瞳孔也剧烈收缩。
<br/>　　因为，在这一刻，他们的身体全都被禁锢在上空，丝毫不得动弹，能在瞬间禁锢住斗尊的……
<br/>　　学院的长老们，成千上万的学生们，脸色皆是吓的苍白无比，静若寒蝉。
<br/>　　迦南学院范围内，凭空出现一股极致的威压，天空在颤抖，仿佛要被凭空出现的威压给压的倾塌下来一样。
<br/>　　空间凝固，真正意义上的凝固，风云骤变，并且上空响起阵阵如闷雷一样的声响。
<br/>　　“咔咔……”
<br/>　　正当所有人处于惊惧的时候，蔚蓝的天空上响起一连串的破碎声，随后露出一抹令人余悸的黝黑。
<br/>　　广场上，唯有古薰儿一人瞳孔剧烈收缩，脑海中响起了某些信息，不敢置信道：“横渡虚空，难道是太虚古龙族？”
<br/>　　在所有人惊骇欲绝的目光中，裂缝越来越长，随后只觉得天空骤然一亮。
<br/>　　不知何时，一柄神剑横空，出现在他们的视线中，紧接着，黝黑的裂缝渐渐敞开，一道身穿白袍的身影自黝黑的虚空中迈出。
<br/>　　来者，是一名青年。
<br/>　　黝黑长发随风飘散，俊逸非凡的容貌仿佛是上天的杰作，狭长的眼眸藐视众生，腰挂龙形玉饰，浑身满是出尘高贵的气质。
<br/>　　“如果她出现任何差池，迦南学院就准备陪葬吧……”
<br/>　　平静的话语宛若惊雷回荡在上空，并且清晰的落入迦南学院一众长老耳中。
<br/>　　旋即，空间一松，青年包括在场几道身影已经不知所踪。
<br/>　　刚才的青年与话语仿佛昙花一现，更像是一种错觉，令在场所有人都有种身处幻境中的感觉。
<br/>　　虚空中，紫袍老者浑身早已被冷汗浸湿，满是皱纹的脸上露出一副惊魂未定的模样。
<br/>　　许久，惊骇出声道：“斗……斗圣！”
<br/>　　更有许多人不敢相信自己刚才看到的一幕，学院副院长琥乾，学院大长老苏千，包裹赛场中的古薰儿与萧炎。
<br/>　　巨尺撑在地上，萧炎清秀的脸上挂满冷汗，暗自传音，道：“师父，刚才的青年似乎是……是慕师兄吧？”
<br/>　　黝黑戒指闪过一抹亮光，但并未回应。
<br/>　　古薰儿眼中满是不敢置信，低喃道：“当初的猜测果然不错，他也是八族其中的一员，只是他的实力未免也太恐怖了些，恐怕连爹爹……”
<br/>　　琥乾则站在看台上，双眼望着上空渐渐合拢的虚空裂缝怔怔出神。
<br/>　　虚空中的紫袍老者离去，苏千第一个回过神来，目光复杂，颤声道：“选拔赛终止，前五十名进入内院，学院高层全部到内院集合。”
<br/>　　……
<br/>　　……
<br/>　　此时，慕清霄已经抱着小医仙出现在了内院天焚炼气塔的最底端。
<br/>　　在他的肩上，七彩小蛇吐着蛇芯，金色的蛇瞳中透着一抹幽怨的光芒，门口处则坐着一名白袍老者，正是天火尊者。
<br/>　　将小医仙身躯平放在地上，慕清霄查探了下她的情况，脸色总算是缓和了些。
<br/>　　封印已经非常薄弱，但他的实力已经早就超越此位面的最巅峰强者，封印厄难毒体，轻而易举。
<br/>　　盘坐而下，双手结印，掌心一缕黑焰跳烁，轻轻落在小医仙的小腹上，最后一副太极八卦图以她为中心展开。
<br/>　　“阴阳八卦印，仙品以下最高级封印。”
<br/>　　神火钻进小医仙的体内，阴阳八卦成形，绝美脸颊上的绚丽光彩也在下一刻消失。
<br/>　　望着小医仙绝美的容颜上露出一抹恬静，慕清霄目光看向七彩小蛇，语气温柔道：“彩鳞，幸苦你了。”
<br/>　　“你知道就好，本王要休息一会。”
<br/>　　清冷且温柔悦耳的声音响起，七彩吞天蟒吐了吐蛇芯，顺着慕清霄的手臂最后盘在了他的手腕上。
<br/>　　慕清霄知道，彩鳞从刚开始就一直在用斗气压制小医仙体内的毒气，否则，毒体可能早就爆发。
<br/>　　“前辈，刚才多谢了。”
<br/>　　天火尊者微微一愣，摆了摆手道：“本就是老夫欠你的，不过你给老夫的惊喜……惊吓可不是一般的大啊，现在的你究竟有多强？”
<br/>　　慕清霄微微一笑，道：“有一句话叫，斗圣之下皆蝼蚁。”
<br/>　　“老夫记得，五年前在底下与你相遇的时候，你也才斗皇境界吧？”
<br/>　　天火尊者指了指练气塔中央的岩浆池，沧桑的眼眸中满是唏嘘。
<br/>　　五年的时间，当年斗皇已是斗圣，若非亲眼所见，他定会嗤之以鼻。
