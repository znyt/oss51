　　今天想跟大家请个假，没办法更新了，也顺便说一下这半年来大回炉的断更原因。
<br/>　　昨天失眠到七点睡不着，今天一整天的精神状态都极差，匆匆写完《龙邪》晚上的更新，老书却没有状态写了。
<br/>　　其实这种疲惫已经持续了二十几天，因为我这些天都在外地参加一个培训，每天早上八点到晚上六点都是没有时间的，经常码字写到凌晨两三点，第二天一大早又爬起来。
<br/>　　感觉身体到达一个极限负荷状态，我想坚持下去但似乎状况不允许。
<br/>　　我一直有失眠头疼的毛病，好像写小说的都毕竟容易焦虑失眠，神经衰弱已经变成常态了，睡前基本上都要靠褪黑素维持。
<br/>　　褪黑素这种东西我都吃了快两年，效果已经不太好，不知道继续这样下去，是否要开始医院配药……
<br/>　　我入行比较晚，记得2015年准备写大回炉的时候，恰逢我大学最后一年的毕业季。
<br/>　　我是在加拿大读的大学，那里的毕业跟国内不知道是否一样，总之整个人都忙的像个陀螺，为了写小说几乎无限度的压榨睡眠时间。
<br/>　　然后有一天，我拿着最后一门课的演讲稿加强记忆，一边和朋友在一家韩国餐厅吃饭，脱稿演讲，还有一个问答环节，特别害怕会被问到一些我不懂的单词。
<br/>　　我是个非常害怕在很多人面前讲话的人，宅嘛，都是玩游戏看小说的二次元生物，不喜欢与人交流的。
<br/>　　于是我就在这种高压下晕倒了……
<br/>　　我朋友吓飞了，餐厅老板娘吓傻了。
<br/>　　大概是不到十秒钟的昏厥，那种感受很可怕，像是灵魂出窍，我能听到老板娘紧紧把我抱在怀里的慌张叫喊声，还有周边人乱七八糟的声音。
<br/>　　但却感觉很远，好像在离我而去，我在另一个时空还是次元听到这些，没有意识和感知。
<br/>　　那一瞬间我的感受很恐怖，回想起来简直像是死亡……
<br/>　　说这些不是在博取什么同情，只是希望读者能理解。
<br/>　　今年的上半年，我的失眠在不断加重，常常三、四十个小时不睡觉。
<br/>　　加之写了三年，似乎深入了这个圈子，看清了一些事情，看多了一些人。
<br/>　　开始也对网文圈有极强的抵触心理，大家是来看我小说的，不是来看我长啥样的。
<br/>　　大回炉的断更一直到我开始写《龙邪》才开始好转，写新书的时候感受到的压力几乎没有，因为反正没什么人看，随便写写，随便更更。
<br/>　　这种心理让我重新找回了对小说的热情，也让我意识到我放弃很多东西来写小说，至始至终都是因为两个字：
<br/>　　热爱！
<br/>　　无关人和事，我只是喜欢写小说而已。
<br/>　　这篇单章明天恢复更新时会删掉。
<br/>　　谢谢大家。
<br/>(本章完)
