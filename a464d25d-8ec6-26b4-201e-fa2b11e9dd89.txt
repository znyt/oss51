　　见慕清霄不肯收手，圣姑脸色难看到了极点，险些将银牙都给咬碎。
<br/>　　被魂剑给锁定，圣姑知道自己避无可避，唯有接下这一招，身影如鬼魅一般偏离灵溪谷的范围。
<br/>　　她与蜀山有些渊源，曾经听说过《御剑伏魔》是蜀山掌门清微的最强剑诀，千年前曾一剑斩杀魔尊强者。
<br/>　　慕清霄能使出清微掌门的仙法，当真让她惊骇欲绝。
<br/>　　她知道这一招的强大，唯有偏移灵溪谷，避免牵连到谷中的紫萱。
<br/>　　望着凝聚灵气护体的圣姑，慕清霄食指与中指一凝，魂剑在光线的照射下，散发着璀璨的光芒，极其刺眼。
<br/>　　“锵锵……”
<br/>　　剑光璀璨至极，剑意威压也不断攀升，直到最后，几乎犹如一轮耀日，让人不敢直视。
<br/>　　慕清霄脸色也有些沉重，虽说在蜀山的时候就尝试过，但真正全力施展出来，真的是第一次。
<br/>　　仙品剑诀的消耗，超乎他的预料，直接将阴阳湖中的灵气抽掉了五分之一，若是普通修士施展，恐怕体内灵气会瞬间枯竭。
<br/>　　“去！”
<br/>　　轻喝一声，隔空一指，魂剑化作极致璀璨的光芒，整片天空仿佛都在这一刻凝固。
<br/>　　“咻咻……”
<br/>　　魂剑剑鸣声响彻长空，没有剑气，唯有璀璨剑身，一闪即逝，天地失色，周围任何事物都显得黯然无光。
<br/>　　魂剑划破长空，在虚空中留下一道漆黑的裂缝，仿佛要将天空一分为二。
<br/>　　圣姑未曾反应过来，一道璀璨的剑光就映入眼帘中，余悸的气息令她汗毛炸立，步伐一乱，顿时觉得腰间传来一阵刺痛。
<br/>　　“噗噗……”
<br/>　　圣姑眼前一黑，口中瞬间喷出几口鲜血，脸色煞白无比，足足几十秒后才稳住心神，眼中满是忌惮。
<br/>　　好在步伐一乱，恰巧躲开刚才的那一剑，否则可不止是重伤这么简单了。
<br/>　　不过，刚才的一剑也让她看出，后者虽能施展出仙法，但根本没有地仙那般可怕，不然她早就消香玉陨了。
<br/>　　此刻，让她惊恐的不是仙法，她不信慕清霄以区区大乘九重巅峰的境界能够连续施展，而是他手中的剑。
<br/>　　“你的剑有古怪，为何我的灵魂会受创！”
<br/>　　看都不看一眼虚空中被斩出的裂缝，圣姑眼中满是复杂。
<br/>　　没想到阴沟里翻了船，以她渡劫九重巅峰境界，距离地仙境只有临门一脚，现在却被一名大乘修士重创。
<br/>　　慕清霄手持魂剑，用灵气将剑身上的血液驱散，没有去理会圣姑的话语。
<br/>　　刚才的一招，让他略微有些失望，果然修士比不得仙人，现在的他施展的仙品剑诀都只是雏形而已。
<br/>　　不过，就算是雏形配上魂剑，也绝对是敌人的噩梦。
<br/>　　魂剑可是他的底牌，拥有直接斩杀神魂的效果，哪怕你是天上仙神，除非拥有护魂法宝，否则都无可避免。
<br/>　　与其说魂剑是一柄剑，倒不如说它是实体的灵魂攻击。
<br/>　　强大的修士就算被杀后，灵魂依旧能够活下来，但在慕清霄手中，唯有一死，绝无侥幸。
<br/>　　抬起魂剑，慕清霄嘴角牵起一抹笑意，道：“算你运气不错，若是我迈入渡劫期，这一剑能直接要你的命，将路让开吧……”
<br/>　　“不可能！”
<br/>　　抹去嘴角的血迹，圣姑脸色苍白到了极点，咬牙道：“保护女娲后裔是白苗族的使命，就算是死……”
<br/>　　“圣姑，让他进来吧。”
<br/>　　就在这时，灵溪谷中突然传来一道空灵的声音，声音悦耳，让圣姑将到嘴边的话语给咽了下去。
<br/>　　慕清霄微微一笑，道：“早这样不就好了，非要逼我动手，你的伤势非常严重，若是不及时治疗，恐怕终身难以迈入地仙境。”
<br/>　　“拜谁所赐……”
<br/>　　圣姑狠狠的瞪了他一眼，右手握着柳腰，身影一闪就向灵溪谷中掠去。
<br/>　　慕清霄将魂剑收起，一个闪身就出现在灵溪谷的竹楼前。
<br/>　　“进来吧。”
<br/>　　竹楼中传来悦耳的声音，慕清霄也不犹豫直接迈入其中，随后映入眼帘的是一名身穿紫色薄衫的绝美女子。
<br/>　　大乘期修士。
<br/>　　慕清霄目光闪烁了一下，这紫萱的容貌果然绝美，修为也不弱。
<br/>　　相貌虽美，但他却没有忘记来此的目的。
<br/>　　盘坐在桌前，紫萱打量了一眼慕清霄，语气微怒，道：“紫萱只是一介小女子，可不曾记得，何时得罪过阁下。”
<br/>　　“你确实不曾得罪过我，但你身上却有我需要的东西。”
<br/>　　慕清霄也不在意，随意的坐在桌前，道：“你是聪明女人，不像刚才那女人一样古板，将水灵珠给我，免得我动手。”
<br/>　　紫萱瞳孔一缩，水灵珠可是她们一族的秘辛，后者是如何得知？
<br/>　　千百年来，水灵珠都由女娲后人所持，是历代女娲后人代代相传的圣物，现在确实在她体内，此事只有她与圣姑知道。
<br/>　　圣姑根本不可能背叛女娲，否则刚才也不会誓死保护她。
<br/>　　“你不必惊讶，我得知水灵珠在你手中只是个偶然，将灵珠给我，你也能放弃女娲后裔的使命，不必在受拘束，不是你一直都想要的吗？”
<br/>　　“你到底是谁？”
<br/>　　紫萱目光一凝，猛的站起身来，向后退了几步，这可是她藏在心里的秘密，后者为何如此了解她。
<br/>　　慕清霄摆了摆手，语气平静道：“我是谁不重要，我不想在说第三遍，水灵珠，徐长卿的命，你自己选吧。”
<br/>　　“水灵珠乃是女娲后裔代代相传的圣物，绝不可能给你，你身为强者，难道就没有原则，居然威胁我一个小女子！”
<br/>　　“原则？”
<br/>　　慕清霄觉得有些好笑，道：“道理都谁都懂，原则就是用来打破的，不是吗？”
<br/>　　“即使没有水灵珠，你也不会死，修为没了可以重修，你不是早就准备抵抗命运，没有水灵珠，徐长卿也不会死，聚时你们做一对神仙眷侣岂不是更好？”
<br/>　　惊恐的看了一眼慕清霄，紫萱怎么也想不到，来者将她的心思摸得一清二楚。
<br/>　　甚至，后者口中所言正是她心中所想，在这一瞬间，她居然有些心动了。
<br/>　　见状，慕清霄嘴角牵起一抹弧度，道：“就算你不给，我也会亲手取出来，只是可怜了你肚子里的孩子。”
