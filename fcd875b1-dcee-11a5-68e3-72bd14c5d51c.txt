　　那里有一滩血迹，异常显眼，但只有从叶子博所在的角度，穿过草丛根部的空隙才能看见，正常站在这片地带的人，视野则是会被上方的茂密杂草挡住，而丝毫发现不了这一异常。
<br/>　　叶子博瞪大了眼睛，死死的盯着那片腥红地带。
<br/>　　血迹的中央躺着一个人，整个胸膛都被穿透，甚至上半身用开膛破肚来形容也不为过，看上去血腥无比，更是死相无比凄惨，就像是被野兽活生生的撕开一般。
<br/>　　只是野兽撕开腹部，会和丧尸和异种一样，毫不客气的将这新鲜的躯体吃下肚。
<br/>　　可当下这些血液还在流淌，还是红色，带着温腾的热气，说明这是刚刚发生过的事。
<br/>　　周围没有野兽，更没有异种和丧尸，这具尸体也没有被啃食的痕迹。
<br/>　　杀掉此人的必然也是人类！
<br/>　　打斗？惨死？
<br/>　　叶子博吃力的用尽生命尽头的最后一丝力气，朝着此人的脸部望去，而后一股颠覆认知的感觉，就如同火山喷发一般呼啸而起。
<br/>　　让他猛然一口血喷出，让自己的整张脸都看上去血肉模糊。
<br/>　　那具尸体的主人，是张玄！
<br/>　　被叶子博以及叶家人误以为逃脱的张玄，竟然死了，悄然无声的死在这里！
<br/>　　震惊让叶子博的情绪喷发，在生命的最后一刻，思维都陷入无解的循环中。
<br/>　　是谁？
<br/>　　是谁杀掉了张玄？
<br/>　　那架直升机，是谁在驾驶？
<br/>　　除了他还张玄，此处还有谁埋伏并等到了最后一刻？！
<br/>　　叶子博的双目瞪到最大，眼神里带着惊恐，生命气息就这么慢慢消失，哪怕死了都无法再知道答案。
<br/>　　那些士兵和叶家人看到叶子博喷了一口血后死亡，都没有太过于在意，反而是对叶家这些年轻人的未来行程更加担忧。
<br/>　　“你们既然知道去哪，也不方便说的话，我们也不好多问。”一名为首的士兵开口，带着一丝劝阻：“虽然我不知道你们这些姓叶的到底有什么秘密，但还是要奉劝一句，东边可不好混，远不如我们这里来的生活安逸。”
<br/>　　“我们知道，但我们必须要去。”叶家年轻人中的老大回应，眼中带着感激：“这些天多谢你们，还有杨飞杰首领，这一次更是让你们跟着白跑这么远。”
<br/>　　士兵首领摆着手：“我们高原基地就是这样，原本居民就不多，能帮上的都会帮，不过既然你们决议如此，相信首领也不会阻拦，不过走之前还是先回一趟基地，你们不会还想徒步吧？那又是要一年半载！”
<br/>　　听到此话，叶家年轻人纷纷目光大亮。
<br/>　　“杨飞杰首领会借我们直升机么？”叶家年轻人问道。
<br/>　　士兵中的小首领想了想，道：“这还需要回去亲自问他，不过你们爷爷的事情我们首领也察觉到了不简单，说不定能帮上什么。”
<br/>　　“那好，多谢！”叶家年轻一代的老大重重点头，当即带着众人就往回走。
<br/>　　士兵首领也跟上大部队，走之前摇摇头下意识的呐呐自语：“这地方血腥味真重，这几个年轻人下手也太狠了，把叶子博捅的血都要流干了……”
<br/>　　一群人快步离开，张玄被截杀在此的秘密，最终也没有被他们发现，唯有叶子博死不瞑目的双眼死死瞪着那个方向。
<br/>　　——————
<br/>　　此时高空中快速飞行的直升机机舱内，叶永阳悠悠睁开眼，顿时被眼前的场景惊呆，窗外呼啸而过的高空，以及眼前这陌生无比的场景，都让他有种错乱的感觉。
<br/>　　年迈的他禁不住张玄攻破大门时的震荡，之后就一直处于昏迷中，并且当下也没有能力动弹，浑身上下的骨头仿佛散架，唯有一双浑浊的双眼在尽可能的观察四周。
<br/>　　直升机？
<br/>　　很快叶永阳就反应了过来，也将渐渐清晰的视线放在了坐在驾驶位上的顾凉晨。
<br/>　　“你，是谁？”干涩的声音从叶永阳的口中发出，带着无尽的疲惫，
<br/>　　这一天的折腾让他经受不住，此时还能醒过来，已经算是身体机能超越常人的老头。
<br/>　　顾凉晨在与张玄一战后精力消耗颇大，之后又强行驾驶着直升机，整个人面色都苍白如纸。
<br/>　　此刻听到叶永阳的声音，顾凉晨一贯冷静的态度让人察觉不到丝毫异常：“救你的人。”
<br/>　　这一回答让叶永阳眉头皱起，满是皱纹的脸上也带着迷茫，他随后问道：“那个把我带出基地的人呢？”
<br/>　　顾凉晨的声音依旧平静无比：“死了。”
<br/>　　“死了？！”叶永阳忍不住一声大呼，更牵动着快散架的身体都疼痛起来，他‘嘶’的一声倒吸一口凉气，而后平复了心情，带着警惕问道：“怎么死的？那个人是谁？你又要带我去哪？”
<br/>　　“他叫张玄，白家花钱请的八阶高级猎者，搭上了叶子博的顺风车来抓你。”顾凉晨三两句将事情的脉络讲解的清晰无比，而后一顿道：“张玄被我杀了，我抢了叶子博的直升机带你去狼牙。”
<br/>　　叶永阳愣了愣：“狼牙？狼牙基地？你是……”
<br/>　　“我是狼牙战团高级军官，军衔中将，楚涵上将的直属暗谍。”顾凉晨没有隐瞒的回答。
<br/>　　叶永阳情绪起伏不定，眼神更是不断变化：“想不到楚涵上将也把目标放在了我身上，我劝你还是送我回去吧，我什么都不会说的。”
<br/>　　这句话说的明显带有排斥意味，顾凉晨也不在意，淡淡开口：“如果你想被越来越多的白家，甚至其他神秘家族派来的人截杀，高原基地不断发生今天这样的事，这些你都无所谓的话，我可以送你回去。”
<br/>　　叶永阳愣住，不禁大为诧异：“你难道不是冲着我来的？”
<br/>　　顾凉晨没有明确回答这一点，而是转而更为详细的解释道：“以神秘家族的手段，你叶永阳出现在高原基地的事，轻而易举的就能查到吧？只要暴露一次，那个地方就不再安全。我时间紧迫，只来得及带走你一个人，你们叶家那几个后代做事不够稳重也易冲动，我一来是来不及全部带走，二来也不想让他们暴露太多，相信这也是你所希望的，毕竟他们要找的只是你。”
