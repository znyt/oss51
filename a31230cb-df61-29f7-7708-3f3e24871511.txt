　　一路飞行。
<br/>　　公孙火舞之事，萧逸早便忘却。
<br/>　　大半天后，他在某处地域停下。
<br/>　　事实上，以他现今实力，全速飞行下，数个时辰就能横跨一个地域。
<br/>　　大半天时间，够他飞跃两个以上的地域了。
<br/>　　也就是说，刚才遇到的那群邪修，并不是那片地域的邪修。
<br/>　　“弄花地域。”萧逸皱眉自语了一声。
<br/>　　现今位于的地域，乃是弄花地域。
<br/>　　如果没有猜错的话，应该是公孙火舞在这个地域便遇到了这群邪修。
<br/>　　随后在一路追杀和逃遁中，跨了两个地域，到了之前那片妖兽森林。
<br/>　　当然，这不是重点。
<br/>　　从那群邪修身上得来的情报中，那群邪修，便是来自于这弄花地域。
<br/>　　重点是，这弄花地域只是个小地域。
<br/>　　恐怕是个比三风地域这等二流地域还不如的三流地域。
<br/>　　萧逸原本还以为能查到一个大型分部。
<br/>　　可现在看来，应该只是追寻到一个小型的邪修分部。
<br/>　　这种小型分部，有用的情报和资料不会太多。
<br/>　　“算了，见一个端一个。”萧逸摇了摇头，自语一声。
<br/>　　嗖…
<br/>　　萧逸身影闪烁，疾速前行。
<br/>　　不过十数分钟，已在一座大城下停下。
<br/>　　“花影城。”萧逸看了眼城墙上的牌匾，随后再次闪身而入。
<br/>　　最后，萧逸在一座府邸前停下。
<br/>　　“花影城，花家，是这里了。”萧逸淡漠自语。
<br/>　　花影城，在整个弄花地域，算不得什么大城。
<br/>　　而花家，则是花影城第一大家族，传承数百年以上。
<br/>　　这样的小家族，在花影城这样的小地方，足以成为霸主家族。
<br/>　　连花影城城主，也对他们忌惮三分。
<br/>　　在萧逸的情报中，花家，便是弄花地域的邪修分部。
<br/>　　不过，在外人眼中，花家决计不可能会与邪修有什么瓜葛。
<br/>　　花影城内，有着猎妖殿、炎殿、风刹殿等等分殿。
<br/>　　可从无任何一殿，怀疑过花家。
<br/>　　花家，就像别的普通家族一样，和城内各大势力有着交情，有着交际。
<br/>　　家族内第一天骄，便是花影城第一天骄。
<br/>　　他们如正常家族一般，参加各种盛事，参与势力间的比拼等等。
<br/>　　谁曾想，便是这样一个再普通不过的家族，却是整个弄花地域的邪修分部。
<br/>　　看似仅仅是花影城霸主，但实则整个分部的邪修力量，几近有颠覆弄花地域的能力。
<br/>　　萧逸感知了一番，确定无误，直接踏步而入。
<br/>　　花家府邸大门前，有两个守门的武者。
<br/>　　“阁下是？”守门武者，拱了拱手，又下意识地拦下了萧逸。
<br/>　　一般家族门前，都会有守门的武者。
<br/>　　而这两个武者，又显得礼貌过人，其中又夹杂着大家族才拥有的强硬之意。
<br/>　　若非萧逸知晓情报，怕是打死也不相信这会是个邪修分部。
<br/>　　嘭…
<br/>　　萧逸不语，身上气息轻轻一震，便震开了这二人。
<br/>　　连同花家大门，也在瞬间被轰成齑粉。
<br/>　　同一时间，萧逸打出一个屏障，包裹了整个花家府邸。
<br/>　　进入府邸，迎面看到三三两两一些家族武者，但人数并不多。
<br/>　　“何人赶来我花家撒野？”几个家族武者，明显看到被轰成粉碎的家族大门。
<br/>　　萧逸不语，一个闪身，来到花家府邸深处。
<br/>　　一路前行，见到的，都是普通的家族武者，且人数不多。
<br/>　　难怪一直以来都无人发现花家是个小型分部。
<br/>　　拿萧逸来说，就他这般在花家内横冲直撞，也没发现邪修的半分踪迹。
<br/>　　直至此刻来到深处，萧逸指尖轻扬，一道火光打出，破了面前的禁制。
<br/>　　禁制一消，里头，是数个邪修在焚炼着妖兽。
<br/>　　一缕缕黑色的诡异火焰，焚烧着数头妖兽，试图让妖兽折服。
<br/>　　邪道阵盘，亦在缓慢地吸收着妖兽灵识。
<br/>　　“嗯？”禁制忽然被打破，里头几个邪修，先是一惊，随后脸色一变。
<br/>　　“你是何人？”
<br/>　　嗖…嗖…嗖…
<br/>　　花家的其他武者，亦瞬间闪身赶来。
<br/>　　一众花家武者，包围了萧逸。
<br/>　　原本面容正常的家族武者，忽然眼露凶光，身上一缕缕阴寒气息散发。
<br/>　　“果真全都是邪修。”萧逸冷笑一声。
<br/>　　花家武者，同样冷笑，“哪里来的不知死活的小子。”
<br/>　　“非要闯来这里，既发现了这里，便也留不得你。”
<br/>　　“出手。”
<br/>　　一众花家武者，瞬间出手。
<br/>　　萧逸甚至没有半分动弹。
<br/>　　空气中，已是一道道火光爆响。
<br/>　　一个呼吸间，一众花家武者，悉数重伤。
<br/>　　萧逸看了眼周遭，皱了皱眉，“怎地这么少人？其他邪修呢？”
<br/>　　周遭花家武者，只有十数人。
<br/>　　从他进入花家府邸后，花家就没有多少人。
<br/>　　虽然这只是一个小型分部，但怎么也不至于只有区区十来人。
<br/>　　这些邪修分部，毕竟是要在整个地域行事的，起码也要在逾百人以上，甚至更多。
<br/>　　一个地域，动辄过百城。
<br/>　　“你到底是什么人？”周遭邪修，脸色大变。
<br/>　　面前年轻人的实力，竟是一个照面便将他们所有人重伤。
<br/>　　“弄花地域之内，没有你这一号人物。”
<br/>　　“我想起来了。”忽然，一个花家长老模样的老者惊声道。
<br/>　　“易霄，这家伙是易霄，前些天连毁我们邪君府数个分部。”
<br/>　　“护法早已传来画像。”
<br/>　　“画像？”萧逸眉头一挑。
<br/>　　一个闪身，来到这老者身前。
<br/>　　感知一番，取出了一个乾坤戒。
<br/>　　只不过，乾坤戒内，根本无半分情报卷宗。
<br/>　　“紫炎易霄，你不必白费力气了。”花家老者冷笑一声。
<br/>　　“护法早便交待好我们，你在弄花分部内，一份情报都休想查到。”
<br/>　　“桀桀桀桀。”
<br/>　　周遭邪修，狞笑不停。
<br/>　　下一秒，身上竟是一阵气势爆发。
<br/>　　“想自爆？”萧逸瞬间反应了过来，手中一道火焰打出。
<br/>　　火焰，瞬间包裹了周遭邪修，封住了他们的元力与小世界。
<br/>　　这群邪修，最强者不过是圣皇境后期。
<br/>　　在萧逸面前，毫无还手之力，就是想自爆也做不到。
<br/>　　这就是巨大实力差距下带来的强弱之分。
<br/>　　萧逸眯了眯眼，并未离去，也未杀这群邪修。
<br/>　　而是大手一挥，收回了包裹花家的屏障，同时，庞大而强悍的感知覆盖了整个花家府邸内外。
<br/>　　半个时辰后。
<br/>　　花家府邸外，一个花家武者回归。
<br/>　　还未反应过来，已被萧逸瞬间禁锢。
<br/>　　萧逸身影一闪，来到这个花家武者面前，一番感知，取出了一个乾坤戒。
<br/>　　直觉告诉他，必有收获。
<br/>　　果然，乾坤戒内，一个让他眼熟的东西，瞬间吸引了他的目光。
<br/>　　“武道冰晶？”萧逸眼眸一冷。
