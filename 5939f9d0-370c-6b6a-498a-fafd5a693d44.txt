<br/>　　一座座的星球，山峰，都在这归墟天的空中漂浮这。
<br/>　　当然最大的便是一处巨大无比的大陆。
<br/>　　这大陆之大，一眼望去都看不见尽头，看向这一块大陆，楚风眠都感觉到上面一股股恐怖到极点的气息。

<br/>　　归墟天，不止是有着无数宝藏，其中更是有着无穷武技的危险。

<br/>　　在这归墟天不止是有着远古时代留下的无数宝藏，其中的种种危险也是遗留了下来，这种危险是绝对致命的。

<br/>　　楚风眠观察这周围的环境，不由的开口道。
<br/>　　这周围的环境，跟远古时代几乎是一模一样。
<br/>　　远古时代，曾经有过一次天地大破裂，令环境改变，只有一些区域流了下来，像是北邙圣域，便是其中之一。

<br/>　　而在这归墟天，如今就连这里的环境，都是跟远古时代，一模一样。

<br/>　　楚风眠动用灵识，猛然向着周围扫去。

<br/>　　这些灵药，许许多多都是在这一个时代之中，已经灭绝的远古灵药，果然在这归墟天中，还有生长。

<br/>　　这归墟天，果然是一处遍地宝藏的地方。
<br/>　　仅仅是楚风眠落下这星球上，都有着如此之多的灵药。
<br/>　　“先将这些灵药收取，在准备前去其他地方。”
<br/>　　楚风眠身形一动，便是大肆收取这其中的无数灵药。
