　　又把祖师祠堂门口青铜定劈成两半，苏重被田不易骂了个狗血淋透，只能抱头鼠窜会后山竹屋避风头。
<br/>　　一走进竹屋周围，苏重浑身就一阵舒畅。
<br/>　　“还是自家小屋子舒服啊。”苏重感慨一句。这周围被他用奇门遁法调整，草木精气汇聚，天地灵气浓郁，非常适合生存修炼。
<br/>　　他种在院子里的花草树木生长飞快。药材也长得茁壮茂盛。
<br/>　　“可惜了这些好药材，都用不到了。”他本打算走嗑药练体流的，毕竟这招他最熟，不止在一个世界干过。
<br/>　　“还是庚金剑气更神奇。”田不易虽然懒散了些，这个师傅当的真是没话说。
<br/>　　只是如果让田不易知道庚金剑气本质，让他知道这是一本魔道功法，不知道会不会把他给劈了。
<br/>　　苏重打了个寒蝉，把这个想法赶出脑海。
<br/>　　“罡煞合一后，庚金剑气凶戾气内敛，杀伐力大增，没了凶邪只剩锋利。不亲自修炼谁也不会知道庚金剑气本质。”苏重对他改进后的功法颇有信心。
<br/>　　而且就算被发现也不怕，就以田不易护犊子性子，只要苏重不欺师灭祖，干出什么灭绝人性的事来，田不易就不会把他怎么样。
<br/>　　“得想法子提高庚金剑气品阶。”
<br/>　　他今天能胜过几位师兄，靠的是庚金剑气威力加持。真论起修为，他比几个师兄差甚远了。
<br/>　　“在身体神魂质变方面，庚金剑气还差的远！”苏重想起之前考虑的问题，如何让庚金剑气生生不息。
<br/>　　遇到问题苏重就会钻藏书阁，大竹峰藏书丰富，很多修炼困难，都是在那里得到启发。一头扎进藏书阁，苏重就感觉不到时间流逝。
<br/>　　后山黑竹林，大竹峰一众弟子，除了苏重都已经到齐。几个人人手一把柴刀。今天又到了考校修为的时候。
<br/>　　“练了半年的聚元功，现在就来试试你们手底下到底有几斤尽量。”苏茹背着手瞪着一众嬉皮笑脸的徒弟。
<br/>　　“大仁，你先来。”
<br/>　　“是！”宋大仁拿着一把柴刀，来到一颗拳头粗细的黑竹面前。
<br/>　　今天测试他们聚元功修炼成果。聚元功练的越好，身体素质越强力量越大。测试方式则是砍伐黑竹。
<br/>　　宋大仁入门最早，已经好多年没砍过黑竹。
<br/>　　黑竹生长特殊，越粗的就越坚韧。以前他们练习，都是对着拇指粗细黑竹砍伐，只要砍满三年就算过关。
<br/>　　像这种拳头粗细黑竹，他还是第一次面对。要是没砍断，看师娘脸色他今天就没有好果子吃。
<br/>　　不过，话说哪次考校他们有过好果子吃来着？
<br/>　　集中精神，跳动全身最大力量，一刀狠狠看向黑竹。
<br/>　　噗嗤！
<br/>　　一声闷响，黑竹应声而断。
<br/>　　宋大仁目瞪口呆。修炼聚元功，力量长的飞快。他知道自己力气大，但没有概念。现在他有概念了。
<br/>　　“大师兄威武！”几个师兄在一旁庆祝。
<br/>　　苏茹满意点头，对这个结果还算认可。
<br/>　　青云内力气大的不是没有，能砍断这种黑竹的也有不少。但那都是常年累月用玄清道熏染练出来的。
<br/>　　在宋大仁这个年龄，能单凭力气做到这种程度的寥寥可数。
<br/>　　“下一个！”苏茹不由感慨，小徒弟真妖孽啊。一套功法就把大竹峰实力拉升一截。
<br/>　　几个人挨个上前劈砍黑竹。
<br/>　　虽然不像宋大仁一刀两断那么利落，但也并不困难。就连张小凡，也在十二刀后把黑竹砍断。
<br/>　　“很好，看来这段时间没偷懒。”苏茹高兴，今天放你们半天假，玩去吧。
<br/>　　几个人一愣，这还是严厉的师娘吗？
<br/>　　“怎么，想留下来继续修炼？”苏茹眉头一拧。
<br/>　　一众徒弟顿时做鸟兽散。
<br/>　　“不易，你说让尔旦来教他们修炼怎么样。”苏茹突然开口。
<br/>　　田不易从竹林里漫步走出：“我觉得可以试试。”就像上次一样，每次考校，他都在一旁观看。
<br/>　　“这混小子天赋奇高，在修炼上有独到见解，就是太孤僻了点，整天躲在藏书阁里看书。得让他和其他徒弟们多接触接触。”田不易道。
<br/>　　“走，去看看那个混小子。”
<br/>　　两人来到苏重竹屋，就看到苏重正坐在院子里发呆。
<br/>　　田不易刚想开口，突然顿住。
<br/>　　苏重身周突然升起一阵旋风，天地灵气瞬间汇聚，不要命的往苏重体内钻。
<br/>　　好半晌，苏重睁开眼叹了一口气。他在改进庚金剑气，可依旧失败。
<br/>　　“尔旦，你这是在干什么？”田不易好奇道。
<br/>　　苏重这才发现师傅师娘到来。
<br/>　　“师傅，我在修炼庚金剑气。”苏重如实道。
<br/>　　“庚金剑气有这种效果？”田不易疑惑，刚才那种疯狂抽吸天地灵气的现象可不多见。
<br/>　　“庚金剑气没有，我给它改了改就有了，只是效果不太好。”苏重说着又眼神迷离陷入沉思。
<br/>　　苏茹和田不易对视一眼不由苦笑。这个小徒弟天赋惊人，就是有些痴。一旦涉及修炼就容易沉迷。
<br/>　　“尔旦，师傅和你商量个事。”田不易咳嗽一声道。
<br/>　　“师傅请说。”苏重回过神恭敬道。
<br/>　　“你那几个师兄一个比一个懒散，我和你师娘是没招了，你能不能帮忙教教他们。”田不易道。
<br/>　　“我？”苏重诧异，怎么想到让自己替他们教徒弟的。
<br/>　　“对，自从上次你一招把他们击败后，你没发现你这些师兄都躲着你走吗？”田不易苦笑。
<br/>　　自己扮演了那么多年的严师，这些徒弟在他面前却一个个嬉皮笑脸。自家小徒弟却一剑就把众人吓住。
<br/>　　苏重想了想，“如果只是让他们好好修炼，我觉得还是没问题的。不行我就和他们切磋剑法，想来他们也不好意思用法宝欺负我。”苏重笑眯眯，很快有了对策。
<br/>　　苏茹险些笑出来，看来这招是走对了。自己那些徒弟看来要倒霉了。
<br/>　　“徒弟，你刚才说，你在修改庚金剑气？”田不易突然反应过来。修改修炼功法，这是普通人敢干的事吗？
<br/>　　“啊，就是效果不太好。”
<br/>　　还效果不太好，没把自己练死就已经是侥天之幸啦！
<br/>　　“功法岂能擅自修改！”田不易勃然大怒，擅自修改功法这不是作死吗！
<br/>　　“师傅息怒，不修改庚金剑气，根本没法练啊。”苏重无奈，要真按原版修炼，他现在坟头青草都得三丈高啦！
<br/>　　“你是说，你早就在修改功法？”田不易还能说什么。
<br/>　　自己等人哪个不是抱着功法小心翼翼前进，生怕出点儿差错。
<br/>　　自己这个小徒弟倒好，直接对功法动手。而且还成功啦。
<br/>　　田不易生气也没办法，他当然知道原版庚金剑气的霸道，不改功法根本就是在要命！
<br/>　　“可有什么困难。”田不易苏然。修改功法哪里那么容易。
<br/>　　“师傅，您那里有什么奇绝阵法吗？”
<br/>　　“阵法？什么作用的？”田不易对徒弟还是蛮大方的。
<br/>　　“牵引、封印、调理灵气这方面的。”
<br/>　　田不易和苏茹对视一眼，脸色突然凝重起来。
<br/>　　天机印！
<br/>(本章完)
