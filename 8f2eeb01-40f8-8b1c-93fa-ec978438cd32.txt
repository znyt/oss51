夏夕绾抬头，撞上了陆寒霆那双狭眸。
<br/>
<br/>        陆寒霆坐在牌桌的主位上，身上是精良的黑色衬衫黑西裤，他在抽烟，修长的两指里夹着猩红的火苗，吞云驾雾。
<br/>
<br/>        霍西泽喊的时候，陆寒霆刚抽了一口烟，然后他抬头看向了夏夕绾，烟雾缭绕模糊了他的俊颜，不过隐约看到他英气的眉心微微蹙了起来，几秒后将嘴里的一口烟雾缓缓吐了出来。
<br/>
<br/>        夏夕绾被强推了进来有些尴尬，现在撞上陆寒霆，更尴尬了几分。
<br/>
<br/>        “霍少，这个小美人是从哪里来的，难道你们1949最漂亮的姑娘还不在这里，霍少你太不够意思了，竟然藏着掖着。”有老总笑道。
<br/>
<br/>        很明显这些男人将夏夕绾当成这里陪酒的姑娘了，对此霍西泽打量了一下对面陆寒霆的面色。
<br/>
<br/>        陆寒霆已经收回了目光，将手里的一张牌给丢了出去，神色深沉淡漠。
<br/>
<br/>        这是什么意思？
<br/>
<br/>        吵架了？
<br/>
<br/>        冷战了？
<br/>
<br/>        装不认识了？
<br/>
<br/>        霍西泽悄咪咪的笑了，太好了，他又可以看八卦了，而且他是看戏不嫌事大，直接招呼夏夕绾，“你，快过来给我们倒酒，怎么一点规矩都不懂呢？”
<br/>
<br/>        有一个油腻的老总伸手就拽住了夏夕绾的纤臂，“小美人，你脸上戴着面纱干什么，看你身材这么好，脸蛋一定也不差吧，摘下面纱让大爷们看一看啊。”
<br/>
<br/>        油腻的老总直接去揭夏夕绾脸上的面纱。
<br/>
<br/>        夏夕绾敏捷的避开了这只咸猪手，秀眉轻拧，“我看你们是搞错了，我不是这里陪酒的小姐，快放开我！”
<br/>
<br/>        “小美人，你知道这里都是些什么人么，你是不是这里陪酒的小姐你说的不算，我们说了才算！”
<br/>
<br/>        夏夕绾想起刚才那个经理千叮万嘱的事情，说这里都是海城的大人物，她得罪不起。
<br/>
<br/>        夏夕绾也不想惹上这些麻烦，但是今天运气太糟糕了，竟然被推进了这个局里。
<br/>
<br/>        “要不我们来打个赌吧，赌这个小美人摘下面纱后，究竟是貌丑无颜，还是貌似天仙。”
<br/>
<br/>        “好啊，这个有趣，我就赌貌丑无颜吧，如果她貌似天仙早就摘面纱了，要知道美貌可是女人最大的武器。”
<br/>
<br/>        “我赌貌似天仙，因为这个小美人的身材真的是…好的让人忍不住啊哈哈哈。”
<br/>
<br/>        这种场合里男人的谈话总是荤素不忌，夏夕绾被困在里面，成为了男人间的话题。
<br/>
<br/>        今天夏夕绾穿了一件小香风裙子，流苏的裙摆在膝盖下方，乖乖的，她骨架纤柔，不是那种很夸张的S曲线，而是少女娉婷的曼妙，透着几分禁忌的纤尘。
<br/>
<br/>        哪怕她什么都没有露，这豪华包厢里穿着清凉的美女们早就被她给压了下去。
<br/>
<br/>        夏夕绾看了主位上的陆寒霆一眼，男人敛着眼睑在抽烟，身边有美女相伴，没有再看她一眼。
<br/>
<br/>        现在的他高高在上，薄冷无情。
<br/>
<br/>        就算她深陷困局，他也不会出手相助了。
<br/>
<br/>        他将她当成了陌生人。
<br/>
<br/>        虽然这就是她希望的，彼此划清界限，但是夏夕绾依然觉得心里很疼。
