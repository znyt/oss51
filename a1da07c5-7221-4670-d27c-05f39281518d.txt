　　紫霄宫中，诸圣在商议天庭之主一事，对此，众人各有想法，争论不休。
<br/>　　“妖帝虽然陨落，但他的后人还在。妖族十太子陆压，为妖皇血脉，我认为可以为天庭之主。”女娲圣人眼光闪烁说道。
<br/>　　“哼！娘娘说笑了，巫、妖二族不顾洪荒亿万生灵，数次大战，终引发大劫，造成无边恶果，乃不赦之罪。而今虽然大战终止，但二族气数也已尽，却是无法再执掌洪荒天地。至于陆压，十日横空，更是开启三族终战的起因，他算什么东西，有何资格执掌天庭？”
<br/>　　还没等其他圣人说话，武祖玄天便冷笑道。心中对于女娲的做法，更是不满了。
<br/>　　“你……”
<br/>　　听了玄天的话，女娲为之一阵气急，心中对玄天恼怒异常。她还没有在说话，通天便已经说道：
<br/>　　“玄天道友说的对，洪荒此次遭受大难，巫妖二族乃是罪魁祸首，妖族已经是不赦之罪了，何德何能能够再次当天庭之主。依我看，我门下弟子多宝道行深厚，法力高强，颇具福缘，当为天帝。”
<br/>　　“师兄所言虽有道理，但却也未必全对。”
<br/>　　此刻开口说话的是西方教的准提圣人，其实，他对于通天的前半句话还是很认同的，觉得妖族不该再掌控天帝之位，但是后面的话，他就不认同了，什么你门下弟子就可以了，那岂不是我们下弟子也可以了。他当即出声道：
<br/>　　“既然要重立天庭，替天行道，那也不该由通天师兄一人说了算，我西方门下弟子，弥勒，有大才，绝对有能力执掌天庭。”
<br/>　　准提此言一出，立刻就在遭到了通天的反对，只听通天说道：“天庭乃我东方之天庭，与你西方有何关系？”
<br/>　　“不错，天庭之事，西方不可插手。”
<br/>　　这时，太上圣人也睁开眼睛，开口说道。
<br/>　　“你……”
<br/>　　见上清与天清如此说话，顿时让准提心中怒意不止，当下，他想要与他们理论一番，但这时，他的师兄接引开口道：
<br/>　　“师弟，不可。”
<br/>　　“师兄，这……”准提不甘，不明白师兄为何为如此，但接引也只是摇了摇头。却是接引心中十分明白，东方三圣一定不会让他们西方插手东方之事，到时候只会越扯越乱。
<br/>　　这时，元始睁开眼睛，淡淡道：“尔等既然做不出决定，那不如便请老师自作决定。如何？”
<br/>　　女娲娘娘见此，也知道天帝之位以后与妖族无关了，便索性道：“此言大善，天帝关乎三界众生安危，此事还是由老师做主为好。”
<br/>　　他们这话，正好应了道祖的心思。实际上天帝之位道祖心中早已有了人选，此次说是让他们自己商议，实则是想看看各大圣人之间的关系纷争如何。天道最重平衡，不可能让三清一家独大。
<br/>　　如之前，有了巫族，便有了他们的宿敌妖族的出现，有了人族，还是有剩下的妖族，也没有全部灭亡，更何况，人族之上，还有六个圣人。
<br/>　　这一切都是为了平衡，只有这样，才能保证大家的利益，延缓大战的可能。这是御人之道，也是御天之道。
<br/>　　见诸圣如此，鸿钧道祖心中暗笑，面上却淡然无比，口中缓缓道：
<br/>　　“昊天、瑶池，尔等且入紫霄宫来。”
<br/>　　两位站在殿门口的童子，闻言立刻前来殿中拜倒，齐声高呼道：“见过老师！”
<br/>　　鸿钧道祖点了点头，伸手向着两人身上一点，随即，昊天与瑶池身形一阵闪动，仙光缭绕，仙音袅袅之后，瞬间化为一对妙龄男女。
<br/>　　“此次重立天庭，尔等两人也跟随我数十元会了，这番机缘，便给你们吧。你们为两颗混沌顽石所化，一至阳，一至阴，如此正好。昊天当为天帝，瑶池当为昊天之妻，瑶池金母。”
<br/>　　昊天与瑶池见此，顿时大喜，接着便高呼大拜道：“多谢老师。”
<br/>　　昊天眼神一阵闪烁，出声道：“老师，我们无甚灵宝，怎能管理好这偌大的天庭呢？”
<br/>　　这是要向鸿钧要宝啊，诸圣闻言，心中顿时大骂脸皮厚。同时，眼睛也看向鸿钧，想看看他怎么办。
<br/>　　鸿钧也不负众人所望，取出四见物品，宝光流转，光华耀人，气息苍古，正是一塔、一镜、一钗、一灵根。
<br/>　　一塔为昊天塔，又名八宝玲珑剔透舍利子如意黄金宝塔，有收妖魔，镇鬼煞之能，拥有浩大无边之力，能降服一切妖魔，必要之时，连仙神也能降服，有吸星换月之能。
<br/>　　一镜为昊天镜，可以上观九霄，下照九幽，而且还有一些隐秘不为人知。倒是与西王母的昆仑镜有些相似，但却是没有昆仑镜那般强大，也无昆仑镜的穿梭虚空，演算天机之能。
<br/>　　一钗为落凤钗，却是主攻击的法宝，也可操纵星河，单手一划，便是一道星河，颇为厉害。
<br/>　　一灵根自然是大名鼎鼎的先天蟠桃树。蟠桃树乃先天十大灵根之一，全称先天壬水蟠桃树，可化为三千六百株桃树，前一千两百株，花果微小，三千年一熟，人吃了成仙得道。中间一千两百株，六千年一熟，人吃了举霞飞升，长生不老。后一千两百株，紫纹细核，九千年一熟，人吃了与天地齐寿，日月同庚。
<br/>　　也就是说，吃了一颗九千年蟠桃，便可以直接成为金仙高手。因为金仙高手便是与天地同寿，日月同庚。只要不作死，或者不出意外，便是这样。
<br/>　　昊天与瑶池见此顿时大喜，接过宝物后，道谢一番，再次问道：“老师，天庭之上的职务众多，我等人手不够，却是该有何人执掌？”
<br/>　　鸿钧道祖淡淡道：“此事你与你众位师兄自行商量即可，为师不再过问。这洪荒大地日后便为人间界，天庭三十三重天为天界，后土所在地府为幽冥界，自此洪荒统称三界。尔等好自为之，不要让其破坏。”
<br/>　　话音刚落，鸿钧道祖便已经消失在紫霄宫之中，不见了踪影，如来时一般，不见丝毫波动。轻轻地来，轻轻地走。
<br/>　　见鸿钧道祖走后，元始站了起来，对着昊天与瑶池淡淡说道：“我玉虚宫门人不多，尔等自行商议即可。”
<br/>　　说罢，便直接绕过众人，向着紫霄宫之外走去。
<br/>　　“我门人更是不多，只有一人，人教以后有事，师弟可要照顾一番才是。”
<br/>　　太上圣人说完，便随着元始走了。
<br/>　　“我截教门下，弟子众多，多有大神通者，日后天庭有事，自是应当多照拂一二。”
<br/>　　通天将元始与太上都走了，他也如此说了一句，便消失了。
<br/>　　“哼！”
<br/>　　女娲更是冷哼一声，便不再理会他人，而是转身离开了。她又不立教，更是无甚门人。天庭又不是妖族天庭，与她何干？
<br/>　　……
