　　叶少阳在出口贴了符，召集小伙伴们守在出口两边，把所有人都引导在巨大的墓室里站好，这些生灵基本上也都很听话。
<br/>　　等所有生灵都出来了，虚空裂缝也渐关闭了。古墓还是古墓，仿佛什么都不曾发生过。
<br/>　　大伙欢呼雀跃，每个人都深切感受到一种劫后余生的喜悦，狂欢了好一会儿。
<br/>　　钟馗让大家安静下来，拿出酒葫芦，拔下塞子，让所有鬼魂都进去——不管之前是正修还是邪修，还是被法术公会的人杀了之后填进那虚幻空间，总之只要是死了的，都要跟他一起去阴司，这点没什么情面可讲。
<br/>　　更何况面对的是钟馗这个鬼中主宰，这些鬼魂只好跟叶少阳一行人道别，钻进了他的葫芦里，那里面也是一方开辟出的独立空间，鬼魂再多也挤得下。
<br/>　　等这些鬼魂走后，剩下的都是妖或者邪灵了。
<br/>　　钟馗检查了一边，又抓了几个躲在人群中的鬼魂，塞进葫芦里，确定没有了，又从九面猫那群人边上走过。
<br/>　　九面猫一干人顿时紧张起来。
<br/>　　“钟大士，我这里不会藏着鬼魂的，我太阴山，普通的鬼魂也看不上。”
<br/>　　钟馗朝她身边走去。九面猫立刻紧张后退，做好出手准备。
<br/>　　钟馗却压低声音说道：“你赶紧给我滚，哪儿来哪儿去，你要是赶对他透露一个字，我立刻就劈了你！”
<br/>　　九面猫一惊，立刻会意，偷偷看了一眼叶少阳，低声道：“好说。”
<br/>　　叶少阳正在清点自己的小伙伴，并没有察觉到他们的动作，钟馗走到古墓出口处，撕下叶少阳贴的灵符，道：“小天师，诸位，我走了！”
<br/>　　“啊，你这就走了？”叶少阳转头看时，钟馗已经往外面走了。九面猫也带着手下们跟着溜出去，等叶少阳想到还有问题要问她的时候，他们已经走了。
<br/>　　叶少阳明白他们是故意躲开自己，但人已经走了也没办法。
<br/>　　仔细一想，他们这么着急离开也是对的，他们毕竟是太阴山的人，跟自己还有阴司都势不两立，之前是为了对付星月奴，被迫结盟，如今回到人间了，双方自然恢复到正常关系——正邪不两立，他们要是继续留下，其实挺尴尬的。
<br/>　　要说真打吧，刚还是并肩作战的队友，实在有点下不去手。
<br/>　　这么一想，叶少阳也就释然了，至于九面猫没说出口的那个秘密……钟馗想来也知道，自己回头可以单独去找他打听，或者崔府君、徐文长也知道，也可以先找他们问问……
<br/>　　钟馗把鬼魂全带走了，还有太阴山那帮人，人数看过去少了很多，但剩下的还是将这个音乐厅大小的墓室站得满满的，一个个都在望着叶少阳。
<br/>　　“你们……不是妖就是邪灵，你们以前就是在人间修行的是吧，那你们现在可以走了，记住，一定要走正道啊，千万不要去邪修，不然将来被我遇到，绝不饶你们！”
<br/>　　有极少数邪物拜谢之后，离开了古墓。
<br/>　　但是绝大多数邪物都站着没动，眼巴巴地看着他。
<br/>　　“你们……怎么不走？”
<br/>　　有几个邪物互相看了看，走到队伍最前面，单膝跪地，拜倒在叶少阳面前。“叶天师，能与您老，还有诸位仙长相识，真是一场缘分，这次多亏诸位搭救，我等才能脱离苦海，之前我们互相串联了一下，我们决定追随叶天师，火里火里去，水里水里去！”
<br/>　　追随自己……
<br/>　　叶少阳一时间有点懵，为难地挠着头。“这个……不合适啊，我这人平时很懒散的，不是个当领导的料，再说我也没打算干什么大事啊，你们跟着我，实在无事可做……”
<br/>　　但这帮邪灵热血澎湃的，怎么劝说都没用，这时候林三生上前，问有没有人愿意跟自己前往空界，加入联军。
<br/>　　先是说了很多政策，例如自己刚拉起队伍，现在正是用人的时候，特别容易建功立业。
<br/>　　“还有就是：我们空界联军，也是捉鬼联盟的一部分，为联军服役，就等于是追随叶少阳，你们考虑一下，愿意走的举手。”
<br/>　　大伙互相望了一会，立刻就有不少人举手，大概有一大半。
<br/>　　小马这时候跳出来说道：“喂喂，还有我们风之谷啊，也是兄弟组织来着，愿意加入风之谷、跟我去干太阴山的到我这边来，不保证怎么样，唯一的好处就是没规矩，没规矩知道吗，想怎么干都行，到处打架，敲竹杠，打家劫舍，反正目的就是要在鬼域横着走……”
<br/>　　小马一段极具煽动的说辞，也打动了不少人，有一伙人站到了他那边去。
<br/>　　“那我就走了，我这来了许久，那边还有不少军务要等我去处理，少阳，你什么时候去见道风？”
<br/>　　“等我先安顿一下，立刻就过去。”
<br/>　　“好，那我先走一步，到时再说。”林三生又跟小青小白他们道别，然后拿出天地规尺，然后拿出一直判官笔，在岩壁上涂抹起来，判官笔上有红色颜料，被他画出了一个类似方框的图案，四周又画了几道符文，在中间额轻轻一点，方框好像窗户一样被推开了，中间一团漆黑，仿佛是一个风口，将附近的空气都吸了进去。
<br/>　　“厉害啊，跟神笔马良似的！”小马探头探脑地过去张望。
<br/>　　“走了！”林三生对大伙抱拳，招呼那些追随者钻进虚空裂缝之中，最后自己再钻进去。
<br/>　　叶少阳一干人，带着剩下的人一路爬出了古墓，从山体裂缝中钻出去。
<br/>　　阳光洒下来，眼睛有点不适应，但叶少阳一干人都仰起头，眯缝着眼睛望着太阳。
<br/>　　好久没看到太阳了，这种感觉……真他妈爽啊。
<br/>　　吴嘉伟感慨道：“挺奇怪的啊，有些东西，你每天都看到，并不觉得什么，一旦很多天见不到，才知道它有多珍贵……”
<br/>　　“你居然会说出这么有哲理的话，我好震惊！”小马竖着大拇指。
<br/>　　大伙都张开双臂，贪婪得晒着太阳，大口地吸着气，放眼望去，荒山野岭的，也没什么风景，但在叶少阳等人的眼中，却是那么地美好，空气中还带着泥土的腥味和草木的香味。不管怎么说，这才是真真正正的人间。
