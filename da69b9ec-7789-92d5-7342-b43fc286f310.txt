        傅夜沉离开后，童筱颖一个人在私宅里闷得发慌，特别是在这种情况下，越容易胡思乱想就越容易钻牛角尖。
<br/>
<br/>        突然间，她变得有些焦虑不安，很没安全感。
<br/>
<br/>        这段时间，每天夜里，她只要一沉睡，脑海里便会像是在做梦一样，总是闪过许许多多莫名其妙的画面。
<br/>
<br/>        起初，在澳洲小住的那段日子，她并未在意，但是到了这里后，更多的事情犹如潮水般在脑子里涌现。
<br/>
<br/>        有快乐的事，有悲伤的事，而所有的事，全都只与大哥一个人有关。
<br/>
<br/>        不行，她得离开这里！
<br/>
<br/>        倘若再待下去，她觉得自己有可能会疯掉。
<br/>
<br/>        童筱颖拿了车钥匙，在车库里开了辆车出来，随后离开了私宅。
<br/>
<br/>        她感觉她很多事情都是第一次去做，明明不会，但是一上手后又莫名其妙地会了，比如开车。
<br/>
<br/>        或许是因为她开了辆豪车的缘故，一路上有不少男人向她投来了异样的目光。
<br/>
<br/>        实在没地方可以去，童筱颖拿出手机输入了随机娱乐地点，结果导航带到去了滨江城里最豪华的娱乐场所——夜色魅惑娱乐城，有的人称它为“夜魅”，但更多的人叫它“夜惑”。
<br/>
<br/>        童筱颖停好车后，刚踏进这里的大门，就有种似曾相识的感觉。
<br/>
<br/>        她左右环顾，就像是在寻梦一样。
<br/>
<br/>        年轻的男服务生见到童筱颖的穿着打扮以及手提的香包都是高定的时尚限量款，便立即殷勤地迎上来，笑盈盈地说道：“这位女士是第一次来吗？我们这里有最好的服务供您选择，请问您是来做美容的，还是来做SPA的？我们这里还有喝茶陪聊服务！”
<br/>
<br/>        “我可以单纯的只喝茶吗？”童筱颖反问。
<br/>
<br/>        年轻的男服务员被她这句话说得差点呛到，但是他仍旧装出一副很有职业操守的样子，微笑着说道：“当然可以，我们这儿有午后下午茶。”
<br/>
<br/>        正好，她童筱颖还没吃午饭。
<br/>
<br/>        其实是傅夜沉给她做了午饭，她因为赌气才不吃的。
<br/>
<br/>        男服务生应了她童筱颖的要求，带她去了一个靠窗的雅座里，为她点了他们的招牌茶和招牌点心。
<br/>
<br/>        现在这个点还未到酒吧和清吧的营业时间，所以来“夜惑”的除了K歌的，剩下的就是一些商务茶会和按摩服务，以及一些孤独的女人来这儿找“男公关”陪聊，不过若是晚上，性质可能就不大一样了。
<br/>
<br/>        童筱颖本想一个人安安静静地喝喝茶，吃吃点心，但是有人不想让她这么安静。
<br/>
<br/>        一个身着西装革履，长相英俊，与傅夜沉有几分神似的男人，突然在她对面的软沙发上坐下。
<br/>
<br/>        童筱颖手里还拿着茶匙，巧克力蛋糕刚到嘴边，看到这个男人时，她身心就像定格了一般，惊怔地瞪大了眼睛。
<br/>
<br/>        她跟他不是第一次见面，上次见面，她记得是在临海城的拍卖会上。
<br/>
<br/>        只是，每次见面，感觉都一样，那就是没什么好感。
<br/>
<br/>        “筱颖，好巧，你也在这里。”傅昼景微笑地问候。
<br/>
<br/>        随即，服务员很识趣地替他端了一杯普洱茶上来。
<br/>
<br/>        童筱颖回过神来后，继续自顾自地吃蛋糕。
<br/>
<br/>        “你变了很多。”傅昼景将童筱颖上上下下打量了一番后，感慨道。
<br/>
<br/>        以前，他所认识的那个童筱颖，总是一副大学生的朴素打扮，哪怕跟他婚后在澳洲生活的那三年，她的穿着都很简朴，充满了青春活力。
<br/>
<br/>        但是现在，他眼前的这个她，从头上别的发夹，到脚上穿着的鞋子，以及身边的香包，全都是国际名牌。
<br/>
<br/>        就连她手边的那把车钥匙，那可是一辆价值上千万的豪车。
<br/>
<br/>        大哥真的很舍得在她身上花钱，把最好的东西都给了她。
<br/>
<br/>        “你跟着我的时候，确实吃了不少苦。现在跟着大哥，在物质生活上一定过得很滋润吧？你们女人就是这样，再单纯的女人，也会变得很现实很物质。”傅昼景冷嘲热讽地说。
<br/>
<br/>        童筱颖微微蹙了下柳眉，不满地反驳：“你自己没能力养女人，就不要抱怨女人太物质。更何况，这世上还有许多女人是靠自己，不是靠男人的。”
<br/>
<br/>        “那筱颍你是哪种呢？”傅昼景抬起手来，撑着脑袋，翘着二郎腿，笑着反问道。
<br/>
<br/>        童筱颖身心一怔，突然发现，她现在好像没有一技之长。
<br/>
<br/>        她除了依附着大哥过日子，每天除了吃喝玩乐以外，就没有其他事可干。
<br/>
<br/>        之前住在傅家大庄园里，还可以陪着婆婆参与一下上流圈子里的活动，但是当她一个人的时候，她好像有些一无是处的感觉。
