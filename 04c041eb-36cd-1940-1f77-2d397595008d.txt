　　尽管满心的纠结，但时间还是飞快的溜走了，当秦琼出现在他的面前时，李承乾知道，不管自己愿意不愿意，想没想明白，该来的都要来了。
<br/>　　老秦自从上次被李承乾所求之后，曾经承诺今后永远为他的护卫，现在今天老秦的身份不是翼国公，而是他的护卫头子。
<br/>　　两人一前一后，身后跟着无数宫女、内侍、禁军，以全副太子仪仗直奔大明宫的宣政殿。
<br/>　　……
<br/>　　今日的大明宫与往日有了很大不同，禁军多了一倍有余，无数朝臣早早便来到了大殿之前的广场之上整整齐齐的站成数列。
<br/>　　李承乾来到宣政殿的时候，东面的天空已经泛起霞光，放眼望去竟向天空在流血一般。
<br/>　　这该死的天气！李承乾暗骂了一句，心中的不安更加强烈，总觉得似乎有什么事情要发生。
<br/>　　……
<br/>　　“陛下临朝，百官觐见！”执礼内侍尖细的嗓音穿透殿前广场，传入每一个人的耳中。
<br/>　　李承乾已经听过无数次这种声音，但每次听到之这声音都会觉得头皮发麻，这次也不例外。
<br/>　　不过就算再怎么样，该上朝还是要上的，老头子虽然说今天传位，但只要老头子一天没有退位，小李就依旧还是太子，就算是觉得那太监再差劲儿也只能忍着。
<br/>　　脑子里转着一些有的没的，由李承乾打头的队伍已经慢慢进入老头子临朝的大殿，看到了高高在上的李二陛下。
<br/>　　“臣等参见陛下！”百官上朝，例行拜见。
<br/>　　不过这次老李却并没有说出平身这句话，只是冷冷的看着下面躬身施礼的重人，半晌方才开口：“朕，戎马半生，打下偌大江山，励精图治一十七载，将国家治理的井井有条，不敢说功盖千古，亦可称冠绝当世。”
<br/>　　“然今日……”
<br/>　　‘咔咔咔’接连半个时辰的忆往昔峥嵘岁月稠，老头子从当年自己的老头子太原起兵一直说到今日的登基大典。
<br/>　　站满了人的大殿之上鸦鹊无声，所有人都低着头，不知道是在缅怀过去还是在畅想未来。
<br/>　　但李承乾却越发觉得事情有些不大对头。
<br/>　　因为这根本就不是登基大典或者禅位应有的程序，而且大殿上的禁军也不是礼兵，而是战兵，这同样说明了今天的事情不会简单。
<br/>　　老头子后悔了？应该不至于！
<br/>　　自己这几天根本就没有多余的动作，如果老头子后悔或者有什么想法，完全可以在这十来天进行。
<br/>　　可是如果说老头子没有后悔，就这样认命了，似乎也不对，因为眼下的老头子不管是气势还是状态，似乎都与前几天大不相同。
<br/>　　站在老头子身边不远的李承乾脑子转的飞快，努力的想着这段时间里发生一切事情，同时目光也在下面的人群中搜索，将上殿的人与印相中的人一一比对。
<br/>　　你还别说，这一比对还真让他发现了一些不对劲的地方。
<br/>　　特么老尉迟那里去了？虽然听说这老货现在准备改邪归正，回家参禅打坐去了，但这么大的事情不应该不来吧？
<br/>　　而如果老尉迟来了，那么为什么他不在大殿上的众人当中？
<br/>　　不，不对，这事儿不对。
<br/>　　越想越不对头的李承乾瞬间便有了一种如坐针毡的感觉，冷汗也开始不自觉的从头上流了下来，时间不大，便已经将后背的衣服完全浸湿。
<br/>　　而就在此时，老头子突然停了下来，转头看着李承乾，半晌方才叫道：“太子！”
<br/>　　没有反应，李承乾此时的腿已经有些麻了，大脑高速转动，想着应对的方法，竟没有注意到老头子在叫他。
<br/>　　终于，在老李叫了他三声之后，一边的内侍看不下去了，捅了他一下：“太子殿下，陛下叫您呢！”
<br/>　　“啊？！哦，父皇，儿臣在！”顶着一头的冷汗，李承乾快步走出自己的位置，来到大殿的中间，心中已经是一片冷凉。
<br/>　　“太子，为何此张！”李二高高在上的坐着，俯视着下面的一切，声音中不带任何感情。
<br/>　　“父皇，儿臣……”李承乾想了半天，也想不出来应该怎么说。
<br/>　　眼下分明已经是大祸临头之兆，一个不好就特么或流放或砍头了，就连脑子转动的速度也比以往慢了许多，那条三寸不烂之舌似乎也有些僵硬。
<br/>　　“平日里你巧舌如簧，为何大殿之上却吞吞吐吐？”老头子见李承乾不说话，便又追问道。
<br/>　　良久的沉默之后，李承乾叹了口气：“父皇，儿臣不知应该说些什么！”
<br/>　　“是么？可是觉得今日情况不对？是不是觉得心中忐忑？”
<br/>　　“是！”李承乾躬身答道。
<br/>　　既然老头子都已经这样说了，这足以说明他刚刚的猜测是对的。
<br/>　　秦琼秦二爷就站在他的身后，老尉迟不知所踪，大殿内外尽是玄甲军的人，这足以说明老头子是早有准备，今日就想拿他开刀。
<br/>　　可笑自己前几天还特么以为老头子真的是头风犯了，感情是老头子在暗中布置一切，只等今日大朝自己来个水缸里面抓王八。
<br/>　　只是，李承乾也觉得冤枉，这特么一切根本就不是自己想好的好么，逼老头子退位是魏征那小老头儿临死前搞出来鬼，和自己有毛的关系啊，想追责就找那小老头儿去好了。
<br/>　　但这些他都只能在心里想想，偷眼看着高高在上的老头子，现在的李承乾除了自认倒霉以及自求多福之外，再也没有其他的办法。
<br/>　　因为他此前根本就没有任何的准备，他也没想到老头子会突然来上这么一手。
<br/>　　而此时大殿上的众人也觉查到了情况不大对头，但是这些人都没有说话，包括老长孙在内。
<br/>　　毕竟现在有李承乾这个‘大个儿’在前面顶着，就算是天塌了也是由他来扛，如果此时乱说话，把李二的注意力吸引过来，那特么可就太糟糕了。
<br/>　　便是这样，偌大的宣政殿中陷入了长久的沉默，李承乾的命运如何似乎全在李二的一念之间，只是不知道这位皇帝陛下到底会做出何种决定。
