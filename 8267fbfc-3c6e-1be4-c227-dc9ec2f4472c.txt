    陈兴滚出数米外，爬起来，猫腰潜行，躲到另一处的掩体。<br/>    <br/>    “灭火，快啊！”<br/>    <br/>    在胖子头领的催促下，几名工人提着灭火器冲过去，对着燃烧的卡车喷出白泡沫。<br/>    <br/>    土山上没有动静，枪手在等待时机。陈兴不敢直接开瞄准镜，营地里的火光还没扑灭，很容易形成反射，暴露自己的位置。等了好一会儿，直到火被扑灭，枪手还是没有开枪。<br/>    <br/>    陈兴不由得想到，如果镇长在这里，随便挥挥手，万炮齐发，直接就能把对面的土山移平。在绝对强大的力量面前，任何伎俩都会显得苍白无力。不过他不是镇长，还得想办法将枪手引出来。<br/>    <br/>    他环视了一眼，不远处的卡西正躲在物资箱子的后面。他低下头，从地上捡了块小石头，在手里掂了掂，然后扔过去。石头落在卡西的头上，卡西吃痛，大叫起来。<br/>    <br/>    “谁，谁！”卡西左右张望，然后就看见不远处的陈兴，顿时翻了下白眼。<br/>    <br/>    陈兴并拢手指，朝土山方向一指，意思是让对方“吸引火力”。<br/>    <br/>    佣兵是个很神奇的职业，上一刻还在针锋相对的两人，下一刻可能为了共同的敌人并肩作战。<br/>    <br/>    卡西犹豫了一下，朝陈兴竖起中指，但还是端起突击步枪，翻滚而出，朝土山方向瞄准。<br/>    <br/>    山上火光一闪，卡西身上蓝影晃动，星沙飞溅，似乎承受着巨大的冲击力，横飞出去。陈兴立即打开镜瞄准，瞄准火光的位置，凝神聚气，一枪击出。<br/>    <br/>    “嘭！”<br/>    <br/>    蓝焰喷射，山上的人影头部炸开，滚下了土坡。<br/>    <br/>    许久过后，山上再无动静，可以确定，危险解除了。陈兴收起枪，去查看卡西的伤势。<br/>    <br/>    “你欠我一颗子弹。”卡西肩膀上焦糊一片，一看到陈兴，就气冲冲地说道。他的伤势不算严重，除了身强体壮外，灵能护甲也吸收了大部分的伤害。<br/>    <br/>    灵能护甲非常实用，与“蓄能射击”“精神冲击”并称三大实用技能。但贵精不贵多，高等级的技能和低等级的相差甚远，大部分人都只能精通一项。<br/>    <br/>    中远程战斗单位，通常会选择蓄能射击，法术类战斗单位，通常会选择精神冲击，直接击溃对手的心灵，而近战类的，自然是灵能护甲，提升自身的防御能力。<br/>    <br/>    不过，这也仅仅是针对普通人，门阀世家的选择会更多。<br/>    <br/>    “现在还吗？”陈兴看了眼腰上的银星，问道。<br/>    <br/>    “噢，我的天啊，你是我见过的最无耻的黄猴子。”卡西捧起双手，一副要抓狂的样子。<br/>    <br/>    肤色总伴随着歧视，和地球上一样，黑人的蔑称是“黑鬼”，白人的蔑称是“白猪”，蓝血人的蔑称是“大鹅”，沙民的蔑称是“沙狗”。<br/>    <br/>    陈兴咧嘴一笑，正想骂回去，身后的黑暗中忽然浮现一道身影。他立即双指并拢，反身一指，一道电弧激射而出。<br/>    <br/>    “啊！”<br/>    <br/>    来人一声痛呼，身上爬满细小的电弧。但这种程度的电击不算什么，她反握着弯刀，横在胸前，说道，“放松点，伙计。”<br/>    <br/>    陈兴一言不发，又一道电弧甩过去。电弧抽打在女人身上，女人浑身一颤，额头上冒出了细密的汗珠。但只是盯着陈兴，没有继续逼近。<br/>    <br/>    “我不喜欢别人的试探。”陈兴面无表情地说道。<br/>    <br/>    “很抱歉，我无意冒犯。”女人扔开弯刀，张开双臂，袒露胸腹，表示放弃抵抗。<br/>    <br/>    陈兴却没打算放过她，伸手一指，电弧蹦射而出，击在女人胸上。女人发出闷哼声，退后了几步，但依然保持着张开双臂的动作。<br/>    <br/>    “滋啦滋啦！”<br/>    <br/>    又连续两道电弧击在女人身上。女人脚一软，跪倒在地上，胸腔不断起伏，喘着粗气，似乎正承受着巨大的痛苦。<br/>    <br/>    “我希望不会有下一次。”陈兴淡淡地说道，转身望向远处的土山。<br/>    <br/>    卡西本想抱怨两句，但也识趣地住了嘴。<br/>    <br/>    “你，你，跟我走。”陈兴指了指阿乔木和另一个粗壮的男佣兵。他的声音不大，却充满了不容置疑的威严。<br/>    <br/>    被指的两人都没有说什么，一个捡起弯刀，另一个背上突击步枪，默默地跟在陈兴后面。荒野之上，强者为尊，规则就是这么简单，没什么好争论的。<br/>    <br/>    “带上我，分钱的事，这么能忘了我。”坐上吉普车的时候，卡西追了过来，“白猪，让开点！”他不由分说地往车上窜，挤开后座的大个子。<br/>    <br/>    白人佣兵龇着牙，朝卡西怒目而视，似乎对这个称呼十分反感。<br/>    <br/>    “放松点，伙计，我们这是去郊游……”卡西自来熟地拍了拍对方的肩膀，然后看了眼腰上的短刀，“或者是割开你的脖子。”<br/>    <br/>    两人对峙了一会儿，白人佣兵嫌恶地甩开他的手，看向前方，没有再说话。<br/>    <br/>    没过多久，四人爬上土山，找到了两具尸体。<br/>    <br/>    枪械和随身物品都被搜走了，只剩下衣服和手腕上的黑表。没有人对黑表感兴趣，一方面，黑表会扫描使用者的DNA/片段，以确定使用权限，哪怕是直系亲属也没法使用；另一方面，携带他人的黑表容易引起嫌疑，麻烦多多，甚至有可能被起诉杀人罪，重则流放到黑死大陆，轻则下降两个大信用等级，得不偿失。如果不是熟人的黑表，作为遗物保留，很少人愿意摊这趟浑水。<br/>    <br/>    “是佣兵团的人。”<br/>    <br/>    阿乔木蹲在尸体旁，指着左胸处一个颜色较新的位置说道。不难看出，那个位置的标志被扯走了，周围还有明显的线头和针眼。<br/>    <br/>    在没有卫星信号覆盖的地区，佣兵团摇身一变，就是土匪强盗，烧杀掳掠，无恶不作。不过大多数情况下，都是那些混得不太好的佣兵团。能正常赚钱，谁愿意干这种勾当？万一一个不小心，一头撞在铁板上，肯定是尸骨无全。<br/>    <br/>    “能看出是哪个佣兵团的吗？”陈兴问道。<br/>    <br/>    阿乔木摇了摇头。<br/>    <br/>    “那就算了。”陈兴说完，将两具尸体踹下土山。用不了多久，数公里外闻到血腥味的食尸犬就会围过来，将尸体吃得干干净净。环保自然，连埋都不用。<br/>    <br/>    <br/>
