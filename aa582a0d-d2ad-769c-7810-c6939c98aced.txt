夏夕绾抬眸就看到了陆寒霆，他还没有回去。
<br/>
<br/>        顾夜瑾和霍西泽也来了，两个人就站在门边。
<br/>
<br/>        霍西泽笑着打了一声招呼，“二嫂，你好。”
<br/>
<br/>        被“二嫂”的夏夕绾只能勉强挤出一丝笑意。
<br/>
<br/>        管家福伯来了，带来一个保温杯，“少奶奶，刚才少爷交代下来给你熬了一点小米粥，还有两个清爽的小菜，你快趁热吃吧，照顾病人的前提是养好自己的身体。”
<br/>
<br/>        夏夕绾晚上真的什么都没有吃，没想到他这么周到体贴，已经让人熬了小米粥。
<br/>
<br/>        夏夕绾看向陆寒霆，男人身高腿长的伫立在窗边，已经脱了身上的黑色大衣。
<br/>
<br/>        这些年除了叶翎，他是第一个让她觉得这么温暖的人，但是她不想再放任自己的心。
<br/>
<br/>        他真的很危险，一步步的诱惑她深陷。
<br/>
<br/>        她怕自己会万劫不复。
<br/>
<br/>        夏夕绾来到陆寒霆的身后，“陆先生，今晚我留在这里陪林婶，你先回去休息吧。”
<br/>
<br/>        陆寒霆侧身，看着夏夕绾，“为什么不接我的电话？”
<br/>
<br/>        夏夕绾一滞，迅速道，“我没有听见。”
<br/>
<br/>        “那微信呢，是不是要说没看见？”
<br/>
<br/>        夏夕绾还没有说话，门边的霍西泽已经“我靠”了起来，“二哥，你竟然有了微信，以前我们让你玩微信的时候你当没听见，给我看看你的微信上是不是只有二嫂一个人，二哥你真的太重色轻友了！”
<br/>
<br/>        陆寒霆一个锋利的目光向霍西泽杀了过去，“滚！”
<br/>
<br/>        福伯意识到气氛的不对劲，迅速走了出去将门给关上了。
<br/>
<br/>        ……
<br/>
<br/>        病房里，夏夕绾还沉浸在霍西泽的话里，他的微信里真的只有她一个人么？
<br/>
<br/>        这时陆寒霆的手指探了过来，直接将她兜里的手机给拿走了。
<br/>
<br/>        “你干什么拿我手机，快还给我。”夏夕绾踮起脚尖就去抢自己的手机。
<br/>
<br/>        陆寒霆打开她的微信看了一眼，果然他发给她的微信已经成了已读，她看到了，但就是不回他。
<br/>
<br/>        “给我个解释。”陆寒霆问。
<br/>
<br/>        夏夕绾想要抢回自己的手机，“你要什么解释，没有解释，就是你看到的那样，我看了，就是不回你。”
<br/>
<br/>        陆寒霆英俊的眉心里覆上了一层阴霾，他连夜赶回来就是听她说这个的？
<br/>
<br/>        现在她纤柔的身体几乎贴着他，她身上那股清甜的体香席卷他的鼻翼，挥之不去。
<br/>
<br/>        这个让他怀念的味道迅速让他滚动着喉头，恨不得将她杠起然后用力的丢进床里。
<br/>
<br/>        “夏夕绾，你再往我身上蹭？”他从喉头里逼出沙哑而凌冽的嗓音。
<br/>
<br/>        夏夕绾瞳仁一缩，当即感觉到了他慢慢灼人的体温，她迅速退了下来，想离他远一点。
<br/>
<br/>        但是陆寒霆按住了她莹润的肩头将她推抵到了墙壁上，不让她逃，“一张照片将我撩成那样，我赶回来你就对我冷暴力恩？明明是你将自己置身险地做错了事，还敢先甩脸色给我看，我看你真是欠收拾了！”
