        童筱颖睁着氤氲的水眸睐着傅夜沉，心里很憋屈：“二婚老公对我太色了怎么办？”
<br/>
<br/>        倏地，她拉起他的手，作势替他号脉。
<br/>
<br/>        傅夜沉倒是饶有兴趣地打量着她这一举动。
<br/>
<br/>        “大哥，你体内火气太旺，得清心寡欲，以免急火攻心，流鼻血什么的，那就挺糟糕的了！”童筱颖有模有样地说。
<br/>
<br/>        但他似乎完全不把她的话当回事，当然，她这话也是胡编乱邹的。
<br/>
<br/>        读高中以前，她确实会替人号脉，是外婆教的，但是这么多年过去了，她没继承外婆的衣钵，差不多把外婆教给她的医理，又悉数还了回去。
<br/>
<br/>        下一秒，童筱颖自己都忍不住笑了起来。
<br/>
<br/>        傅夜沉嘴角噙笑，微微偏头，趁势要吻下来时，童筱颖双手抵着他的胸膛，轻轻地拍了拍，然后竖起食指，十分尴尬地指了指悬在电梯天棚一角的那台红外线高清摄像机。
<br/>
<br/>        这大晚上的，监控室里一定有保安盯着的吧！
<br/>
<br/>        傅夜沉的目光随之顺着童筱颖所指的摄像头看去，他就盯着摄像头，黑眸微微眯了眯，眼神犀利。
<br/>
<br/>        另一边，监控室里，坐在监控台前的男保安看到傅夜沉的眼神时，心里“咯吱”了一下，冷不丁地打了个寒颤。
<br/>
<br/>        监控室里也是有监控的，所以……
<br/>
<br/>        下一秒，男保安立即识趣地撕下台面上的一张黄色便利贴纸，将总裁专属电梯的监控画面给挡住。
<br/>
<br/>        非礼勿视！
<br/>
<br/>        回眸之际，傅夜沉修长的指腹捏住童筱颖的下巴轻轻地抬了抬，他长腿逼近，将她按入怀中，低下头，薄唇像磁石般地吸住了她的两片唇瓣。
<br/>
<br/>        他温柔地在她的唇上轻轻辗转，这份温柔让她忍不住沉醉其中。
<br/>
<br/>        童筱颖缓缓地合上眼帘，长长的羽睫在她白皙的肌肤上投射出一圈淡淡的阴影。
<br/>
<br/>        她好喜欢他这个温柔的吻，像棉花糖那样的软、像果冻那样的嫩、还像……
<br/>
<br/>        电梯“叮”地一声，到了总裁办公室的楼层，傅夜沉却突然放开了她。
<br/>
<br/>        童筱颖倏然睁开了双眼，如梦初醒般仰望着傅夜沉。
<br/>
<br/>        他拉起她的手，放在湿润的唇边，落下缠绵的最后一吻，笑而不语，然后十分淡定地将她打横抱起踱步走出电梯。
<br/>
<br/>        刚刚的那种感觉，还像初恋那样的甜。
<br/>
<br/>        到了总裁办二楼的休息室里，他给她放好了洗澡水，抱她进入了浴室，待她洗完澡一瘸一拐地从浴室里出来后，他又贴心地将她抱到床上，拿来了药油，给她那扭伤的脚腕按摩涂药，之后要拿来了吹风机，给她耐心地吹干长发。
<br/>
<br/>        这里没有她要换洗的衣服，所以她身上穿着的是他的白衬衫。
<br/>
<br/>        衬衫的衣摆刚好遮住她的臀，两条雪白的大腿并在一起，让人很容易遐想出戏。
<br/>
<br/>        原本，她以为等他洗完澡后，接下来应该要做些什么，但是他只是伸出手臂给她枕在脑后，轻轻地搂着她，一起躺在了床上。
<br/>
<br/>        “大哥，我想听你唱歌。”她依偎在他怀中，像是在撒娇。
<br/>
<br/>        他十分淡定地回答：“我不会唱。”顿了顿，他又补充道，“你想听我唱哪首，我去学。”
<br/>
<br/>        “我想听那首《月亮代表我的心》。”她不假思索地说。
<br/>
<br/>        他轻轻地吻了吻她的头顶，应道：“好，等我学会了，我再唱给你听。”
<br/>
<br/>        童筱颖甜甜地笑着，往傅夜沉那温暖的怀中钻了钻，不知不觉中，她缓缓地阖上了眼帘。
<br/>
<br/>        这天晚上，她做了一个莫名其妙的梦。
<br/>
<br/>        梦中，一片白色的空间里，一个五六岁的小男孩正在轻轻地摇动着一个小摇篮，摇篮里睡着一个小手握拳，被裹在襁褓中的粉嫩嫩的小婴儿。
<br/>
<br/>        小男孩摇着摇着，探起身去，突然亲了亲小婴儿的脸，然后对这小家伙喃喃自语说了些什么。
<br/>
<br/>        翌日。
<br/>
<br/>        童筱颖醒来的时候，身旁的床位一片冰凉，早已不见了傅夜沉的影子。
<br/>
<br/>        她坐起身来，发现自己脚腕已经不痛了，正欣喜的时候，楼下隐隐传来说话声，引起了她的好奇心。
<br/>
<br/>        听声音，像是大哥和昼景的？
<br/>
<br/>        童筱颖连忙从床上下来，蹑手蹑脚地走到旋转楼梯口。
<br/>
<br/>        “爷爷若是知道你和筱颍的事情，是不会同意你和筱颍在一起的。”傅昼景眉头紧锁，坐在一旁的沙发上，抬眼看向正一本正经坐在办公桌后面批文件的大哥傅夜沉。
<br/>
<br/>        “阿景，如果你来是跟我说这件事情，那么你可以走了。”傅夜沉连头也没抬一下，冷峻的脸庞上不显丝毫神色，并毫不客气地对傅昼景下逐客令。
