　　但是李浩然空有一身神通，却也不知道怎么样才能回去，最后想来想去，得出的结论跟四宝和吴嘉伟一样：这一切都是因为叶少阳而起，徐福不可能把叶少阳丢在这里完全不管，所以，只有找到叶少阳，才能找到徐福，找到徐福，才有机会离开这个世界！
<br/>　　徐福！！
<br/>　　李浩然咬着牙，在心中大骂徐福，他已决定，如果将来见到徐福，一定要杀了他，以报今日之仇！
<br/>　　不过，这一切也只有先找到叶少阳再说了。
<br/>　　对这一切，叶少阳完全不知情，不知道有这么多人在找他，甚至不知道自己的朋友和仇人都到了这个世界，他只是在按照自己的计划继续行动。
<br/>　　叶少阳被张道长邀请到清风观里喝茶聊天。面对张道长的打听，叶少阳不敢说自己是从未来世界穿越而来，毕竟这个时代还没有一百年之后的那种穿越文化，多数人还是无法理解穿越这种事，叶少阳也不想费太多的口舌去解释，干脆就告诉张道长，自己本来是一个天师，因为某些原因，现在法力大减，所以看上去只有真人的实力。
<br/>　　对这个解释，张道长深信不疑。叶少阳在之前作法的时候展示出来的实力，不管是道心的坚定，还是作法的娴熟程度，都远远不是一个真人该有的。
<br/>　　得知真相之后，张道长对叶少阳的态度也是更加的尊敬，两人聊了一会，张道长也不顾自己的年纪快能当他爹了，主动跟他称兄道弟起来。
<br/>　　“贤弟，你今番来我江西，是有什么打算？”
<br/>　　叶少阳想了一下，道：“实不相瞒，我是要去龙虎山，见一个前辈，我有要事找他。”
<br/>　　张道长一听就更郁闷了：“龙虎山在鹰潭，在江西的东北方，贤弟为什么走到我们中部这边来？”
<br/>　　叶少阳苦笑道：“说起这个真有点尴尬……”
<br/>　　当下把自己先遇战乱，再遭土匪的经过的说了一遍，张道长听到他是因为没钱了，想要去公堂义庄赶尸赚钱才走到这里来时，实在憋不住大笑起来，竖着大拇指说道：“叶老弟，缺钱了卖田卖房卖力气的事我都见过，想要靠赶尸赚路费的事，我还真是头一回听说。”
<br/>　　叶少阳耸肩道：“那我有什么办法，我只会捉鬼降妖，而今恰逢乱世，也没人看风水之类的，只好去赶尸了。”
<br/>　　张道长说道：“赶尸并不怎么赚钱，还很辛苦。”
<br/>　　叶少阳道：“我知道，我要是但凡有别的法子，我也不会干这个。”
<br/>　　张道长笑起来，亲自端起茶壶，给叶少阳续水。“叶老弟，如果不嫌弃的话，老兄我借给你这笔盘缠。”
<br/>　　“你借给我？”
<br/>　　“你我同门，就算是江湖救急吧，叶老弟硬要收借，那就算是借吧。”
<br/>　　叶少阳犹豫起来，说起来，自己需要的盘缠并不多……要是张道长能借给自己，倒也是给自己解决了一个大麻烦，不用再往义庄去，但叶少阳自己认为不是这个世界的人，实在不好欠别人的东西，免得将来不知道怎么还。
<br/>　　张道长见他犹豫，立刻也就猜到了他的想法，当下热情洋溢地替叶少阳答应下来。
<br/>　　“就这么办，叶老弟，你不必跟我客气，虽然我这清风观不是什么钱庄，但一点盘缠钱还是有的，你我也算有缘，我都叫你一声叶老弟了，在我这做客，临走了我连盘缠都不送，岂不是让人笑话。”
<br/>　　叶少阳听他这么说，也只好依从了，当下说道：“说是这么说，但是在我，也不好白要你的钱，你有什么需要帮忙的事情，我可以出点力气。”
<br/>　　张道长立刻表示没有。
<br/>　　喝完了茶，张道长要去收徒，叶少阳也想去找翠云，跟她说一声张道长赞助钱财的事，于是两人一起前往思故村。叶少阳让张道长先去刘四家里收徒，自己去了刘老汉家里。
<br/>　　刘老汉早他一步回家，早把叶少阳的事迹跟老伴和翠云说了，翠玉一直悬着的心总算放下了。刘老汉和老伴则兴奋的不行，热情地接待叶少阳，刘老太怕叶少阳累了，还煮了几个草鸡蛋，非要让他吃，给叶少阳弄得很不好意思。
<br/>　　正说着话，刘四媳妇过来串门，请他们去家里吃饭，孩子没事了，他们夫妻俩心头长期悬着的一块石头也算是落下去了，对叶少阳这个恩人，自然想要殷勤招待一番。
<br/>　　刘老太也很兴奋，最后两个女人一合计，就在刘老太家烧火做饭，中午大家一起聚餐。
<br/>　　村里面虽然菜肴不多，但是有不少野菜蘑菇和一些从山里打的野味，两个妇女喜气洋洋地收拾起来，翠云看着手痒，也去帮忙。
<br/>　　中午，几个女人折腾出了一桌农家土菜，开了一坛自家酿的老酒，招待叶少阳和张道长。张道长一高兴，把收徒仪式也给办了，赐给刘悟得一件随身携带的杨公盘作为礼物。
<br/>　　叶少阳借着酒劲，也送了一枚自己做的鸡血石挂坠给他。刘四夫妻很是高兴，提议让孩子认叶少阳做叔叔。叶少阳差点没一口酒喷出去。
<br/>　　刘悟得跟自己师父是平辈的，还是好友，自己叫他都是叫师叔的，让这样一个长辈叫自己叔叔，叶少阳担心自己折寿，只好找借口给推了。
<br/>　　吃完饭，叶少阳带着翠云，跟两家人告别，他要跟着张道长去他的道观，然后从那里出发。
<br/>　　两对夫妻依依不舍地送他们到了村口，正要分别的时候，刘老汉把叶少阳拉到一边，把一个小东西塞到了他手中。
<br/>　　叶少阳低头一看，是一枚大洋，登时笑道：“老叔，婶子送的那些山货我就收下了，但是钱还是算了，你家也不富裕，这不合适。”
<br/>　　刘老汉把大洋塞在他手里，严肃地说道：“这不是一般的钱，这是我那个侄儿刘震匀送给我的信物。”
<br/>　　“刘震匀……哦哦，想起来了，那位刘司令！”之前当过革/命军，后来落草为寇、成为大土匪头子的那位。在叶少阳心中，这个人还真的有点传奇性，因此印象还挺深的。
<br/>　　（在火车上写出两章，人在长沙，3号回家）
