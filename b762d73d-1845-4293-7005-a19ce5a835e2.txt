　　蛇博士陈默早期是白镇的捕蛇人，看到很多无辜的人死在毒蛇的手里，就做起了蛇伤医生。
<br/>　　再到现在的蛇博士，他一生的意愿就是悬壶济世，蛇可伤人，更能救人。
<br/>　　它就好像是一把无坚不摧的宝剑，用的好，可以行侠仗义，救黎民于水火，但如果在心邪之人的手里，那么便是一场浩劫。
<br/>　　陈默为了培养出一条绝世蛇种，不知道耗费了多少的心力，一是为了救人，二更是为了当年履行当年跟故人的一个承诺。
<br/>　　现在绝世蛇种已成，却成了杀人的机器。
<br/>　　无论是他的徒弟，还是这条蛇，都让他痛心疾首。
<br/>　　此时陈默发出一种受伤的狮子般的怒吼声：“江行泽，你的命是我给的，为师能给你，也能将你的命给重新要回来，你犯下如此丧心病狂的罪孽，我若不收你，老天也会收你了，所以你不要怪为师”
<br/>　　一句话说话，陈默杀气顿现，手中出现了一根银针，眼神散发着冰冷的光芒。
<br/>　　“师父你是不是早就想杀我，就算我不这么干，你恐怕也留我不得”江行泽癫狂的笑了起来，同时也非常的忌惮他手中出现的银针，可以杀人于无形。
<br/>　　倏然，江行泽躲到了庞大蛇躯的吸血蛇身后。
<br/>　　陈默仿佛听到了什么天大的笑话：“为师想杀你，当日为何救你”
<br/>　　江行泽立刻反驳：“就是因为你救了我，我每日喝吸血蛇的血，我跟它越发的亲近，你把怕我夺走了你的绝世蛇种”
<br/>　　“哈哈...”
<br/>　　陈默闻言发笑，体内迸发一种骇人的恐怖气息，发丝迎风而动，眼神浑然一凝：“我陈默手中的蛇种，当今世上谁能夺的走，你虽是我的徒弟，但你还差的远”
<br/>　　他可以很负责的说，江行泽连他一半的本事都没学会。
<br/>　　都说教会徒弟，饿死师父，可有些本事，靠的是天赋，不是教你，你就能学的会。
<br/>　　陈默可是人类觉醒者，他天生就对毒蛇有一种超强统治力。
<br/>　　这是普通人永远望尘莫及的。
<br/>　　“师父，长江后浪推前浪，你看看你现在还使唤的动吸血蛇吗”
<br/>　　江行泽戏谑的一笑，曾经他在陈默的面前，大气都不敢喘，可现在他无所顾忌。
<br/>　　嗯？
<br/>　　陈默紧皱眉头，看着庞大蛇躯的吸血蛇，浑身散发阴冷的气息，怒声：“畜生，还不快滚过来”
<br/>　　“吼...”
<br/>　　吸血蛇对着陈默仰天咆哮一声，扶摇直上，凶煞可怖，根本没有了往日的乖巧。
<br/>　　到是恨不得一口活吞了他。
<br/>　　“师父，你就别费劲了，现在吸血蛇已经认我为主，成为我的宠物，它又怎么会听你的”
<br/>　　怪不得他有恃无恐，原来已经让吸血蛇听命于他。
<br/>　　忽然陈默想起了什么：“你是不是将我给你的蛇魂喂给吸血蛇吃了？”
<br/>　　江行泽嘎嘎一笑：“师父就是师父，什么都逃不过你的法眼，没错，我是用你给的蛇魂养给吸血蛇吃，在加上我跟它现在血脉相连，流淌的一种血液，它认我为主也是理所应当”
<br/>　　“师父，今天就是徒儿出师的一天，你应该为我高兴”江行泽得意的说着，以后天高任鸟飞。
<br/>　　所谓师父领进门，修行靠个人，陈默摇了摇头，俨然的说了一句：“你控制不了这条蛇的，泽儿，回头是岸吧，纵然你能控制它一时，等随着它的实力持续变强，你拿什么控制它”
<br/>　　生物有灵，亦冷血，尤其是这种凶煞的蛇种，就跟定时炸弹一样，随时会爆。
<br/>　　江行泽放肆的叱喝：“师父，你老了你知道吗，当初你控制不了那条银白蛇，现在你连吸血蛇都控制不住了，你真的老了，以后是我们年轻人的天下，吸血蛇在我的手里，才能变成蛇王的存在，我身上流着跟它同一种血，它怎么可能背叛我”
<br/>　　陈默悲凉的望着意气风发的徒弟，有种兔死狗烹的伤感，这徒弟他是拉不回了。
<br/>　　江行泽却越发嚣张：“师父，念在我们师徒一场，我现在带着吸血蛇离开山野集团，你别拦我，否则别怪徒儿不讲情面”
<br/>　　“你真的不可救药，那么为师就送你上路吧”
<br/>　　捏在陈默手中的银针骤然出手，化为一抹光影，朝着江行泽的额头射去。
<br/>　　在这里解决了他，免得这个孽徒出去害人害已。
<br/>　　“铿锵...”
<br/>　　银针出手，最后却射在了吸血蛇火红鳞片上，直接脱落，连鳞片都没刺进去。
<br/>　　此时的吸血蛇将江行泽护在身后，一双铜铃般的蛇眸死死的盯着陈默看，满是嗜血。
<br/>　　“畜生，别忘记了是谁将你培养成绝世蛇种”
<br/>　　吸血蛇通灵摄厉，看陈默就像看什么杀父仇人，就是这个该死的人类将它奴役，关在这座不见天日的牢笼中。
<br/>　　一条粗大的蛇尾朝着陈默呼啸的拍去，力道之大，不下数千斤之力。
<br/>　　轰..．
<br/>　　牢笼无法无天的震动起来，陈默何等人物，岂能被蛇尾拍到，瞬间闪躲开了。
<br/>　　霎时间，一阵口哨声连绵不绝的响起，蕴含特别奥秘的咒语之力，任何蛇种听到，都要动弹不动，痛苦不堪。
<br/>　　昂..．
<br/>　　吸血蛇怔怔了下，铜铃般的蛇眸红光一片，发出沙哑的声音：“人类，我要吃了你”
<br/>　　不奏效，吸血蛇成长的非常之快，早就今非昔比，既然不受蛇类咒语的影响。
<br/>　　“师父都说你老了，你的手段不管用了”
<br/>　　江行泽刚听到口哨咒语声，还有点担心吸血蛇受制，可吸血蛇丝毫没有受影响，顿时让他喜出望外。
<br/>　　“孽徒，等为师收拾了这条畜生，再来取你的命”
<br/>　　陈默同时被自己的徒弟跟这条吸血蛇背叛，怒容满面。
<br/>　　今天有他在，谁也别想从这座牢笼出去。
<br/>　　“师父，你不仁那就别怪我不义，吸血蛇杀了他，吃了他，我没他这个师父了”
<br/>　　“吼..”
<br/>　　昏暗的牢笼弥漫血的味道，只见一位清瘦的中年男子跟二十米长的巨蟒展开了一场生死大战。
<br/>　　这景象凄绝！亦艳绝！
<br/>　　吸血蛇的强大，已经超出了蛇博士陈默的预料，渐渐的，他发现自己无所不能的控蛇手段对这条蟒蛇已经完全都不奏效。
<br/>　　它咆哮狰狞，血口一张，陈默只能暴退，根本不能硬碰硬。
