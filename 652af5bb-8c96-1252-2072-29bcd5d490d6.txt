　　此猫散发出来的阴冷气息，不正是他寻找的特殊蛇类气息吗。
<br/>　　不怪涂小安大惊小怪，试想一只猫的身上，散发出来的是蛇类的气息，这不是颠倒黑白，全乱套了吗。
<br/>　　之前就早说过，气息是做不了假，人有人气，妖有妖气。
<br/>　　不同的动物，气息都截然不同，这就跟每个人的指纹都不会相同一样。
<br/>　　况且还是两种完全不同的生物。
<br/>　　正当涂小安惊奇不已的时候，倏然地，他的脸色骤变，一股强大无匹的残暴气息排山倒海的朝他扑面而来。
<br/>　　几乎是一霎那的功夫，涂小安连退三大步，这股残暴气息如龙卷风，风卷残云的将整个实验室搅的一片狼藉，各种药水器具，杂物统统被瞬间切成了粉碎。
<br/>　　一只隐藏在黑暗角落的蜘蛛从自己的蜘蛛网上跌落，瞬间毙命。
<br/>　　无数纤小生物，在这股气息下，全部死去，根本无法承受压力。
<br/>　　楼层之中的蛇类嘶嘶作响，众蛇都缩成一团，瑟瑟发抖。
<br/>　　涂小安稳住身形，内心惊骇之余，微微松了一口气，这气息要是在强悍一丝，他的这具虚假的人形态，恐怕就要溃散了。
<br/>　　紧接着，涂小安放眼望去，只见不远处一只浑身雪白，毛发炸立的猫周边强风赫赫，那双红目仿佛要射出精光出来。
<br/>　　此猫的眼瞳妖邪的很，但却像极了蛇眼，恍然间，涂小安差点以为这是一只披着猫皮的蛇。
<br/>　　从眼睛，从气息，都让人误以为是一条蛇，无比强大的蛇，甚至不会比涂小安这条蛇王要逊色。
<br/>　　否则外面的那些蛇类，也不会不听从他蛇王的旨意。
<br/>　　明明是一只猫精，为什么会有这种异象，涂小安一时半刻真的想不通。
<br/>　　只见这只猫精越发的暴戾起来，惊雷让它一时吃亏，它将涂小安当成了死敌，无比暴躁，那双妖艳的红眸越发光亮，那红光像是染色体一样的将它浑身雪白的毛发都变得殷红起来。
<br/>　　它周身蛇类气息愈演愈烈起来，要不是涂小安已经登基坐殿，他恐怕都要臣服在这股蛇类气息之下。
<br/>　　如果觉醒太阴蛇在，绝对会顶礼膜拜了，根本无法承受这股蛇类气息。
<br/>　　“它在干嘛？要变身？”
<br/>　　涂小安惊讶之余，看的目瞪口呆，旋即，他倒吸了一口凉气，变得全身殷红的猫精也就算了，但体型依然较小，可骤然间，整个实验室剧烈的震动起来。
<br/>　　只闻轰然一声。
<br/>　　等涂小安再看过去的时候，眼中满是古怪。
<br/>　　这散发蛇类气息的猫精，既然也具备大小神通，那娇小的猫体型已然暴涨了几十倍有余，高度快接近二米，身宽也有一米，活脱脱从娇小可爱的猫，变成了一只巨无霸的猛兽，变的比一头老虎还要高大威猛。
<br/>　　最让涂小安震惊的是，此猫体型暴涨之后，它的猫尾发生了翻天覆地的变化。
<br/>　　变成了一条长长拖地的蛇尾，粗大，有力。
<br/>　　“这莫非是猫与蛇的混合体不成？”
<br/>　　涂小安真的可谓吃了一大惊，但随即就镇定了下来，他也是算见过大世面的，离奇的事情也见过不少了。
<br/>　　心理素质早就强大起来。
<br/>　　越奇葩的事情，都总有一个解释，只是他现在还没真正找到源头罢了。
<br/>　　管你是猫还是蛇，在本蛇王的面前，都要给我匍匐下来。
<br/>　　涂小安妖治的眼神刚霸气侧漏起来，顿时间，他的眼前一花，胸口被一股无法形容的巨力撞到。
<br/>　　“轰...！”
<br/>　　实验室一面洁白的墙壁瞬间凹凸了进去，一股蜘蛛网般的裂痕迅速的蔓延开来，涂小安整个人像被烙印在墙壁中，感觉浑身要散架了。
<br/>　　痛，痛，痛，他既然感觉到了疼痛的错觉，要知道这是虚假的身体。
<br/>　　这变身之后的猫精好快的速度，明明变成洪水猛兽，却速度更快，力量就更不用说了，何止千斤之重。
<br/>　　涂小安都还没反应就中招了，这或许也是两者的距离太短，毕竟只是在一间实验室。
<br/>　　涂小安清秀的脸盘都扭曲起来，一个余光下，变身猛兽的猫精一双妖邪如毒蛇的眼睛死死的锁定住他，恨不得将他一口吞下。
<br/>　　这时...
<br/>　　“叮...宿主的人形态损坏百分之五十，损坏程度超过百分之八十，将立刻的溃散”
<br/>　　系统的提示音一出，涂小安暗暗蹙眉起来，假的就是假的身体，就是徒有其表。
<br/>　　要是他蛇躯的强度，这一撞虽然可怕，但依然能做到无碍。
<br/>　　“给我洞察，我到要看看你是只什么怪物”
<br/>　　涂小安趁着猫精没立刻冲自己发动攻击，直接一个意识洞察丢了过去。
<br/>　　“叮...海棠，二十九级高等猫科生物，寿命一百三十二岁，拥有大小变体超能力手段，静则乖巧，动则残暴”
<br/>　　猛兽版的猫精资料瞬间出现在涂小安的脑海中。
<br/>　　但几乎片刻，涂小安就否认了这份数据资格，这数据根本不对。
<br/>　　数据上说，这是一只等级高达二十九的高等猫精，甚至有可能突破三十级，成为三阶生灵。
<br/>　　寿命更是夸张的比一个人类还长寿，区区一只猫，既然活了一百多岁了。
<br/>　　但这些涂小安都能理解，毕竟二阶生物，活的久一点也无可厚非，但系统数据并没有说有关蛇的半点讯息。
<br/>　　这分明是一头猫跟蛇的混合体，别的都不说，它身后长长拖地的粗大蛇尾让人触目惊心。
<br/>　　高等生物是有尊号的，如孔雀青枭，这只猫精叫海棠。
<br/>　　海棠？海棠？
<br/>　　这名字怎么那么熟悉，涂小安稍稍一想，顿时想起了天生眼瞎带墨镜的陈高阳。
<br/>　　他的女朋友，不就是叫海棠吗。
<br/>　　几乎瞬间，涂小安醍醐灌顶，在看这只猫精，满脸的古怪跟复杂。
<br/>　　“高阳哥哥，以后海棠就是你的眼，你的手，你的脚”
<br/>　　偷窃陈高阳的回忆，那海棠的声音柔声细气，悦耳动听，纵然涂小安知道陈高阳的女朋友，不过是一只猫精的假扮。
<br/>　　但如今正主猫精在前，它凶煞残暴的一面，实在无法跟那道柔声细语的声音联系到一起去。
